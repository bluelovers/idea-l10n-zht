报告 switch 表达式或带有表达式体的增强 switch 语句的规则。 这些可以转换为代码块。

示例：


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

在应用快速修复后：


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };

仅当项目或模块的语言级别为 14 或更高时，此检查才会报告。
