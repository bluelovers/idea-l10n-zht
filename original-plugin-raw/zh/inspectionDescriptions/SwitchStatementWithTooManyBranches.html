报告 case 标签太多的 switch 语句或表达式。

如此之长的 switch 语句可能令人困惑，或许应进行重构。 有时这并非问题（例如一个域非常复杂，并且有包含大量常量的枚举）。

示例：


    switch (x) {
        case 1 -> {}
        case 2 -> {}
        case 3 -> {}
        case 4 -> {}
        case 5 -> {}
        case 6 -> {}
        case 7 -> {}
        case 8 -> {}
        case 9 -> {}
        case 10 -> {}
        case 11,12,13 -> {}
        default -> {}
    }

使用最大分支数字段可以指定 case 标签的预期最大数量。
