在 Kotlin/JS 项目中报告扩展 React Props 或 State 类型但未标记为 external 的类、接口和对象。

React 组件类型应当为外部接口，以便正确地与 JavaScript 运行时互操作。

示例：

interface MyProps : Props { // 警告：应为 external 接口
    var title: String
}

class MyState : State { // 警告：应为 external 接口
    var count: Int = 0
}

在应用快速修复后：

external interface MyProps : Props {
    var title: String
}
