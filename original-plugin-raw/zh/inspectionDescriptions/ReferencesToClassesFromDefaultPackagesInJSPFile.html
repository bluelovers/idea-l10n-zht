报告对 JSP 文件中默认软件包中类的任何引用。

示例:


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // 错误
    new com.example.GoodClass(...); // 优良
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

注意： 在 JSP 编译期间不会解析此类引用，因为 JSP 的编译类通常会放在非默认包中（例如，Tomcat 的 “org.apache.jsp"，WebSphere 的 “com.ibm._jsp”），但在 Java 中不允许从默认包导入。
