报告可以析构的声明。

示例：


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { my ->
          println(my.second)
          println(my.third)
      }
  }

该快速修复会析构声明并引入具有来自相应类的名称的新变量：


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { (_, second, third) ->
          println(second)
          println(third)
      }
  }
