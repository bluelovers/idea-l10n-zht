报告 self 作为限定符但转义闭包并无此要求的用法。

建议移除冗余的 self。

示例： 


  struct URL {
      func loadAsync() {
          print(self.path)

          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

在应用快速修复后： 


  struct URL {
      func loadAsync() {
          print(path)

          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
