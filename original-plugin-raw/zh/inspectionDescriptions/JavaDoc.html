报告以下 Javadoc 注释缺陷：
  • 在需要有 Javadoc 的位置没有 Javadoc
  • 缺少必要的标记
  • 标记无效或不完整
  • Javadoc 描述缺失或不完整

示例：


  class Main {
  }

在应用快速修复后：


  /**
   *
   */
  class Main {
  }

示例：


  String id(String value) {
    return value;
  }

在应用快速修复后：


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
使用标签页和相应的检查选项可定义需要具备 Javadoc 的可见性范围，以及类、方法、字段和内部类所需的 Javadoc 标记。
