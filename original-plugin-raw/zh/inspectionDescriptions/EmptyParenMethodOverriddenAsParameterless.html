报告没有形参子句且重写带有空形参子句的方法的方法。

如果一个方法有副作用，则在方法中使用圆括号是一个常见的约定。 根据里氏替换原则，当被重写的方法有形参子句时，表明有副作用，重写方法也必须被声明为有副作用的方法。

该快速修复会添加一个空形参子句。

示例：


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x = 2
  }

在应用快速修复后：


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x() = 2
  }
