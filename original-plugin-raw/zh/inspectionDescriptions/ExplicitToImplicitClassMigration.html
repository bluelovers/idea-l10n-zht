报告可被转换为紧凑源文件的普通类

示例：


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

在应用快速修复后：


  void main() {
      IO.println("Hello, world!");
  }

配置检查：

  • 选择将 'System.out.print()' 和 'System.out.println()' 替换为 'java.lang.IO' 类中的方法以将这些调用转换为 IO 方法。
