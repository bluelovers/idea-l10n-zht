报告具有 = { ... } 和推断返回类型的函数。

示例：


  fun sum(a: Int, b: Int) = { a + b } // 此函数的返回类型为 '() -> Int'。

有几个快速修复。 应用快速修复后：


    fun sum(a: Int, b: Int) = a + b // 移除大括号
    fun sum(a: Int, b: Int): () -> Int = { a + b } // 显式指定返回值类型
    fun sum(a: Int, b: Int) = run { a + b } // 转换为 run { ... }
    fun sum(a: Int, b: Int) = { -> a + b } // 显式指定 lambda 签名
