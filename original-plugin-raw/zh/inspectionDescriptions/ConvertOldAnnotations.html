报告 TestNG org.testng.annotations.Configuration 注解。

最好使用现代 @BeforeXXX/@AfterXXX 注解来配置测试套件。

示例：


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

在应用快速修复后：


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
