报告不遵循命名惯例的属性。

报告了以下属性：

  • 顶级属性
  • 对象和伴生对象中的属性

您可以在检查选项中指定所需的模式。

建议的命名惯例：必须以大写字母开头，使用驼峰拼写法并且没有下划线。

示例：


  // 顶级属性
  val USER_NAME_FIELD = "UserName"
  // 包含对单例对象的引用的顶级属性
  val PersonComparator: Comparator<Person> = /*...*/

  class Person {
    companion object {
      // 伴生对象中的属性
      val NO_NAME = Person()
    }
  }
