报告字符串模板中用于括简单标识符的大括号的用法。

使用“移除大括号”快速修复可移除冗余的大括号。

示例：


  fun redundant() {
     val x = 4
     val y = "${x}" // <== 冗余
  }

  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== 可以
  }

在应用快速修复后：


  fun redundant() {
     val x = 4
     val y = "$x"
  }

  fun correctUsage() {
      val x = "x" <== 已更新
      val y = "${x.length}"
  }
