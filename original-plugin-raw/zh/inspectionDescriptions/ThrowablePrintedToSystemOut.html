报告以异常作为实参的 System.out.println() 调用。

使用 print 语句记录异常会隐藏堆栈跟踪，这可能会使问题的调查变得复杂。 建议您改用记录器。

也会报告以异常作为实参的 System.out.print()、System.err.println() 和 System.err.print() 调用。 建议使用记录器来记录异常。

例如，不建议使用：


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

用以下代码代替：


  try {
      foo();
  } catch (Exception e) {
      logger.warn(e); // 记录器调用可能有所不同
  }
