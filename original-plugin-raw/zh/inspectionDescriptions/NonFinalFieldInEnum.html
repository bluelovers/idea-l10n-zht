报告枚举类型中的非 final 字段，因为很少需要它们，且提供全局可变状态。

示例：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

在应用快速修复后：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
配置“在快速修复不可用时忽略字段”复选框以仅高亮显示可通过该快速修复设为 final 的字段。
