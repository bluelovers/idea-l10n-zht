报告 kotlin.coroutines.suspendCoroutine 的用法，并建议在出现 kotlinx.coroutines 依赖项时将其替换为 kotlinx.coroutines.suspendCancellableCoroutine。

suspendCancellableCoroutine 变体提供了内置的取消支持，这对于结构化并发和正确的资源管理非常重要。

示例：


    suspend fun usage(): String {
        return suspendCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }

在应用快速修复后：


    suspend fun usage(): String {
        return suspendCancellableCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }
