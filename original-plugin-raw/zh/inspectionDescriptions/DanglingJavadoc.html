报告不属于任何类、方法或字段的 Javadoc 注释。

Javadoc 工具忽略悬空的 Javadoc 注释，并且不会将它们包含在生成的 HTML 文档中。

示例:


  class A {
    /**
     * Dangling comment
     */
     /**
      * Method javadoc
      */
      public void m(){}
  }

通过快速修复可完全移除这样的注释或将它们转换为块注释。 在应用快速修复后：


  class A {
  /*
    Dangling comment
   */
  /**
   * Method javadoc
   */
  public void m(){}
}
