报告 protected 嵌套类。

示例：


  public class Outer {
    protected static class Nested {} // 警告
    protected class Inner {}         // 警告
    protected enum Mode {}           // 警告取决于设置
    protected interface I {}         // 警告取决于设置
  }

配置检查：

  • 使用忽略 'protected' 内部枚举选项可忽略 protected 内部枚举。
  • 使用忽略 'protected' 内部接口选项可忽略 protected 内部接口。
