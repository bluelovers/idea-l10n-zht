当使用 @override 装饰的方法在其上级类中没有匹配的方法时报告

示例：


from typing import override

class Parent:
    def foo(self) -> int:
        return 1

    def bar(self, x: str) -> str:
        return x

class Child(Parent):
    @override
    def foo(self) -> int:
        return 2

    @override # 重写函数缺少 super 方法
    def baz(self) -> int:
        return 1
