报告 final 类中没有 protected 修饰符的 protected 成员。

由于无法继承 final 类，将方法标记为 protected 可能令人困惑。 最好将此类成员声明为 private 或软件包可见。

示例：

record Bar(int a, int b) {
  protected int sum() { 
     return a + b;
  }
}

在应用快速修复后：

record Bar(int a, int b) {
  int sum() { 
     return a + b;
  }
}

如本例所示，可以显式或隐式地将类标记为 final。
