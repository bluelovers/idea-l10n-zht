报告在 java.util.concurrent.locks.Condition 对象上对 notify() 或 notifyAll() 的调用。

这可能是编程错误，本应使用 signal() 或 signalAll() 方法的某些变体，否则可能出现 IllegalMonitorStateException。

示例：


  class C {
    final Lock l = new ReentrantLock();
    final Condition c = l.newCondition();

    void release() {
      l.lock();
      try {
        c.notifyAll(); // 此处可能本应使用 'signalAll()'
      } finally {
        l.unlock();
      }
    }
  }
