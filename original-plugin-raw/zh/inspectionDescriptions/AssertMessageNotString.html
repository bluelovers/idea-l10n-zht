报告非 java.lang.String 类型的 assert 消息。

使用字符串可以提供更多信息，有助于诊断失败或断言原因。

示例：


  void foo(List<Integer> myList) {
    assert myList.isEmpty() : false;
  }

使用仅当 assert 消息为 'boolean' 或 'java.lang.Boolean' 时发出警告选项可以仅在 assert 消息类型为 boolean 或 java.lang.Boolean 时发出警告。 boolean 详细消息不太可能提供有关断言失败的其他信息，可能是由于错误输入 : 而不是 & 所致。
