报告可以简化的布尔表达式。

示例：


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

在应用快速修复后：


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

示例：


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

在应用快速修复后：


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }
