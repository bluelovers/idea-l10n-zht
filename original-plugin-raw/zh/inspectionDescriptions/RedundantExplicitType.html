报告局部变量的显式给定类型，这些类型显而易见且冗余，例如 val f: Foo = Foo()。

示例：


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

在应用快速修复后：


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
