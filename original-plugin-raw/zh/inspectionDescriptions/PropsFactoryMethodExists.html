报告可将动态构造函数调用安全替换为工厂方法调用的情况。

示例：


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Props(classOf[Foo])

在应用快速修复后：


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Foo.props()
