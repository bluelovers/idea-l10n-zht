报告任何无需创建类实例并且可以设置为 static 的方法。

示例:

class MyClass(object):
    def my_method(self, x):
        print(x)

如果应用从方法中生成函数快速修复，代码将变为:

def my_method(x):
    print(x)


class MyClass(object):
    pass

如果选择将方法设为 static快速修复，将添加 @staticmethod 装饰器:

class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
