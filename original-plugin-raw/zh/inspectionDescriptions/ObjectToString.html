报告对使用 java.lang.Object 中默认实现的 toString() 的调用。

该默认实现很少是有意为之，可能是误用。

此检查将忽略对包含 java.lang.Object、接口或 abstract 类类型的对象的 toString() 调用。

示例：


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // 警告
          /* ... */
      }

      void foo2(Object obj) {
          String s = obj.toString(); // 此处无警告
          /* ... */
      }
  }
