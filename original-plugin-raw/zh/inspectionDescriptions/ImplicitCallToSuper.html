报告不以调用 "super" 构造函数或同一类的另一个构造函数开头的构造函数。

可以认为是此类构造函数隐式地从调用 super() 开始。 某些编码标准倾向于显式地调用 super()。

示例:

  class Foo {
    Foo() {}
  }

在应用快速修复后：

  class Foo {
    Foo() {
      super();
    }
  }

使用检查设置可以忽略直接从 Object 扩展的类。 例如： 

  class Foo {
    Foo() {} // Not reported
  }

  class Bar extends Foo {
    Bar() {} // 对 'super()' 的隐式调用
  }
