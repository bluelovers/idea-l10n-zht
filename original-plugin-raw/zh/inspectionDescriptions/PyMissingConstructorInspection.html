报告类中缺少对 super 构造函数进行调用的情况。

示例：

class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        pass

Pear 类在 __init__ 方法中应有 super 调用。

应用快速修复后，代码变为：

class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        super().__init__()
