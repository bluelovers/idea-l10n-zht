报告 <code> 标记在 Javadoc 注释中的用法。 从 Java 5 起，这些标记可以替换为 {@code ...} 结构。 这允许在注释中使用尖括号 < 和 > 而不是 HTML 字符实体。

示例：


  /**
   * @return empty <code>ArrayList&lt;Integer&gt;</code>
   */
  List<Integer> getList(){ ... }

在应用快速修复后：


  /**
   * @return empty {@code ArrayList<Integer>}
   */
  List<Integer> getList(){ ... }
