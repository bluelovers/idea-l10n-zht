报告用作另一个方法调用的形参的方法调用。

该快速修复会引入一个变量，使代码更简单，也更易于调试。

示例：


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int x = f(y());
  }

在应用快速修复后：


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int y = y();
      int x = f(y);
  }

使用检查选项可以切换以下各项的报告：

  • 字段初始值设定项中的方法调用
  • 对 static 方法的调用
  • 对简单 getter 的调用
