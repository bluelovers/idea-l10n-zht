报告单个 try 语句中相同的 catch 部分。

将这些部分折叠成一个 multi-catch 块可以减少代码重复，并防止出现一个 catch 部分更新而另一个部分未更新的情况。

示例：


    try {
        doSmth();
    }
    catch (IOException e) {
        LOG.error(e);
    }
    catch (URISyntaxException e) {
        LOG.error(e);
    }

可通过快速修复使代码更加紧凑：


    try {
        doSmth();
    }
    catch (IOException | URISyntaxException e) {
        LOG.error(e);
    }
