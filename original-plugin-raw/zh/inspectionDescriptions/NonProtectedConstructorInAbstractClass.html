报告 abstract 类的 public 构造函数。

abstract 类的构造函数只能从其子类的构造函数进行调用，将其声明为 public 可能令人困惑。

该快速修复会将此类构造函数设为 protected。

示例：


  public abstract class Foo {
    public Foo () { // 警告：有 'public' 修饰符
        /* ... */
    }
  }

在应用快速修复后：


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

配置检查：

使用下面的忽略非 public 类选项可以忽略非 public 类中的 public 构造函数。
