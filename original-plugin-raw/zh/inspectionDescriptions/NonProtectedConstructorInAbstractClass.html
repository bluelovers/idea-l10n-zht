报告声明为 public 的 abstract 类的构造函数。

abstract 类的构造函数只能在其子类的构造函数中调用。 因此，令 abstract 类的构造函数为 public 可能会使人混淆。

该快速修复会将此类构造函数设为 protected。

示例:


  public abstract class Foo {
    public Foo () { // 警告：有 'public' 修饰符
        /* ... */
    }
  }

在应用快速修复后：


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

配置检查：

使用下面的选项可以忽略非 public 类。
