报告对 super(A, B) 的任何调用不符合以下要求的情况：

  • B 是 A 的实例
  • B 是 A 的子类

示例：

class Figure:
    def color(self):
        pass


class Rectangle(Figure):
    def color(self):
        pass


class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square 不是 Rectangle 的子类

作为修复，您可以令 Square 为 Rectangle 类的实例。
