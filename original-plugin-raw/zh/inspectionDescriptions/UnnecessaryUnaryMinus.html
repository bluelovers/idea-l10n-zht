报告不必要的一元减号。 此类表达式可能难以理解并且可能包含错误。

例如：

void unaryMinus(int i) {
    int x = - -i;
  }

此处建议使用以下快速修复：

  • 移除 i 变量前的 - 运算符：

    void unaryMinus(int i) {
    int x = i;
  }

  • 使用前缀递减运算符替换 - 运算符：

    void unaryMinus(int i) {
    int x = --i;
  }

另一个示例：

void unaryMinus(int i) {
    i += - 8;
  }

在应用快速修复后：

void unaryMinus(int i) {
    i -= 8;
  }
