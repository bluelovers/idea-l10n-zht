报告子类方法的参数类型与对应父类方法的参数类型同名但属于不同包的情况。 在这些情况下，子类方法不能重写 super 方法。

建议设置与父类方法中相同的参数类型。

示例:


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(Object o) {} // 对形参类型发出警告
 class Object {}
}

在应用快速修复后：


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(java.lang.Object o) {} // 新形参类型
 class Object {}
}
