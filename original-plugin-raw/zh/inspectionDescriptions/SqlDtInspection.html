报告日期和时间字面量中的错误。 此检查在 MySQL、Oracle、Db2 和 H2 中可用。

示例 (MySQL)：

SELECT TIME '10 -12:13:14' FROM dual;
SELECT TIME ' 12 : 13 : 14 ' FROM dual;
SELECT TIME '12 13 14' FROM dual;
SELECT TIME '12-13-14' FROM dual;
SELECT TIME '12.13.14' FROM dual;
SELECT TIME '12:13:' FROM dual;
SELECT TIME '12:13' FROM dual;
SELECT TIME '12:' FROM dual;

在此示例中，日期会忽略 MySQL 日期字面量和时间字面量标准。 因此，这些将高亮显示。 有关 MySQL 中日期和时间字面量的详细信息，请参阅 dev.mysql.com 上的 Date and Time Literals。

以下日期和时间字面量对 MySQL 有效。

SELECT TIME '12:13:14' FROM dual;
SELECT TIME '12:13:14.555' FROM dual;
SELECT TIME '12:13:14.' FROM dual;
SELECT TIME '-12:13:14' FROM dual;
SELECT TIME '10 12:13:14' FROM dual;
SELECT TIME '-10 12:13:14' FROM dual;
