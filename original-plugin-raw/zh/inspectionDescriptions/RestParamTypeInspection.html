报告具有不受支持类型的资源方法形参（使用 @PathParam、@QueryParam 等注解）。

注解的形参、字段或属性的类型必须满足以下要求之一：

  1. 为基元类型
  2. 有一个接受单个 String 实参的构造函数
  3. 有一个名为 valueOf() 或 fromString() 的 static 方法，它接受单个 String 实参，例如 Integer.valueOf(String)。
  4. 拥有 ParamConverterProvider JAX-RS 扩展 SPI 的注册实现，它返回一个能够将字符串转换为特定类型的 ParamConverter 实例。
  5. 为 List<T>、Set<T> 或 SortedSet<T>，其中 T 为 2、3 或 4 以上。 生成的集合为只读。
