检测可以通过将实参转换为 Set 而加快速度的函数调用。

当实参为集合时，'minus' 或 'intersect' 等运算更高效。 将 Iterable<T> 或 Array<T> 显式转换为 Set<T> 通常可以提高代码效率。

该快速修复会向函数调用添加显式转换。

示例：


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b).size

在应用快速修复后：


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b.toSet()).size
