报告不从接收器访问任何内容的冗余 with 函数调用。

示例：


  class MyClass {
      fun f(): String = ""
  }

  fun testRedundant() {
      with(c) { // <== 'with' 冗余，因为 'c' 未使用
          println("1")
      }
  }

  fun testOk() {
      val c = MyClass()
      with(c) { // <== 可以，因为 'f()' 是有效的 'c.f()'
          println(f())
      }
  }
