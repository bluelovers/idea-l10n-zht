报告 Spring Shell org.springframework.shell.standard.ShellMethodAvailability 注解内的无效引用。

示例：


@ShellComponent
public class MyCommands {

    @ShellMethod("sum")
    @ShellMethodAvailability("sumAvailability") // 注解值将高亮显示为未解析的符号
    public int sum(int a, int b) {
        return a + b;
    }

    @ShellMethodAvailability("sumMethod") // 注解值将高亮显示为未解析的符号
    public Availability isSumAvailable() {
        return Availability.available();
    }
}
