报告在 java.util.concurrent.locks.Condition 对象上进行的 wait() 调用。 这可能是编程错误，本应使用 await() 方法的某种变体。

示例：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

优良的代码应类似于：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
