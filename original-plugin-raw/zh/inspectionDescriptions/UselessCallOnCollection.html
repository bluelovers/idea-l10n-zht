报告标准库中已筛选的集合上的 filter… 调用。

只有在类型不同于结果类型的接收器上调用时，标准库中 filterNotNull() 或 filterIsInstance 之类的几个函数才有意义。 在其他情况下，可以忽略此类调用，因为结果将相同。

移除冗余调用快速修复可用于自动修正代码。

示例：


  fun test(list: List<String>) {
      val x = list.filterNotNull() // 快速修复会简化为 'list'
      val y = list.filterIsInstance<String>() // 快速修复会简化为 'list'
  }
