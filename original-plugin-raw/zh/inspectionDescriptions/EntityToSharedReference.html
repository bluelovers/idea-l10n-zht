报告从 Entity 到 SharedEntity 的无效引用。

不能从 SharedEntity 引用 Entity。 仅当属性为可 null 或具有 @CascadeDeleteBy 注解时，才能从 Entity 引用 SharedEntity。 否则，可能会导致运行时异常。

示例：


  interface A: Entity {
    var x: SharedEntity
    var y: SharedEntity
  }

可以通过快速修复来生成注解或使类型为可 null


  interface A: Entity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
