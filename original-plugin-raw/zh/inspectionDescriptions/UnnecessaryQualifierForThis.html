报告不必要的 this 或 super 限定。

在 this 或 super 上使用限定符来消除代码引用的歧义可能很容易通过自动重构变得不必要，为了清楚起见应该删除。

示例：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

在应用快速修复后：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
