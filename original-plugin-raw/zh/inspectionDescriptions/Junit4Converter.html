报告可以转换为 JUnit 4 测试类的 JUnit 3 测试类。

示例：


  public class MainTestCase extends junit.framework.TestCase {
    public void test() {
      Assert.assertTrue(true);
    }
  }

在应用快速修复后：


  public class MainTestCase {
    @org.junit.Test
    public void test() {
      Assert.assertTrue(true);
    }
  }

当项目或模块的语言级别为 5 或更高并且在类路径上可以使用 JUnit 4 时，此检查才会报告。
