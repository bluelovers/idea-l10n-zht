报告调用 ReactorDebugAgent.init() 时 Hooks.onOperatorDebug() 的用法。

如果 ReactorDebugAgent 在类路径中，并且 spring.reactor.debug-agent.enabled 为 true（默认值），启动时自动调用 ReactorDebugAgent.init()。 同时使用 Hooks.onOperatorDebug() 和 ReactorDebugAgent.init() 会导致 Reactor 生成双倍的调试堆栈帧。 此外，Hooks.onOperatorDebug() 会导致性能开销。

该快速修复会移除对 Hooks.onOperatorDebug() 的调用。

示例：


  public void hook() {
    Hooks.onOperatorDebug(); // 报告 '初始化 ReactorDebugAgent 时调用 Hooks.onOperatorDebug()'
  }

在应用快速修复后：


  public void hook() {
  }

此检查仅在项目或模块的 Spring Boot 版本为 2.2.0 或更高时触发。
