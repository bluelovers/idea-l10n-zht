报告基于实参的冗余 let 调用。

当 lambda 形参仅用作调用表达式中的限定符时，let 冗余。

如果需要为限定表达式命名，请声明一个局部变量。

示例：


  fun splitNumbers() {
      "1,2,3".let { it.split(',') }
  }

该快速修复会移除额外的 let() 调用：


  fun example() {
      "1,2,3".split(',')
  }

或者：


  fun splitNumbers() {
      val numbers = "1,2,3"
      numbers.split(',')
  }
