报告不正确的 Spring Shell 命令声明：

  • 缺少命令描述
  • 命令不在 org.springframework.shell.standard.@ShellComponent 内

示例：


// 由于缺少 @ShellComponent，将报告所有方法
class MathCommands {

  @ShellMethod("")
  public int sum(int a, int b) { // 由于描述为空，将报告方法
    return a + b;
  }

  @ShellMethod
  public int mul(int a, int b) { // 由于缺少描述，将报告方法
    return a * b;
  }
}
