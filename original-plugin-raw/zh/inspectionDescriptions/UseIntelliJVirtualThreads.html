报告使用 Thread.ofVirtual 而不是 IntelliJVirtualThreads.ofVirtual 的情况。

Thread.ofVirtual 是一个用于创建虚拟线程的通用 Java API，不适合 IntelliJ 平台的需求。

快速修复将使用 IntelliJVirtualThreads.ofVirtual 执行替换。

示例： 


// 错误：
Thread thread = Thread.ofVirtual().start();

// 正确：
Thread thread = IntelliJVirtualThreads.ofVirtual().start();
