报告可以替换为 Collection.indices 或 for 循环内集合迭代的 until 和 rangeTo 运算符。

使用语法糖可使代码更简单。

该快速修复会将手动范围替换为相应的构造。

示例：


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

在应用快速修复后：


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
