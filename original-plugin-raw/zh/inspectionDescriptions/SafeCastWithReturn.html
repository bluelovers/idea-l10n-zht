报告带 return 且可以替换为 if 类型检查的安全转换。

使用相应函数可以简化代码。

该快速修复会将安全转换替换为 if 类型检查。

示例：


  fun test(x: Any) {
      x as? String ?: return
  }

在应用快速修复后：


  fun test(x: Any) {
      if (x !is String) return
  }

同样，它能检测到表单中的不安全转换 


    fun foo(o: Any) {
    o as String ?: return
}

并转换为类型检查：


    fun foo(o: Any) {
    if (o !is String) return
}
