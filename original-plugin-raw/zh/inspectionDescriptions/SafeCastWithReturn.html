报告带 return 且可以替换为 if 类型检查的安全转换。

该快速修复会将安全转换替换为 if 类型检查。

示例：


  fun test(x: Any) {
      x as? String ?: return
  }

在应用快速修复后：


  fun test(x: Any) {
      if (x !is String) return
  }

Analogously, it detects unsafe casts in form 


    fun foo(o: Any) {
    o as String ?: return
}

and transforms into a type check:


    fun foo(o: Any) {
    if (o !is String) return
}
