报告对 lambda 参数的赋值，包括复合赋值和递增或递减参数。

虽然有时是有意为之，但这种结构可能会引起混淆，通常由拼写错误或使用了错误变量引起。

该快速修复会添加新变量的声明。

示例:


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

在应用快速修复后：


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

使用如果赋值是原始参数的转换，则忽略选项来忽略根据先前值修改参数值的赋值。
