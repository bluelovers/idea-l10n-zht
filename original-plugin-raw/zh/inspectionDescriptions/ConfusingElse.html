在 if—else 语句和语句链中报告冗余的 else 关键字。

如果 else 关键字因为之前分支中的代码以 return、throw、break 或 continue 语句结尾而无法到达，则它是冗余的。 在这种情况下，来自 else 分支的语句可以放在 if 语句之后，并且 else 关键字可以删除。

示例：


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

在应用快速修复后：


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

禁用 当 'if' 语句之后没有其他语句时报告 选项以忽略 if—else 语句是代码块中最后一条语句的情况。
