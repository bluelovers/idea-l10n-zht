返回可以替换为表达式主体语法的 return 表达式（一行代码或 when）。

表达式主体语法建议在样式指南中给出。

转换为表达式主体快速修复可用于自动修正代码。

示例：


  fun sign(x: Int): Int {
      return when { // <== 可以简化
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

在应用快速修复后：


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
