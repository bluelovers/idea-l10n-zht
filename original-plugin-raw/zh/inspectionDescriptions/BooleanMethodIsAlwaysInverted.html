报告一个返回类型为 boolean 的方法，该方法仅在否定上下文中使用。

您可以使用快速修复来重命名方法和反转方法。 由于性能原因，某些方法可能不会在编辑器中高亮显示。

示例：


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

在应用快速修复后：


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
