报告被声明为 public 的 Object.finalize() 方法的任何实现。

根据 Object.finalize() 的约定，只有垃圾回收器调用此方法。 将此方法设为 public 可能令人困惑，因为这意味着该方法可供用户使用。

该快速修复会将方法设为 protected，防止它被其他类显式调用。

示例：


  class X {
    public void finalize() {
      /* ... */
    }
  }

在应用快速修复后：


  class X {
    protected void finalize() {
      /* ... */
    }
  }
