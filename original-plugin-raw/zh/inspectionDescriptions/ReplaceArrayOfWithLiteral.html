报告可以替换为数组字面量 [...] 的 arrayOf 调用。

示例：

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // 可替换的 'arrayOf()'
  class MyClass

在应用快速修复后：

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass
