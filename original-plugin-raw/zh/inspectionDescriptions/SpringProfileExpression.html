报告不正确的 @Profile 表达式：

  • Spring 配置文件不得为空
  • 在 Spring 配置文件表达式中，'&' 和 '|' 运算符不得在没有圆括号的情况下混用

示例：


  
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  
  @Configuration
  @Profile() // 报告 "配置文件表达式必须包含文本"
  public class MyConfiguration {...}

  
  @Configuration
  @Profile("a & b | c") // 报告 "格式不正确的配置文件表达式"
  public class MyConfiguration {...}
