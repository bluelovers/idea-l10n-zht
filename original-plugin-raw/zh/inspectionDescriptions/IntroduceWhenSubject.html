报告可以通过引入主体实参进行简化的 when 表达式。

示例：


  fun test(obj: Any): String {
      return when {
          obj is String -> "string"
          obj is Int -> "int"
          else -> "unknown"
      }
  }

该快速修复会引入一个主体实参：


  fun test(obj: Any): String {
      return when (obj) {
          is String -> "string"
          is Int -> "int"
          else -> "unknown"
      }
  }
