报告具有 @deprecated Javadoc 标记但没有 @java.lang.Deprecated 注解的模块声明、类、字段或方法。

示例：


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

在应用快速修复后：


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

此检查仅在项目或模块的语言级别为 5 或更高时有效。

使用下面的复选框，对使用 @Deprecated 注解但在 @deprecated Javadoc 标记中没有解释的符号发出警告。
