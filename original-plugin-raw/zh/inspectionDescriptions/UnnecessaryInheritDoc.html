报告仅包含 {@inheritDoc} 标记的 Javadoc 注释。 由于 Javadoc 会在没有注释的情况下复制超类的注释，因此仅包含 {@inheritDoc} 的注释不会添加任何内容。

此外，它还会报告无效位置（例如，字段）中的 {@inheritDoc} 用法。

建议移除不必要的 Javadoc 注释。

示例：


  class Example implements Comparable<Example> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }

在应用快速修复后：


  class Example implements Comparable<Example> {
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }
