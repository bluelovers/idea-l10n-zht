如果未使用返回值，则报告具有 Deferred 结果类型的函数调用。

如果未使用 Deferred 返回值，则调用站点不会等待补全此函数。

示例：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }

该快速修复会提供具有 Deferred 初始值设定项的变量：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      val answer = calcEverythingAsync()
  }
