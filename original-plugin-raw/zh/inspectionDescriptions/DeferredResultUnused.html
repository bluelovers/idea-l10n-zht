如果未使用返回值，则报告具有 Deferred 结果类型的调用。

如果有返回值，函数将不会等到它完成。

示例:


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }

建议通过快速修复引入具有 Deferred 初始值设定项的变量：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      val answer = calcEverythingAsync()
  }
