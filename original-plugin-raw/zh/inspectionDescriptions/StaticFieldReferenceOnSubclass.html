报告对其中的调用由声明类的子类而不是由声明类本身限定的 static 字段的访问。

Java 允许此类限定，但是此类访问可能表明继承和重写之间存在难以捉摸的混淆。

示例：


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Child.foo);
  }

应用快速修复后，结果如下所示：


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Parent.foo);
  }
