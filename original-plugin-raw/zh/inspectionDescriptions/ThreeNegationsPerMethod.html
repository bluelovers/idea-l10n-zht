报告有三个或更多否定的方法。 此类方法可能令人困惑。

示例：


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

没有否定，该方法变得更容易理解：


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

配置检查：

  • 使用忽略 'equals()' 方法中的否定选项可禁用 equals() 方法中的检查。
  • 使用忽略 'assert' 语句中的否定可禁用 assert 语句中的检查。
