报告在对象构造期间对当前类的 abstract 方法的调用。

如果方法位于以下位置，则会在对象构造期间调用：

  • 构造函数
  • 非 static 实例初始值设定项
  • 非 static 字段初始值设定项
  • clone() 方法
  • readObject() 方法
  • readObjectNoData() 方法

此类调用可能会导致难以捉摸的错误，因为对象初始化可能发生在方法调用之前。

示例：


  abstract class Parent {
    abstract void abstractMethod();
  }

  class Child extends Parent {
    Child() {
      abstractMethod();
    }
  }

此检查与以下检查共享功能：

  • 在对象构造期间调用的可重写方法
  • 在对象构造期间调用的重写的方法

一次只能启用一个检查，以防止警告重复。
