报告 == 或 != 运算符的用法，用于应替换为 contentEquals() 的数组。

== 和 != 运算符比较数组引用而不是它们的内容。

示例：

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // 参考比较
  }

在应用快速修复后：

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
