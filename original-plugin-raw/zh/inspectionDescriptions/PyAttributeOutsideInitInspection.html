报告实例特性定义在__init__ 方法外部的问题。

示例：

    class Book:
    def __init__(self):
        self.author = 'Mark Twain'

    def release(self):
        self.year = '1889'

应用快速修复后，代码示例变为： 

    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'

    def release(self):
        pass
