报告未被声明为 public 和 static、不返回 void 或接受实参的 JUnit 4.0 @BeforeClass 或 @AfterClass 方法。 此类方法很容易无意间创建，但是 JUnit 测试运行程序不会执行这些方法。

示例：


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) {
    }
  }

应用快速修复后，该方法变为：


  class MainTest {
    @BeforeClass
    public static void beforeClass() {
    }
  }
