报告格式错误的 @Rule 或 @ClassRule 注解用法。 无法运行规则格式错误的测试。
  • @Rule 注解成员必须为 public 以及 org.junit.rules.TestRule 或 org.junit.rules.MethodRule 的子类型
  • @ClassRule 注解成员必须为 public 和 static 以及 org.junit.rules.TestRule 的子类型

示例：


  public class MomentousTest {
    @Rule
    private final TemporaryFolder tempFolder =
      new TemporaryFolder();

    // ... 测试转到此处
  }

通过快速修复来修正修饰符：


  public class MomentousTest {
    @Rule
    public final TemporaryFolder tempFolder =
      new TemporaryFolder();

    // ... 测试转到此处
  }
