报告与元素的默认模态匹配的模态修饰符（对于大多数元素来说是 final，对于具有 override 的成员来说是 open）。

示例：


  final class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }

在应用快速修复后：


  class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }
