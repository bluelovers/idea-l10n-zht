报告不正确的仓库方法形参类型。

示例：


  public class Person {
    private int id;
    @Column
    private String lastname;

    @Column
    private int age;
  }

  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // 应为 'String' 类型
    List<Person> deleteByAge(String str);   // 应为 'int' 类型
  }
