在 @DirtiesContext 注解中报告不正确的 ‘mode' 配置。

示例:


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // 在注解的测试类上设置方法模式没有意义。 对于类级别的控制，请改用 classMode。
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // 当通过 @ContextHierarchy 将上下文配置为层次结构的一部分时，应使用 hierarchyMode
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // 在注解的测试类上设置方法模式没有意义。 对于方法级控制，请改用 methodMode。
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // 当通过 @ContextHierarchy 将上下文配置为层次结构的一部分时，应使用 hierarchyMode
    public void testFoo() {...}
  }
