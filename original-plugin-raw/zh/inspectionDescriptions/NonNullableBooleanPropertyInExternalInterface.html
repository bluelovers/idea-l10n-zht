在 Kotlin/JS 项目中报告外部接口中不可为 null 的 Boolean 属性。

外部接口中的 Boolean 属性应当可为 null，以处理 JavaScript 值可能为 undefined 的情况。

示例：

external interface MyInterface {
    var enabled: Boolean // 警告：应当为 Boolean?
    var visible: Boolean // 警告：应当为 Boolean?
}

在应用快速修复后：

external interface MyInterface {
    var enabled: Boolean?
    var visible: Boolean?
}
