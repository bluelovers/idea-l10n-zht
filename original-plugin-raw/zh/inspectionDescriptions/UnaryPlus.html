报告使用 + 一元运算符的情况。 一元加号通常为 null 运算，它的存在可能代表编码错误。 例如，与递增运算符（如 +++ 中）或相等运算符（如 =+ 中）结合使用。

示例：


  void unaryPlus(int i) {
    int x = + +i;
  }

建议使用以下快速修复：

  • 移除 i 变量前的 + 运算符：

    
  void unaryPlus(int i) {
    int x = i;
  }

  • 使用前缀递增运算符替换 + 运算符：

    
  void unaryPlus(int i) {
    int x = ++i;
  }

使用下面的复选框可以报告与二进制或另一个一元表达式一起使用的一元加号。 这意味着检查将不会报告将一元加号表达式用于数组初始值设定项表达式或用作方法实参的情况。
