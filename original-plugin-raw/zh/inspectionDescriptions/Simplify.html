报告可以简化的语句，例如常量条件、相同的 if 分支、冗余布尔表达式等。 使用下面的复选框来控制分析选项：
  • 简化 "if (true)", "while(false)": 建议用布尔文字替换 if/while 语句。
  • 简化 "condition == true": 建议用布尔文字替换相等语句。
  • 简化 "condition1 ? true : condition2;": 建议简化三元运算符，其中表达式之一是布尔文字。
  • 简化 "if (condition1) return true; return condition2;": 建议简化 if，其中如果条件为 true 则返回布尔文字，否则返回另一个条件。
