报告可以简化的语句，例如常量条件、相同的 if 分支、冗余布尔表达式等。 使用下面的复选框可以控制分析选项：
  • 简化 "if (true)", "while(false)"：建议将 if/while 语句替换为布尔字面量。
  • 简化 "condition == true"：建议将相等语句替换为布尔字面量。
  • 简化 "condition1 ? true : condition2;"：建议简化其中一个表达式是布尔字面量的三元运算符。
  • 简化 "if (condition1) return true; return condition2;"：建议简化符合以下条件的 if：如果条件为 true 则返回布尔字面量，否则返回另一个条件。
