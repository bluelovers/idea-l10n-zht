报告 JPA 实体中不可变集合的用法。

检测使用具有不可变集合类型（List、Set、Map）的 @OneToMany 或 @ManyToMany 注解的属性。 建议使用可变集合类型（MutableList、MutableSet、MutableMap）来确保 Hibernate 兼容性。

嵌入代码段： 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: List = listOf(),
)
应用该快速修复后： 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: MutableList = mutableListOf(),
)
