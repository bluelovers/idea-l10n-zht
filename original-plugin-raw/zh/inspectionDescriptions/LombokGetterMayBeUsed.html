报告可由 lombok @Getter 注解替换的标准 getter 方法。

示例：


  import java.util.Date;

  public class MyClass {
    /**
     * 处理日期。
     */
    private Date processDate;

    /**
     * 返回日期。
     *
     * @return The date
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

在应用快速修复/清理后：


  import lombok.Getter;
  import java.util.Date;

  @Getter
  public class MyClass {
    /**
     * 处理日期。
     * -- GETTER --
     * 返回日期。
     *
     * @return The date
     */
    private Date processDate;
  }

它仅在配置 lombok 库时报告。 要移除 lombok，请参阅 lombok 网站。
