报告 property() 的实参和用 @property 注解的函数的问题。

class C:
    @property
    def abc(self):  # Getter 应返回或生成某些内容
        pass

    @abc.setter
    def foo(self, value):  # 函数名称与装饰器不匹配
        pass

    @abc.setter
    def abc(self, v1, v2):  # Setter 签名应为 (self, value)
        pass

    @abc.deleter
    def abc(self, v1):  # Deleter 签名应为 (self)
        pass

快速修复提议更新形参。
