报告参数名称和数量与超类中的方法相同，并且其中至少一个参数不是相同类型但类型兼容的实例方法。

在本例中，子类中的方法将重载而不是重写超类中的方法。 如果不是有意为之，可能会导致潜在错误。

示例:


  public class Foo {
    void foo(int x) {}
  }

  public class Bar extends Foo {
    void foo(Number x) {} // 在重写可能是有意为之的情况下，
                          // 方法 'foo()' 重载超类的兼容方法
  }

使用此选项可选择该检查是否也应报告形参类型不兼容的情况。
