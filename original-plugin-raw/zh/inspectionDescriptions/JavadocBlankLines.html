报告 Javadoc 注释中的空行。

Javadoc 中的空行可能表示将文本拆分为不同段落的意图。 但是，Javadoc 工具和 IntelliJ IDEA 在呈现文档注释时会忽略它们。

该快速修复会建议将空行替换为段落标记 (<p>)。

示例：


  class Main {
    /**
     * 不起作用。
     *
     * 完全不起作用
     */
    void foo() {}
  }

在应用快速修复后：


  class Main {
    /**
     * 不起作用。
     * <p>
     * 完全不起作用
     */
    void foo() {}
  }
