报告集合与 String 的否定 isEmpty() 和 isNotEmpty()，或者 String 的 isBlank() 和 isNotBlank()。

使用相应函数可使代码更简单。

该快速修复会将否定调用替换为标准库中的相应调用。

示例：


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // 执行 smth
      }
  }

在应用快速修复后：


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // 执行 smth
      }
  }
