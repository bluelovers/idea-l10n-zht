报告可以用单个 String 串联替换的 StringBuffer、StringBuilder 或 StringJoiner 的用法。

使用 String 串联会使代码更短更简单。

此检查仅在建议的替换不会导致现代 JVM 的性能显著下降时报告。 在许多情况下，String 串联可能执行效果更好。

示例：


  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

在应用快速修复后：


  String result = "i = " + i + ";";
  return result;
