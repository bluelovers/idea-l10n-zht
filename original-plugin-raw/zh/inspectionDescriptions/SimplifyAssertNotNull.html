报告检查已声明变量非 null 值的 assert 调用。

使用 !! or ?: 可使代码更简单。

该快速修复会将变量初始值设定项中的 assert 替换为 !! 或 ?: 运算符。

示例：


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

在应用快速修复后：


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }
