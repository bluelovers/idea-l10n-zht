报告所访问字段的名称不是其预期名称的 getter 或 setter 方法。 例如，当 getY() 返回 x 字段时。 这通常可能表明复制粘贴错误。

示例：

class Point {
    private int x;
    private int y;

    public void setX(int x) { // 警告：setter 'setX()' 分配字段 'y'
      this.y = x;
    }

    public int getY() { // 警告：getter 'getY()' 返回字段 'x'
      return x;
    }
  }

使用下面的复选框可报告类中字段的名称与 getter 或 setter 的名称相匹配的情况。
