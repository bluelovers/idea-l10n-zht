报告使用 catch 块屏蔽的断言语句和测试框架断言。 此类断言永远不会失败，因为将捕获并无提示地忽略 AssertionError。

示例 1：


void javaAssertion() {
  try {
    ...
    assert 1 == 2; // 断言被无提示忽略
  } catch (AssertionError e) {}
}

示例 2：


@Test
void testWithAssertJ() {
  try {
    ...
    assertThat(1).as("test").isEqualTo(2); // 断言被无提示忽略
  } catch (AssertionError e) {}
}

示例 3：


@Test
void testWithJunit() {
  try {
    ...
    assertEquals(1, 2); // 断言被无提示忽略
  } catch (AssertionError e) {}
}
