报告包含 if 语句并且以 break 结尾但又不改变语义的循环。 这可以防止冗余的循环迭代。

示例：


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

在应用快速修复后：


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
