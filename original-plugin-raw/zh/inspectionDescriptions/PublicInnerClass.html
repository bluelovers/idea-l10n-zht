报告 public 嵌套类。

示例：


  public class Outer {
    public static class Nested {} // 警告
    public class Inner {}         // 警告
    public enum Mode {}           // 警告取决于设置
    public interface I {}         // 警告取决于设置
  }

配置检查：

  • 使用忽略 'public' 内部枚举选项可忽略 public 内部枚举。
  • 使用忽略 'public' 内部接口选项可忽略 public 内部接口。
