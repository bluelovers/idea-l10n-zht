报告传递始终为 null 或始终为非 null 实参的 Optional.ofNullable() 用法。 在这种情况下使用 Optional.ofNullable() 没有意义：Optional.empty() 或 Optional.of() 应分别用于显式声明创建始终为空或始终非空的可选项的意图。 也可能是 Optional.ofNullable() 实参中存在错误，因此应对其进行检查。

示例： 


Optional<String> empty = Optional.ofNullable(null); // 应为 Optional.empty();
Optional<String> present = Optional.ofNullable("value"); // 应为 Optional.of("value");
