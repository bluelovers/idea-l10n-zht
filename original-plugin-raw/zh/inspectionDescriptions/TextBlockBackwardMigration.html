报告可以替换为常规字符串文字的文本块。

在代码迁回 Java 15 以前的 Java 版本时，可能会出现此类文本块。

示例：


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

在应用快速修复后：


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

仅当项目或模块的语言级别为 15 或更高时，此检查才会报告。
