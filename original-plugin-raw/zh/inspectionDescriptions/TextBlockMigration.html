报告可通过替换为文本块来简化的 String 串联。

要求：

  • \n 发生两次或多次。
  • 文本块未串联。

使用应用于单字符串文字选项，可建议修正包含换行符的单一文字。

示例：


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

在应用快速修复后：


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;

仅当项目或模块的语言级别为 15 或更高时，此检查才会报告。
