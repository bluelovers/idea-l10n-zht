报告比较 .length 或 .size，并建议将其替换为 .sizeIs。

示例：


  var seq: Seq[Int]
  seq.length == n
  seq.length != n
  seq.length < n
  seq.length > n

  seq.size == n
  seq.size != n
  seq.size < n
  seq.size > n

在应用快速修复后：


  var seq: Seq[Int]
  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n

  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n
