报告有三个或以上分支并且可被替换为带主体的 when 表达式的 if 语句。

示例：


  fun translateNumber(n: Int): String {
    return if (n == 1) {
      "one"
    } else if (n == 2) {
      "two"
    } else {
      "???"
    }
  }

该快速修复会将 if 表达式转换为 when：


  fun translateNumber(n: Int): String {
    return when (n) {
      1 -> {
        "one"
      }
      2 -> {
        "two"
      }
      else -> {
        "???"
      }
    }
  }
