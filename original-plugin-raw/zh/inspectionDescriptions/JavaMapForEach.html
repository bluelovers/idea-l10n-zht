报告可以替换为 Kotlin 的 forEach 的 Java Map.forEach 方法调用。

示例：


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}

该快速修复会移除圆括号：


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}
