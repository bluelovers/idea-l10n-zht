创建生成时已改写 -isEqual: 方法的方法体， 例如，调用代码 | 改写方法或代码 | 生成 | -isEqual: 和 -hash 操作时。
此模板可编辑。 除了 Objective-C 表达式和注释，您可以使用预定义的变量， 然后将这些变量扩展为相应的值。
预定义变量将接收以下值：
${CALL_SUPER}   超类方法调用，用于 void 方法 — [super methodName] 带或不带参数； 对于其他方法 — return [super methodName] 带或不带参数
${RETURN_TYPE}   创建方法的返回类型
${DEFAULT_RETURN_VALUE}   方法默认返回的值
${CONTAINING_CLASS}   方法被改写的类的名称
${CUSTOM_CODE}   type-specific isEqualToType：比较方法名称（如果已生成）
C 头文件模板（包含选项卡）也可用 （${USER}、 ${DATE} 等）
