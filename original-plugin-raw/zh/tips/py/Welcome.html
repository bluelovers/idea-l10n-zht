PyCharm 2021.2 中的最新变化
  • Python 3.10: 模式匹配
  • Python 3.10: 允许将联合类型编写为 X | Y
  • 能够从 VCS 或光盘安装 Python 包
  • Markdown 中的 Doctest 注入演示 Python 代码在 REPL 模式下的执行
  • 能够将文件夹标记为测试源根
  • 自动检测安装在所选 Python 解释器上的测试运行程序
  • Pytest 改进: 管理测试输出并在 pytest 断言失败消息中交换实际/预期值
  • 支持 JSDoc 中的 TypeScript 类型
  • 自动导入 CommonJS 模块
  • 保存时重新加载浏览器页面
  • 用户界面改进: 自定义项目图标，更多保存操作
