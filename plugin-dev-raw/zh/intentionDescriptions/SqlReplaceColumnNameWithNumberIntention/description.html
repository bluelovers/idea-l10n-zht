將列名取代為位置參照。

某些 DBMS 可以通過序列號參照 ORDER BY 和 GROUP BY 中的列。 除了 Cassandra、ClickHouse、Hive、Spark 和 HSQLDB 之外，所有方言都支援 ORDER BY。 MySQL、Maria、PostgreSQL（及其子方言）、Exasol、SQLite 和 Snowflake 支援 GROUP BY。

範例（PostgreSQL）：

CREATE TABLE tab(a INT, b INT);
SELECT * FROM tab ORDER BY 1;

您可以使用此意圖動作將 SELECT 查詢中的 1 取代為列名。 因此，在套用該意圖動作後，查詢將如下所示。

SELECT * FROM tab ORDER BY a;
