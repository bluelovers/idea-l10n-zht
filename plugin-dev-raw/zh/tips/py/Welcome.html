PyCharm 2021.2 中的最新變化
  • Python 3.10: 模式符合
  • Python 3.10: 允許將聯合類型編寫為 X | Y
  • 能夠從 VCS 或光盤安裝 Python 包
  • Markdown 中的 Doctest 注入示範 Python 程式碼在 REPL 模式下的執行
  • 能夠將資料夾標記為測試源根
  • 自動檢測安裝在所選 Python 解釋器上的測試執行程序
  • Pytest 改進: 管理測試輸出並在 pytest 斷言失敗訊息中交換實際/預期值
  • 支援 JSDoc 中的 TypeScript 類型
  • 自動匯入 CommonJS 模組
  • 儲存時重新載入瀏覽器頁面
  • 使用者介面改進: 自訂專案圖標，更多儲存操作
