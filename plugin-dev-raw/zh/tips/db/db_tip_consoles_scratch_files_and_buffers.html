資料庫主控台 是已附加到資料來源的 SQL 檔案。建立資料來源時，會自動建立資料庫主控台。要開啟主控台，請右鍵點擊資料來源並選擇 新增 | 查詢主控台。

臨時檔案 是未附加到資料來源的 SQL 檔案。對於專案上下文之外的一些程式碼想法，將臨時檔案稱為臨時註釋或草稿。要建立臨時檔案，請導覽至 檔案 | 新增 | 臨時 (或按 &shortcut:NewScratchFile;)。

臨時緩衝區 是簡單的文本檔案，沒有任何編碼輔助功能。臨時緩衝區可用於簡單的任務列表和給自己的筆記。沒有專用的選單項用於建立新的臨時緩衝區，但您可以使用 尋找動作 彈出視窗(&shortcut:GotoAction; )並執行 新增臨時緩衝區 動作。
