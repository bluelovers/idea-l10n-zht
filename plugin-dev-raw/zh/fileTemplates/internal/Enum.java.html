適用於通過在專案工具視窗中調用新建 | Java 類 | 枚舉創建的新 Java 枚舉。

此內建模板是可編輯的。 除了 Java 表達式和註釋外，您還可以使用預定義變數（如下所列），這些變數隨後將像巨集一樣被擴展為相應值。

您也可以指定自訂變數。 自訂變數使用以下格式：${VARIABLE_NAME}，其中 VARIABLE_NAME是變數的名稱（例如，${MY_CUSTOM_FUNCTION_NAME}） 。 在 IDE 創建包含自訂變數的新檔案之前，您會看到一個對話框，您可以在其中定義模板中自訂變數的值。

通過使用 #parse 指令，可以包括 包含 頁籤中的模板。 要包含模板，請在引號中將模板的全名指定為參數（例如，#parse("File Header.java")）。
預定義變數接收以下值：
${PACKAGE_NAME}   在其中創建新枚舉的軟體套件名稱
${NAME}   您在創建新類對話框中指定的新枚舉的名稱
${USER}   當前使用者的系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母（Jan、Feb 等）
${MONTH_NAME_FULL}   當前月份的全名（January、February 等）。
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PROJECT_NAME}   當前專案名稱
