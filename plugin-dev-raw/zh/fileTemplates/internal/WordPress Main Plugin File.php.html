一個 WordPress 主延伸模組檔案的內建模板。 每次建立 WordPress 延伸模組類型的新項目時都會使用它。
模板是可編輯的。 除了 PHP 語句、表達式和註釋外，您還可以使用下面列出的預定義變數。 每個變數都像巨集一樣擴展為相應的值。
通過 #parse 指令，可以通過將所需模板的全名指定為引號中的參數，從而包含 Includes 頁籤中的模板。
預定義變數接收以下值：
${PLUGIN_NAME}   派生自目錄名稱的當前延伸模組名稱
${FILE_NAME}   當前檔名
${USER}   當前的使用者系統登入名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PRODUCT_NAME}   當前 IDE 名稱
${PROJECT_NAME}   當前專案名稱
${DS}   美元符號，計算為普通的 “$” 字元。
