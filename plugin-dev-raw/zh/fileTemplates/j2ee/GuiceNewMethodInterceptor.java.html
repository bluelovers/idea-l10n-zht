這是您每次建立新的 Guice 方法偵聽器類別時使用的內建模板。
使用 #parse 指令，將所需模板的全名指定為引號中的參數可以在包含頁籤中包含模板。 例如：
#parse("File Header.java")
預定義變數接收以下值：
${PACKAGE_NAME}   在其中建立新類別的軟體套件名稱
${NAME}   在建立新類別對話框中指定的新類別的名稱
${USER}   當前的使用者系統登入名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母。 例如：Jan、Feb 等
${MONTH_NAME_FULL}   當前月份的全名。 例如：January、February 等
${DAY}   當前日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PROJECT_NAME}   當前專案的名稱
