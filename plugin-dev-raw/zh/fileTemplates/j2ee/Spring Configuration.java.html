適用於通過在專案工具視窗中呼叫新增 | Spring 組件 | 組態建立的新 Spring 組態。
此內建模板是可編輯的。 除了 Java 表達式和註釋外，您還可以使用預定義變數（如下所列），這些變數隨後將像巨集一樣被展開為相應值。

您也可以指定自訂變數。 自訂變數使用以下格式：${VARIABLE_NAME}，其中 VARIABLE_NAME 是變數的名稱（例如，${MY_CUSTOM_FUNCTION_NAME}）。 在 IDE 使用自訂變數建立新檔案之前，您會看到一個對話框，您可以在其中定義模板中自訂變數的值。

通過使用 #parse 指令，您可以包括 Include 頁籤中的模板。 要包含模板，請在引號中以參數形式指定模板的全名（例如，#parse("File Header.java")）。
預定義變數接受以下值：
${PACKAGE_NAME}   在其中建立新類別的軟體套件名稱
${NAME}   您在建立新類別對話框中指定的新類別的名稱
${USER}   當前使用者的系統登入名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年份
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母（Jan、Feb 等）
${MONTH_NAME_FULL}   當前月份的全名（January、February 等）。
${DAY}   當月幾號
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PROJECT_NAME}   當前專案的名稱
