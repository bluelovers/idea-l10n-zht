在此模板中，您可以借助 #parse 指令指定要包含在檔案模板（檔案頁籤）中 的程式碼片段。 一個模板可以由多個檔案模板共享。 除了RBS 語句、表達式和註釋外，您還可以使用預定義變數，這些變數隨後將像巨集一樣被擴展為相應值。
預定義變數接收以下值：
${FILE_NAME}   當前文件名稱
${USER}   當前的使用者系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PRODUCT_NAME}   當前 IDE 名稱
${PROJECT_NAME}   當前專案名稱
