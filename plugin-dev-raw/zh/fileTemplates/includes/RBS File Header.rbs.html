在此模板中，您可以借助 #parse 指令指定要包含在文件模板（文件選項卡）中 的代碼片段。 一個模板可以由多個文件模板共享。 除了RBS 語句、表達式和注釋外，您還可以使用預定義變量，這些變量隨後將像宏一樣被擴展為相應值。
預定義變量接收以下值：
${FILE_NAME}   當前文件名稱
${USER}   當前的用戶系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PRODUCT_NAME}   當前 IDE 名稱
${PROJECT_NAME}   當前項目名稱
