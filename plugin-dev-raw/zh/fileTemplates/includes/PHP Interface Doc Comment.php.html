一個 PHP 接口的 PHPDoc 註釋的內建模板。 它包含一個程式碼片段，可以在 #parse 指令的幫助下將其包含在檔案模板（Templates 標籤）中。
模板是可編輯的。 除了靜態文本、程式碼和註釋外，您還可以使用預定義變數，這些變數將像巨集一樣被擴展為相應值。
預定義變數接收以下值：
${NAME}   接口名稱。
${NAMESPACE}   接口命名空間的完全限定名稱，沒有前導斜杠。
${DS}   美元符號，計算為普通的 “$” 字元。
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母。 例如：Jan、Feb 等
${MONTH_NAME_FULL}   當前月份的全名。 例如：January、February 等
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${SECOND}   當前秒
${PROJECT_NAME}   當前專案名稱
${USER}   當前使用者
${CARET}   標記新增註釋後文字游標必須移動到的位置。
