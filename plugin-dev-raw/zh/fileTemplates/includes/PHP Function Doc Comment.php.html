一個 PHP 函數的 PHPDoc 註釋的內建模板。 它包含一個程式碼片段，可以在 #parse 指令的幫助下將其包含在檔案模板（Templates 標籤）中。
模板是可編輯的。 除了靜態文本、程式碼和註釋外，您還可以使用預定義變數，這些變數將像巨集一樣被擴展為相應值。
預定義變數接收以下值：
${NAME}   函數名稱。
${TYPE_HINT}   函數返回值的類型提示。 如果沒有推斷出返回類型，則評估為 'void'。
${PARAM_DOC}  

參數的 PHPDoc 註釋。

生成為一系列行 '* @param type name'。 如果沒有參數，評估為空內容。

${THROWS_DOC}  

異常的文檔註釋。

生成為一系列行 '* @throws type'。 如果沒有拋出異常，評估為空內容。

${STATIC}   如果函數/方法是靜態的，則取值為 'static'，否則為空字串。 例如： 
      #if (${STATIC} == "static") * @static
      #end
${DS}   美元符號，計算為普通的 “$” 字符。
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母。 例如：Jan、Feb 等
${MONTH_NAME_FULL}   當前月份的全名。 例如：January、February 等
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${SECOND}   當前秒
${PROJECT_NAME}   當前專案名稱
${USER}   當前使用者
${CARET}   標記添加註釋後文字游標必須移動到的位置。
