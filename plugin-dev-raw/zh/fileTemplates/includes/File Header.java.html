包含一個代碼片段，可以在 #parse 指令的幫助下將其包含在文件模板（Templates 選項卡）中。
此內置模板是可編輯的。 除了靜態文本、代碼和注釋外，您還可以使用預定義變量，這些變量隨後將像宏一樣被擴展為相應值。
預定義變量接收以下值：
${PACKAGE_NAME}   在其中創建新文件的軟件包名稱
${USER}   當前的用戶系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母。 例如：Jan、Feb 等
${MONTH_NAME_FULL}   當前月份的全名。 例如：January、February 等
${DAY}   當前月份的日期
${DAY_NAME_SHORT}   當前日期名稱的前 3 個字母。 例如：Mon、Tue 等
${DAY_NAME_FULL}   當前日期的全名。 例如：Monday、Tuesday 等
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PROJECT_NAME}   當前項目名稱
