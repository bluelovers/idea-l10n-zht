在生成方法（例如，套用從用法創建方法意圖操作）時填充方法體。
此內建模板是可編輯的。 除了 Java 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數接收以下值：
${RETURN_TYPE}   創建方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${METHOD_NAME}   創建的方法的名稱
${CLASS_NAME}   創建方法的類的限定名稱
${SIMPLE_CLASS_NAME}   實作方法的類的非限定名稱
所有來自 檔案頭 模板（包含頁籤）的預定義變數也可用。 （例如： ${USER}、 ${DATE} 等。）
