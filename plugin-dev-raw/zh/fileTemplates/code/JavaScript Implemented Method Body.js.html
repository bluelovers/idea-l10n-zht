此內置模板用於在每次生成方法（例如，通過從用法創建方法意圖操作）時填充方法體。
模板是可編輯的。 除了 Javascript 表達式和注釋，您還可以使用預定義變量。 當模板被應用時，這些變量將被替換成相應的值。
預定義變量接收以下值：
${IS_ASSIGNMENT_TARGET}   如果創建的方法是賦值目標，則返回 'true'。
${IS_NEW_CALL}   如果創建的方法是在 ‘new’ 調用下，則返回 ‘true’。
${HAS_EXPECTED_TYPE}   如果創建的方法在應為類型的位置，則返回 'true'。
${DEFAULT_RETURN_VALUE}   創建方法的預期類型的默認值（如果沒有預期類型，則為 'undefined'）
${METHOD_NAME}   創建的方法名稱
所有來自 文件頭 模板（包含選項卡）的預定義變量也可用。 （例如： ${USER}、 ${DATE} 等。）
