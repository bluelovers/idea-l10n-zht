這是內建模板，用於將 GSP 中的文本 例如：
     <html> <body> Hello! </body> </html>
取代為國際化 GSP 自訂標記用法， 例如：
     <html> <body> <g:message code="HelloKey"/> </body> </html> 。

模板是可編輯的。 除了 GSP 上下文中允許的表達式外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。

預定義變數接收以下值：
${PROPERTY_KEY}   在相應的屬性檔案中指定的屬性鍵名稱。
${PROPERTY_VALUE}   在相應的屬性檔案中指定的屬性值。 這是一個原始的 GSP 文本選擇值。
