在生成被覆寫下標（例如，使用程式碼 | 生成 | 覆寫方法操作）時填充下標的主體。
模板是可編輯的。 除了 Swift 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數接收以下值：
${CALL_SUPER}   super 下標調用 - 帶參數的 return super[]
${RETURN_TYPE}   創建的下標的返回類型
${DEFAULT_RETURN_VALUE}   下標預設返回的值
${SIGNATURE}   下標簽名
${HAS_SET}   當下標有 setter 時，為 "true"
${HAS_SUPER_SET}   當被覆寫下標有 setter 時，為 "true"
${GETTER_EFFECTS}   屬性 getter 的 getter 效果（async、throws）
${GET_ATTRIBUTES}   下標 getter 的特性
${SET_ATTRIBUTES}   下標 setter 的特性
Swift 檔案頭模板（包含頁籤）中的所有預定義變數也可用（例如 ${USER}、${DATE} 等）。
