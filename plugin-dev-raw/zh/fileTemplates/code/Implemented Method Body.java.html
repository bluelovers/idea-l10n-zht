在生成被實作的方法（例如，套用實作 abstract 類別或實作 abstract 方法意圖動作或呼叫程式碼 | 實作方法… 功能）時填充方法體。
此內建模板是可編輯的。 除了 Java 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數接收以下值：
${RETURN_TYPE}   建立方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${METHOD_NAME}   實作的方法的名稱
${CLASS_NAME}   實作方法的類別的限定名稱
${SIMPLE_CLASS_NAME}   實作方法的類別的非限定名稱
檔案頭模板（Includes 頁籤）中的所有預定義變數也可用（例如 ${USER}、${DATE} 等）。
