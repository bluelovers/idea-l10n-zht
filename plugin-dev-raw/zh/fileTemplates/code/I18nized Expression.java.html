將字串文字表達式（如 String s = "Hello";） 取代為國際化的 Java 程式碼表達式 （如 String s = myResourceBundle.getString("hello");）。
此內建模板是可編輯的。 除了 Java 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數接收以下值：
${RESOURCE_BUNDLE}   java.util.ResourceBundle 類型的表達式，這一類型的實例在此上下文中可用。
${PROPERTY_KEY}   在相應的屬性檔案中指定的屬性鍵名稱。 通常，它是用於 ResourceBundle.getString() 方法參數的值。
${PROPERTY_VALUE}   在相應的屬性檔案中定義的屬性值。 這就是原始的 Java 字串字面值。
