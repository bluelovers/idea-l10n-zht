創建生成時已實作 -description 方法的方法體， 例如，調用程式碼 | 生成 | -description 操作時。
此模板可編輯。 除了 Objective-C 表達式和註釋，您可以使用預定義的變數， 然後將這些變數擴展為相應的值。
預定義變數將接收以下值：
${RETURN_TYPE}   創建方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${CONTAINING_CLASS}   方法被實作的類的名稱
${CUSTOM_CODE}   生成的方法的方法體
C 檔案頭模板（包含頁籤）的所有預訂變數也可用 （例如， ${USER}、 ${DATE} 等）
