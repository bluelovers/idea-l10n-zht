一個 PHP setter 方法的內建模板。 例如，當呼叫 "新增 getter 和 setter" 快速修復時，就會用到它。
預定義變數接收以下值：
${STATIC}   如果屬性是靜態的，則取值為 'static'，否則為空字串。
${NAME}   使用駝峰式命名法樣式的方法名，沒有 "set" 前綴，第一個字元大寫。 例如，屬性名稱 '_my_property' 被 轉換為 'MyProperty'。
${CLASS_NAME}   包含類別名稱。
${SCALAR_TYPE_HINT}   如果語言級別為 PHP 7.0 或更高版本，則包含標量類型提示
${FIELD_NAME}   原始屬性名稱保持原樣。
${PARAM_NAME}   setter 參數名稱。 幾乎與 ${FIELD_NAME} 相同， 但刪除了第一個下劃線（如果有的話）。 例如，屬性名稱 '_myProperty' 被 轉換為 'myProperty'。
${TYPE_HINT}   屬性類型提示，如果未指定，則為空字串。
${VOID_RETURN_TYPE}   指示 void 返回類型是否可用於專案中的 PHP 語言級別。
${DS}   美元符號，計算為普通的 “$” 字元。
