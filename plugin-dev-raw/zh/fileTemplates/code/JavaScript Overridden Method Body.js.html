此內建模板用於在每次生成被覆寫方法（例如，調用程式碼 | 覆寫方法… 功能）時填寫方法體。
模板是可編輯的。 除了 Javascript 表達式和註釋，您還可以使用預定義變數。 當模板被應用時，這些變數將被取代成相應的值。
預定義變數接收以下值：
${CALL_SUPER}   super 方法調用，對於 void 方法 - super.method_name(); 帶或不帶參數； 對於其他方法 - return super.method_name(); 帶或不帶參數
${PLAIN_CALL_SUPER}   super 方法調用，super.method_name(); 帶或不帶參數；
${RETURN_TYPE}   創建方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${METHOD_NAME}   被覆寫的方法的名稱
所有來自 檔案頭 模板（包含頁籤）的預定義變數也可用。 （例如： ${USER}、 ${DATE} 等。）
