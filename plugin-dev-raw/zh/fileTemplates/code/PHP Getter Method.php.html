一個 PHP getter 方法的內建模板。 例如，當調用 "添加 getter 和 setter" 快速修復時，就會用到它。
預定義變數接收以下值：
${STATIC}   如果屬性是靜態的，則取值為 'static'，否則為空字串。
${GET_OR_IS}   'get/is' 前綴。 如果一個屬性有一個布林型態（類型提示），則評估為 'is'，否則評估為 'get' 字串。
${NAME}   使用駝峰式命名法樣式的方法名，沒有 "get" 前綴，第一個字符大寫。 例如，屬性名稱 '_my_property' 被轉換為 'MyProperty'。
${CLASS_NAME}   包含類名稱。
${RETURN_TYPE}   如果語言級別為 PHP 7.0 或更高版本，則包含返回類型提示
${FIELD_NAME}   原始屬性名稱保持原樣。
${TYPE_HINT}   屬性類型提示，如果未指定，則為空字串。
${DS}   美元符號，計算為普通的 “$” 字符。
