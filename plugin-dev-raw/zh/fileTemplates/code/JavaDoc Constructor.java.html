定義用於為構造函數生成 JavaDoc 的模板，例如套用添加 Javadoc 意圖操作時。

除了 Java 表達式和註釋，您還可以使用預定義變數，這些變數隨後將像巨集一樣被擴展為相應值。

通過使用 #parse 指令，可以包括 Includes 頁籤中的模板。 要包含模板，請在引號中將模板的全名指定為參數（例如，#parse("File Header.java")）。
預定義變數接收以下值：
${PARAMS}   參數名稱列表
${TYPE_PARAMS}   類型參數名稱列表
${THROWS}   拋出的異常列表
${CONTAINING_CLASS}   父類的名稱
${PACKAGE_NAME}   在其中創建新枚舉的軟體套件名稱
${NAME}   您在創建新類對話框中指定的新枚舉的名稱
${USER}   當前使用者的系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母（Jan、Feb 等）
${MONTH_NAME_FULL}   當前月份的全名（January、February 等）。
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PROJECT_NAME}   當前專案名稱
