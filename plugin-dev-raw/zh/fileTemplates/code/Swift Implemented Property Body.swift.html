在生成已實作屬性（例如，使用程式碼 | 生成 | 實作方法動作）時填充屬性的主體。
模板是可編輯的。 除了 Swift 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數接收以下值：
${RETURN_TYPE}   建立的屬性的類型
${DEFAULT_RETURN_VALUE}   返回值的建議預設值（可以為空）
${SIGNATURE}   屬性簽名
${HAS_SET}   當屬性有 setter 時，為 "true"
${GETTER_EFFECTS}   屬性 getter 的 getter 效果（async、throws）
${GET_ATTRIBUTES}   屬性 getter 的特性
${SET_ATTRIBUTES}   屬性 setter 的特性
Swift 檔案頭模板（包含頁籤）中的所有預定義變數也可用（例如 ${USER}、${DATE} 等）。
