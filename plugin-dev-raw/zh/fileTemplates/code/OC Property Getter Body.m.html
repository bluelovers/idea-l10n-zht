創建生成時已改寫屬性的 getter 方法的方法體， 例如，調用程式碼 | 改寫方法操作時。
此模板可編輯。 除了 Objective-C 表達式和註釋，您可以使用預定義的變數， 然後將這些變數擴展為相應的值。
預定義變數將接收以下值：
${CALL_SUPER}   超類方法調用，用於 void 方法 — [super methodName] 帶或不帶參數； 對於其他方法 — return [super methodName] 帶或不帶參數
${RETURN_TYPE}   創建方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${CONTAINING_CLASS}   方法被改寫的類的名稱
${IVAR_IS_AVAILABLE}   當屬性有實例變數 時，為 "true"
${IVAR}   實例變數的名稱
${SETTER_SEMANTICS}   在 ARC 模式下：已設定 copy 屬性時，為 "copyArc"，否則為 "assign"
在非 ARC 模式下：已設定 copy 屬性時，為 "copy"，已設定 retain 屬性時，為 "retain"，否則為 "assign"
C 標頭檔模板（包含頁籤）也可用 （${USER}、 ${DATE} 等）
