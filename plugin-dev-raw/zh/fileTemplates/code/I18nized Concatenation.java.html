將字符串串聯（如 String s = "Hello " + name;） 替換為國際化的 Java 代碼表達式（如 String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);）。
此內置模板是可編輯的。 除了 Java 表達式和注釋外，您還可以使用預定義變量，這些變量隨后將被擴展為相應值。
預定義變量接收以下值：
${RESOURCE_BUNDLE}   java.util.ResourceBundle 類型的表達式，這一類型的實例在此上下文中可用。
${PARAMETERS}   字符串串聯中使用的變量，通常作為實參傳遞給國際化表達式。
${PROPERTY_KEY}   在相應的屬性文件中定義的屬性鍵名稱。 通常，它是用于 ResourceBundle.getString() 方法形參的值。
${PROPERTY_VALUE}   在相應的屬性文件中定義的屬性值。 這就是原始的 Java 字符串字面值。
