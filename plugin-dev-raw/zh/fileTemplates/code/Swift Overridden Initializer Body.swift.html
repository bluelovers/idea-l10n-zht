Fills the body of an overridden initializer when it is generated, for example, with the Code | Generate | Override Methods action.
該模板可編輯。 除了 Swift 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數將接受以下值：
${CALL_SUPER}   a super initializer call - super.init() with or without parameters
${SIGNATURE}   方法的簽名
Swift 檔案頭模板（Include 頁籤）中的所有預定義變數也可用（例如 ${USER}、${DATE} 等）。
