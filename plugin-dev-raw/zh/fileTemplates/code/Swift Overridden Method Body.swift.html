在生成被覆寫方法（例如，使用程式碼 | 生成 | 覆寫方法動作）時填充方法體。
模板是可編輯的。 除了 Swift 表達式和註釋外，您還可以使用預定義變數，這些變數隨後將被擴展為相應值。
預定義變數接收以下值：
${CALL_SUPER}   super 方法呼叫，用於 Void 方法：帶或不帶參數的 super.methodName()； 對於其他方法：帶或不帶參數的 return super.methodName()
${RETURN_TYPE}   建立的方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${SIGNATURE}   方法簽名
Swift 檔案頭模板（包含頁籤）中的所有預定義變數也可用（例如 ${USER}、${DATE} 等）。
