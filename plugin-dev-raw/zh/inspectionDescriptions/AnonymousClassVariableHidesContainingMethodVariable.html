報告匿名類中名稱與包含方法或 lambda 表達式的局部變數或參數相同的欄位。

由於進行此類命名，在包含方法中有意使用同名變數或參數的情況下，您可能會誤用匿名類欄位。

建議通過快速修復重命名該欄位。

範例：


  class Test {
    public Test(String value) {
      Object foo = new Object() {
        private String value = "TEST";
        public void foo() {
          System.out.println(value); //存取的是欄位，而不是參數
        }
      };
    }
  }
