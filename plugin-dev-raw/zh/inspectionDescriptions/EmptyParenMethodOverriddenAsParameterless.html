報告沒有參數子句且重寫帶有空參數子句的方法的方法。

如果一個方法有副作用，則在方法中使用圓括號是一個常見的慣例。 根據里氏取代原則，當被重寫的方法有參數子句時，表明有副作用，重寫方法也必須被聲明為有副作用的方法。

該快速修復會添加一個空參數子句。

示例：


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x = 2
  }

在應用快速修復後：


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x() = 2
  }
