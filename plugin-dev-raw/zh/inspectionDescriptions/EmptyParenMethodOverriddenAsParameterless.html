報告沒有參數子句且覆寫帶有空參數子句的方法的方法。

如果一個方法有副作用，則在方法中使用圓括號是一個常見的約定。 根據里氏取代原則，當被覆寫的方法有參數子句時，表明有副作用，覆寫方法也必須被宣告為有副作用的方法。

該快速修復會新增一個空參數子句。

範例：


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x = 2
  }

在套用快速修復後：


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x() = 2
  }
