報告 "reactive" 方法中的 subscribe() 調用。

返回發佈程序(包括 Uni 和 Multi)的方法不應直接調用 subscribe() 方法，因為它可能破壞反應式調用鏈。

不要使用 subscribe()，而是考慮使用組合運算符，例如 flatMap()、merge() 等。

示例：


  Multi<String> stringMulti(){
    Multi<String> multi = Multi.createFrom().item("abc");
    multi.subscribe(); // <- 在非阻塞上下文中阻塞 'subscribe' 調用
    return multi;
  }
