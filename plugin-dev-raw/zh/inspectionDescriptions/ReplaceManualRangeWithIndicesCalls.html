報告可以取代為 Collection.indices 或 for 迴圈內集合迭代的 until 和 rangeTo 運算符。

使用語法糖可使程式碼更簡單。

該快速修復會將手動範圍取代為相應的建構。

範例：


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

在套用快速修復後：


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
