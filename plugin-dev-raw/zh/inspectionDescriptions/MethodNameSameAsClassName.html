報告與其類同名的方法。

雖然 Java 語言允許這樣的命名，但它通常表明有錯誤，因為其本意是創建構造函數。 這樣的命名是不良做法。

範例:


  class MyClass {
    int val;

    // 方法 MyClass 的名稱與其包含類相同。
    // 可能是錯誤添加了 'void'。
    void MyClass(int val) {
      this.val = val;
    }
  }

在適當的情況下，通過快速修復將方法轉換為構造函數:


  class MyClass {
    int val;

    MyClass(int val) {
      this.val = val;
    }
  }

通過另一種快速修復重命名該方法。
