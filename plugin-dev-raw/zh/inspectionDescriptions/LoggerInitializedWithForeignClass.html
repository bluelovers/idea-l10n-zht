報告用來初始化的 class 文字所屬的類與 Logger 所屬的類不同的 Logger 實例。 從另一個類複製貼上某些程式碼時，很容易發生這種情況，並可能導致在意外類別下記錄事件，並導致篩選器套用不正確。

提供了一個快速修復來用周圍類中的文字取代外類文字。

範例：


  public class Paramount {
    protected static final Logger LOG = Logger.getLogger(Critical.class);

    // …其他欄位和方法
  }

在套用快速修復後：


  public class Paramount {
    protected static final Logger LOG = Logger.getLogger(Paramount.class);

    // …其他欄位和方法
  }

設定檢查：

  • 使用該表可以指定此檢查識別的記錄器工廠類和記錄器工廠方法。
  • 使用忽略使用超類初始化的記錄器選項可以忽略使用包含該記錄器的類的超類初始化的記錄器。
  • 使用忽略非 public 類中的記錄器，僅對 public 類中的記錄器發出警告。
