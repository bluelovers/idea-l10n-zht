報告以下 EJB 規範違規：
  • 類別未實作預期接口（EJBLocalObject、EJBObject 等）
  • 類別未定義 public 無實參建構函式
  • 使用類別而不是接口或者使用接口而不是類別
  • EJB 方法為 static
  • 無狀態會話 Bean 類別定義了多個 `ejbCreate` 方法
  • 會話 Bean 類別的方法返回類型不正確
  • EJB 定義了 `finalize` 方法

範例：


public class SessionEJB implements java.io.Serializable { // 錯誤：類別應該擴展 `SessionBean`
  private SessionEJB() {} // 錯誤：類別應該定義 public 無實參建構函式
}
