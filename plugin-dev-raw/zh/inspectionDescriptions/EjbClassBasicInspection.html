報告以下 EJB 規範違規：
  • 類沒有實作預期的接口（EJBLocalObject、EJBObject 等）
  • 類未定義 public 無參數構造函數
  • 使用類而不是接口，反之亦然
  • EJB 方法為 static
  • 無狀態會話 Bean 類定義了幾個 `ejbCreate` 方法
  • 會話 Bean 類的方法返回類型不正確
  • EJB 定義了 `finalize` 方法

範例:


public class SessionEJB implements java.io.Serializable { // 錯誤：類應該擴展 `SessionBean`
  private SessionEJB() {} // 錯誤：類應該定義 public 無參數構造函數
}
