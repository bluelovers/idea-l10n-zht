報告使用 new 或 getOrCreate 函數在 ChangeScope 中創建時 Entity 的未分配屬性。

必須分配 Entity 的所有非可 null 屬性，或者使用 @Many 註解對其進行標記。 否則，可能會導致執行時異常。

範例：


  interface A: Entity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }

  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

可以通過快速修復來為缺失的必需屬性生成賦值


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }
