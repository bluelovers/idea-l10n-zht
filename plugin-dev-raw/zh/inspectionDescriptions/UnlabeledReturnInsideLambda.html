報告內聯 lambda 內未標記的 return 表達式。

此類表達式可能令人困惑，因為可能不清楚哪個範圍屬於 return。

更改為 return@… 快速修復可用於自動修正程式碼。

範例：


  fun test(list: List<Int>) {
      list.forEach {
          // 此返回表達式從函數測試返回
          // 可以將其更改為 return@forEach 以更改範圍
          if (it == 10) return
      }
  }

在套用快速修復後：


  fun test(list: List<Int>) {
      list.forEach {
          if (it == 10) return@test
      }
  }
