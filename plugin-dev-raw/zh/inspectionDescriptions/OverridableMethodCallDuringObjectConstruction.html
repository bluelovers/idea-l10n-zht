報告在物件建構期間對當前類別的可覆寫方法的呼叫。

如果方法位於以下位置，則會在物件建構期間呼叫：

  • 建構函式
  • 非 static 實例初始設定式
  • 非 static 欄位初始設定式
  • clone() 方法
  • readObject() 方法
  • readObjectNoData() 方法

    • 如果方法未被宣告為 final、static 或 private，則該方法可覆寫。 即使可覆寫，package-local 方法也被認為是安全的。 此類別呼叫可能會導致難以捉摸的錯誤，因為物件初始化可能發生在方法呼叫之前。

    範例：

    
  class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }
  }

    此檢查與以下檢查共享功能：

    • 物件建構期間呼叫的 abstract 方法
    • 在物件建構期間呼叫的覆寫的方法

    一次只能啟用一個檢查，以防止警告重複。
