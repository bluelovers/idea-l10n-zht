報告過強的類型轉換。 例如，將物件轉換為 ArrayList，將其轉換為 List 也可以。

注意：與冗餘類型轉換檢查非常相似，如果您有意使用過強的轉換來生成 ClassCastException，則對此檢查套用修正可能會改變程序的語意。

範例：


  interface Super {
    void doSmth();
  }
  interface Sub extends Super { }

  void use(Object obj) {
    // 警告：可以使用 ((Super)obj).doSmth()
    ((Sub)obj).doSmth();
  }

在程式碼中有符合的 instanceof 檢查時，使用下面的複選框可忽略轉換。
