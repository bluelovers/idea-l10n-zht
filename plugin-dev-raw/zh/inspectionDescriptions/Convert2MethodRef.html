報告可以取代為方法引用的 lambda。 方法引用比 lambda 更容易理解和閱讀，儘管它們通常取決於您的喜好。

示例：


  Runnable r = () -> System.out.println();

在應用快速修復後：


  Runnable r = System.out::println;

即使 lambda 不調用任何方法，該檢查也可能會建議方法引用，例如將 obj -> obj != null 取代為 Objects::nonNull。 使用設定 | 編輯器 | 程式碼樣式 | Java | 程式碼生成設定來配置特殊方法引用。
