報告有三個或以上分支並且可被取代為帶主體的 when 表達式的 if 語句。

範例：


  fun translateNumber(n: Int): String {
    return if (n == 1) {
      "one"
    } else if (n == 2) {
      "two"
    } else {
      "???"
    }
  }

該快速修復會將 if 表達式轉換為 when：


  fun translateNumber(n: Int): String {
    return when (n) {
      1 -> {
        "one"
      }
      2 -> {
        "two"
      }
      else -> {
        "???"
      }
    }
  }
