報告覆寫 equals() 方法但不覆寫 hashCode() 方法的類別，反之亦然。將類別新增到 Collection 或 HashMap 時，可能會導致問題。

該快速修復會為不存在的方法生成預設實作。

範例：


class StringHolder {
  String s;

  @Override public int hashCode() {
    return s != null ? s.hashCode() : 0;
  }
}

在套用快速修復後：


class StringHolder {
  String s;

  @Override public int hashCode() {
    return s != null ? s.hashCode() : 0;
  }

  @Override
  public boolean equals(Object o) {
    if (this == o) return true;
    if (!(o instanceof StringHolder)) return false;

    StringHolder holder = (StringHolder)o;

    if (s != null ? !s.equals(holder.s) : holder.s != null) return false;

    return true;
  }
}
