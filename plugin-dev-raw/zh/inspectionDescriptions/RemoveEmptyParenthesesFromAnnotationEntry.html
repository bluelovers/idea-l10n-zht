報告註解條目中的冗餘空圓括號。

使用“移除不必要的圓括號”快速修復可清理程式碼。

範例：


  annotation class MyAnnotationA
  annotation class MyAnnotationB(val x: Int)
  annotation class MyAnnotationC(val x: Int = 10) // 存在預設值

  @MyAnnotationA() // <== 圓括號冗餘
  fun testA() {
  }

  @MyAnnotationB() // <== 缺少實參，圓括號為必選項
  fun testB() {
  }

  @MyAnnotationC() // <== 圓括號冗餘
  fun testC() {
  }
