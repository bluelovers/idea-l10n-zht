報告對另一個物件的 private 或 protected 欄位的存取。 Java 允許存取與當前物件屬於同一類別的物件的此類別欄位，但某些編碼樣式不鼓勵這種用法。 此外，這種對 private 欄位的直接存取在面向組件的架構（例如 Spring 或 Hibernate）中可能會失敗，它們期望對其他物件的所有存取都通過方法呼叫進行，以便框架可以使用代理調解存取。

範例：


  public class Base {
    protected int bar;

    void increment(Base base) {
      bar++;
      base.bar++; // warning: direct access to another object's non-public field
    }
  }
可以通過快速修復封裝該欄位。

組態檢查：

  • 使用忽略來自同一類的存取選項可忽略來自同一類的存取，只報告來自內部或外部類的存取。

    要忽略來自內部類別的存取，請使用嵌套的忽略來自內部類別的存取。

  • 使用忽略來自 'equals()' 方法的存取可忽略來自 equals() 方法的存取。
