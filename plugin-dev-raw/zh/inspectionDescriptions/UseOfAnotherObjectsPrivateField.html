報告使用另一個物件的 private 或 protected 欄位的物件。

Java 允許將此類欄位用於與當前物件屬於同一類的物件，但某些程式碼樣式不鼓勵這種用法。 此外，這種對 private 欄位的直接存取在面向組件的體系結構中可能會失敗，例如 Spring 或 Hibernate，它們期望對其他物件的所有存取都通過方法調用進行，以便框架可以使用代理來調解存取。

範例:


public class Base {
    protected int bar;

    class Inside {
        int f(Base base) {
            return base.bar;  // 警告：直接存取非 public 欄位
        }
    }
}

class SomewhereElse {
    void m(Base base) {
        base.bar = 1; // 警告：直接存取非 public 欄位
    }
}

設定檢查：

  • 使用忽略來自同一類的存取選項可忽略來自同一類的存取，只報告來自內部或外部類的存取。

    要忽略來自內部類的存取，請使用嵌套的忽略來自內部類的存取。

  • 使用忽略來自 'equals()' 方法的存取可忽略來自 equals() 方法的存取。
