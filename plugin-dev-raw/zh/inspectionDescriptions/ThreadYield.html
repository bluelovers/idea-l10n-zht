報告對 Thread.yield() 的呼叫。

yield() 的行為是不確定的並且依賴於平台，很少適合使用此方法。 它的使用應當與詳細的分析和基準測試相結合，以確保真正達到預期效果。

範例：


  public static void main(String[] args) {
    Runnable r = () -> {
      for (int i = 0; i < 10; i++) {
        System.out.println(i);
        Thread.yield();
      }
    };
    new Thread(r).start();
    new Thread(r).start();
  }
