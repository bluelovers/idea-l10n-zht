報告 null 檢查條件並建議用 Optional 鏈取代它們。

範例：


  return str == null ? "" : str.trim();

套用快速修復後：

return Optional.ofNullable(str).map(String::trim).orElse("");

雖然取代並不總是更短，但它可能有助於進一步重構（例如，將方法返回值更改為 Optional）。

請注意，當條件的非 null 分支返回 null 值時，相應的映射步驟將產生一個空的 Optional，可能會改變語意。 如果不能靜態證明保留語意，則快速修復操作名稱將包含 “（可能更改語意）” 通知，並關閉檢查高亮顯示。

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。
