報告可以取代為一般字串文字的文本塊。

在程式碼遷回 Java 15 以前的 Java 版本時，可能會出現此類文本塊。

範例：


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

在套用快速修復後：


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

僅當專案或模組的語言級別為 15 或更高時，此檢查才會報告。
