  • AttributeOverride
  • AssociationOverride
  • MapsId
  • MapKey
  • Criteria API methods

範例:



  @Entity
  @AssociationOverrides({
    @AssociationOverride(name = "missingAttribute", joinColumns = { ... }), // 錯誤：未解析的屬性
    @AssociationOverride(name = "existingAttribute", joinColumns = { ... })
  })
  public class JavaEntity {
    @OneToOne
    AnotherEntity existingAttribute;
  }
