報告以下 JPA 註解和 API 方法中的未解析符號：
  • AttributeOverride
  • AssociationOverride
  • MapsId
  • MapKey
  • Criteria API 方法

範例：



  @Entity
  @AssociationOverrides({
    @AssociationOverride(name = "missingAttribute", joinColumns = { ... }), // 錯誤：未解析的特性
    @AssociationOverride(name = "existingAttribute", joinColumns = { ... })
  })
  public class JavaEntity {
    @OneToOne
    AnotherEntity existingAttribute;
  }
