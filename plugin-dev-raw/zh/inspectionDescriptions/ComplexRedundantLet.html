報告基於實參的冗餘 let 調用。

當 lambda 參數僅用作調用表達式中的限定符時，let 冗餘。

如果需要為限定表達式命名，請宣告一個局部變數。

範例：


  fun splitNumbers() {
      "1,2,3".let { it.split(',') }
  }

該快速修復會移除額外的 let() 調用：


  fun example() {
      "1,2,3".split(',')
  }

或者：


  fun splitNumbers() {
      val numbers = "1,2,3"
      numbers.split(',')
  }
