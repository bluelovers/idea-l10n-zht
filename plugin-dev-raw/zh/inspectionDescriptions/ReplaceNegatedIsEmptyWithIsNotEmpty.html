報告集合與 String 的否定 isEmpty() 和 isNotEmpty()，或者 String 的 isBlank() 和 isNotBlank()。

使用相應函數可使程式碼更簡單。

該快速修復會將否定調用取代為標準庫中的相應調用。

範例：


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // 執行 smth
      }
  }

在套用快速修復後：


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // 執行 smth
      }
  }
