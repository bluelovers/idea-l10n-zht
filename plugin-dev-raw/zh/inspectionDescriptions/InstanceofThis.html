報告在其中檢查 this 表達式的 instanceof 或 getClass() == SomeClass.class 的用法。

此類表達式表明物件導向的設計失敗，應該取代為多態構造。

範例：


class Super {
  void process() {
    if (this instanceof Sub) { // 警告
      doSomething();
    } else {
      doSomethingElse();
    }
  }
}
  
class Sub extends Super {}

要修正該問題，可以使用覆寫方法：


class Super {
  void process() {
    doSomethingElse();
  }
}
  
class Sub extends Super {
  @Override
  void process() {
    doSomething();
  }
}
