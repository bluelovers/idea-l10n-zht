報告 javac 編譯器發出未檢查的警告的程式碼。 每個未檢查的警告都可能在執行時觸發 ClassCastException。

示例：


  List items = Arrays.asList("string", "string");
  List<Integer> numbers = Collections.unmodifiableList(items); // 未檢查的賦值

該快速修復會嘗試泛化該包含檔案，此舉可能會揭示編輯器中以及編譯期間以前僅在執行時出現的問題：


  List<String> items = Arrays.asList("string", "string");
  List<Integer> numbers = Collections.unmodifiableList(items); // 不兼容的類型
