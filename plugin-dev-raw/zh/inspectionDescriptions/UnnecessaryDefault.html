報告由於所有可能的值都被 case 分支覆蓋而導致其 default 分支絕不會被接受的枚舉 switch 語句或表達式。

此類別元素冗餘，對於 switch 表達式更是如此，因為當 case 分支沒有覆蓋所有枚舉常數時，它們不會進行編譯。

需要將語言級別組態為 14，才會報告 switch 表達式。

提供的快速修復可以移除 default 分支。

範例：


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
      default -> 3;
    };
  }

在套用快速修復後：


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
    };
  }

使用僅報告 switch 表達式選項可以僅報告 switch 表達式中的冗餘 default 分支。
