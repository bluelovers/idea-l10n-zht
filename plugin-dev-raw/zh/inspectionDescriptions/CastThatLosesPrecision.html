報告可能導致精度損失的基元數值類型之間的轉換運算。

此類轉換不一定是問題，但如果意外造成精度損失，則可能會導致難以跟踪錯誤。

範例：


  int a = 420;
  byte b = (byte) a;

使用忽略從 int 到 char 的轉換選項可以忽略從 int 到 char 的轉換。 在實作 I/O 操作時經常使用這種類型的轉換，因為 java.io.Reader 類的 read() 方法會返回一個 int。

使用忽略從 int 128-255 到字節的轉換選項可以忽略從 int 到 byte 的常量值 (128-255) 的轉換。 此類值將溢出到仍能容納在一個字節內的負數。
