報告不必要的一元減號。 此類別表達式可能難以理解並且可能包含錯誤。

範例：

void unaryMinus(int i) {
    int x = - -i;
  }

此處建議使用以下快速修復：

  • 移除 i 變數前的 - 運算符：

    void unaryMinus(int i) {
    int x = i;
  }

  • 使用前綴遞減運算符取代 - 運算符：

    void unaryMinus(int i) {
    int x = --i;
  }

另一個範例：

void unaryMinus(int i) {
    i += - 8;
  }

在套用快速修復後：

void unaryMinus(int i) {
    i -= 8;
  }
