報告在其差異不允許的位置用作實參的類型變數（只要它們沒有被標記為 unchecked）。 類型變數的方差可以是協變 (out)、逆變 (in) 或 不變（既不是 out 也不是 in）。

以下規則用於檢查特性和方法宣告中使用的類型實參：

  • unchecked 類型實參將被忽略。
  • 協變類型實參只能用於產生值的位置（例如，返回類型和塊參數）。
  • 逆變類型實參只能用於使用實參的位置（例如，參數和塊返回類型）。
  • 不變類型實參可以用於產生值和使用實參的位置。

範例：

class Example[in A, out B]
  # 'Example' 類別的協變類型變數 'B' 用於逆變位置，…
  def call: (B) -> A
end
