檢測以下錯誤級別的 EJB 規範違規：
  • Finder 和 Selector 方法未正確定義
  • CMP 和 CMR 欄位存取器未正確定義

範例：


public class SessionEJB implements javax.ejb.SessionBean {
  // 錯誤：方法 'ejbCreateXXX' 應該為 public
  protected void ejbCreateExample() { ... }

  // 錯誤：方法 'ejbCreateAnotherExample' 應該返回 'void'
  public int ejbCreateAnotherExample() { ... }
}
