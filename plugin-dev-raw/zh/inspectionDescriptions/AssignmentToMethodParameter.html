報告對方法參數的賦值或修改。

雖然有時是有意為之，但此結構可能令人困惑，因此在一些 Java 專案中被禁止使用。

該快速修復會新增新變數的宣告。

範例：


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

在套用快速修復後：


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

使用如果賦值是原始參數的轉換，則忽略選項可忽略根據先前值修改參數值的賦值。
