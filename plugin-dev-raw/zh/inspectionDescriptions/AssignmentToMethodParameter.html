報告對宣告為方法參數的變數的賦值。 還會報告任何遞增或遞減變數的嘗試。

雖然有時是有意為之，但這種結構可能令人困惑，因此在一些 Java 專案中被禁止使用。

該快速修復會添加新變數的宣告。

範例:


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

在套用快速修復後：


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

使用如果賦值是原始參數的轉換，則忽略選項可忽略根據先前值修改參數值的賦值。
