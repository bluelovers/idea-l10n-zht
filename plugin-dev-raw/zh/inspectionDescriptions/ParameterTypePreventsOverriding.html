報告與相應 super 方法的參數類型同名但屬於不同軟體套件的子類方法的參數類型。 在這些情況下，子類方法不能覆寫 super 方法。

範例：


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(Object o) {} // 對參數類型發出警告
 class Object {}
}

在套用快速修復後：


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(java.lang.Object o) {} // 新參數類型
 class Object {}
}
