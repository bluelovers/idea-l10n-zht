報告升序計數迴圈中的 list.remove(index) 調用。

這很可疑，因為移除後列表變短，下一個元素被跳過。 簡單的修正方法是在移除後減小索引變數，但更穩健的做法可能是通過迭代器或使用 removeIf() 方法（Java 8 及更高版本）移除。 如果您不希望 remove() 在迴圈中被多次調用，不妨在它後面添加 break。

範例：

  public static void main(String[] args) {
    process(new ArrayList<>(
      Arrays.asList("1", "2", "|", "3", "4")));
  }

  static void process(List<String> list) {
    for (int i = 0; i < list.size(); i++) {
      if (list.get(i).equals("|")) {
        list.remove(i);
        continue;
      }
      System.out.println(list.get(i));
    }
  }

程式碼看起來像是會列印 1 2 3 4，但實則在輸出中將跳過 3。
