報告長的 Reactive Streams 轉換鏈。

每個運算符方法呼叫，例如 map() 或 filter()，都會為這些運算符建立一些物件。 在每個訂閱上對每個流元素呼叫長運算符鏈可能會導致性能開銷。 為避免這種情況，請盡可能將長運算符鏈合併為一個運算符呼叫。

範例：


Flux.just(1, 2, 3)
        .map(it -> it + 1)
        .map(it -> it + 2)
        .map(it -> it + 3)

在套用快速修復後：


Flux.just(1, 2, 3)
        .map(it -> it + 1 + 2 + 3)
