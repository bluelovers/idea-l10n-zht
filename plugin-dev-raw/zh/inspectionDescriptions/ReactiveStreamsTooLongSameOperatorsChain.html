報告長的 Reactive Streams 轉換鏈。

每個運算符方法調用，例如 map() 或 filter()，都會為這些運算符創建一些物件。 在每個訂閱上對於每個流元素調用一長串運算符可能會導致性能開銷。 為避免這種情況，請盡可能將一長串調用合併為一個操作員調用。

範例:


Flux.just(1, 2, 3)
        .map(it -> it + 1)
        .map(it -> it + 2)
        .map(it -> it + 3)

在套用快速修復後：


Flux.just(1, 2, 3)
        .map(it -> it + 1 + 2 + 3)
