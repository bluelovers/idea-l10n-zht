報告可以使用 Kotlin Spring 擴展替代帶有 ::class.java 實參的呼叫的位置。

Spring 為帶有類型令牌參數的泛型 Java 方法提供了許多 Kotlin 擴展。 此檢查會報告無需更改語意即可將 Java 樣式的呼叫取代為 Kotlin 擴展呼叫的 Spring API 用法。

範例：


import org.springframework.core.env.PropertyResolver

fun example(resolver: PropertyResolver): String? =
    resolver.getProperty("test.property", String::class.java)

套用修正後：


import org.springframework.core.env.PropertyResolver
import org.springframework.core.env.getProperty

fun example(resolver: PropertyResolver): String? =
    resolver.getProperty<String>("test.property")
