報告遞歸 equals(==) 調用。

== （Kotlin 的結構比較）可能會被 Java 開發者錯誤地解釋為引用比較。 因此，該檢查會報告此類情況，並建議將 == 取代為 ===（Kotlin 的引用比較）。

範例:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this == other) return true
          return false
      }
  }

在套用快速修復後：


  class X {
      override fun equals(other: Any?): Boolean {
          if (this === other) return true
          return false
      }
  }
