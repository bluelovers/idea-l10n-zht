報告遞歸 equals(==) 調用。

在 Kotlin 中，== 會通過在背景調用 equals 方法來比較物件值。 另一方面，=== 則通過引用來比較物件。

=== 通常用於 equals 方法實現。 但 === 可能會錯誤地與 == 混淆，從而導致無限遞歸。

範例：


  class X {
      override fun equals(other: Any?): Boolean {
          if (this == other) return true
          return false
      }
  }

在應用快速修復後：


  class X {
      override fun equals(other: Any?): Boolean {
          if (this === other) return true
          return false
      }
  }
