報告為不詳盡的枚舉類型執行的 switch 語句。

範例：


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {

      }
    }
  }

在套用快速修復後：


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {
        case A -> {}
        case B -> {}
        case C -> {}
      }
    }
  }

使用 忽略具有預設分支的 switch 語句 選項忽略具有 default 分支的 switch 語句。
