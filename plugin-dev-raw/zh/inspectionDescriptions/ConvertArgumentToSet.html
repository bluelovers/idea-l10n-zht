檢測可以通過將實參轉換為 Set 而加快速度的函數調用。

當實參為集合時，'minus' 或 'intersect' 等運算更高效。 將 Iterable<T> 或 Array<T> 顯式轉換為 Set<T> 通常可以提高程式碼效率。

該快速修復會向函數調用添加顯式轉換。

範例：


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b).size

在套用快速修復後：


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b.toSet()).size
