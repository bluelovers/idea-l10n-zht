報告覆寫 equals() 但不覆寫 hashCode()（或者相反動作）的類別。 同時還報告覆寫 equals() 或 hashCode() 的物件宣告。

這可能會導致在將類別新增到 Collection 時產生不良行為

範例：


  class C1 {
      override fun equals(other: Any?) = true
  }

  class C2 {
      override fun hashCode() = 0
  }

  object O1 {
      override fun equals(other: Any?) = true
  }

  object O2 {
      override fun hashCode() = 0
  }

該快速修復會為類別覆寫 equals() 或 hashCode() 方法並為物件刪除這些方法：


   class C1 {
       override fun equals(other: Any?) = true
       override fun hashCode(): Int {
           return javaClass.hashCode()
       }
   }

   class C2 {
       override fun hashCode() = 0
       override fun equals(other: Any?): Boolean {
           if (this === other) return true
           if (javaClass != other?.javaClass) return false
           return true
       }
   }

   object O1 {
   }

   object O2 {
   }
