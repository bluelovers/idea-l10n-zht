報告用作 if、while、for 或 do 語句或條件表達式的賦值。

雖然有時是有意為之，但這種用法令人困惑，並且可能表明存在拼寫錯誤（例如，應使用 = 而不是 ==）。

該快速修復會將 = 取代為 ==。

範例：


  void update(String str, boolean empty) {
    // 警告：'empty' 被重新賦值，
    // 不與 str.isEmpty() 進行比較
    if (empty = str.isEmpty()) {
      ...
    }
  }

在套用快速修復後：


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
