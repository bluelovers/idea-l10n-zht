報告僅從 then 分支拋出 java.lang.Throwable 而沒有 else 分支的 if 語句。 此類語句可以轉換為更緊湊的 assert 語句。

檢查還報告 Guava 的 Preconditions.checkNotNull()。 它們可以取代為不需要庫 Objects.requireNonNull() 調用。

範例：


  if (x == 2) throw new RuntimeException("fail");
  if (y == null) throw new AssertionError();
  Preconditions.checkNotNull(z, "z");

在應用快速修復後：


  assert x != 2 : "fail";
  Objects.requireNonNull(y);
  Objects.requireNonNull(z, "z");

預設情況下，此檢查在編輯器中提供快速修復，而沒有高亮顯示程式碼。
