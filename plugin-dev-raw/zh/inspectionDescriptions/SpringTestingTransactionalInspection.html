報告在註解為 @Transactional 的測試類別中使用 @BeforeTransaction 和 @AfterTransaction 註解的無效事務生命周期方法宣告。

註解的方法必須沒有實參和返回類型。

範例：


  @ContextConfiguration
  @Transactional
  public class AbstractShowcaseTest {
    @BeforeTransaction // 方法返回類型應為 'void'
    public boolean setupData() {...}

    @AfterTransaction // 實參數量錯誤
    public void disposeData(boolean a) throws Exception {...}
  }
