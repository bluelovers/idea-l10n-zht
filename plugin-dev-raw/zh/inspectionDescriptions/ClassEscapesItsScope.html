當簽名中類的可見性不如成員本身的可見性時，報告類在欄位或方法簽名中的用法。 雖然是合法的 Java，但此類成員在可見性範圍之外是無用的。

範例：

  • 返回 private 內部 class 的 public 方法。
  • 類型為 package-local class 的 protected 欄位。

在 Java 9 中，模組可能會通過將其軟體套件排除在匯出之外來隱藏某些類。 因此，如果匯出的 API 的簽名包含非匯出類，此類 API 在模組之外就是無用的。

設定檢查：

  • 使用模組的 API 公開未匯出的類(Java 9+) 選項可以報告公開未匯出類的模組 API。
    請注意，專案或模組的語言級別為 9 或更高時，該選項才有效。
  • 使用公共 API 公開不可存取的類選項可以報告公開不可存取類的公共 API。
  • 使用 Package-local API 公開 private 類選項可以以報告公開 private 類的 package-local API。
