報告實作 java.lang.Comparable 但不覆寫 equals() 的類別。

如果未覆寫 equals()，equals() 實作與 compareTo() 實作將不一致。 如果將這種類的物件新增到 java.util.SortedSet 等集合，該集合將違反根據 equals() 定義的 java.util.Set 的約定。

範例：


  class Length implements Comparable<Length> {
    private int cm = 0;

    @Override
    public int compareTo(@NotNull Length o) {
      if (cm == o.cm) return 0;
      return cm < o.cm ? -1 : 1;
    }
  }

在套用快速修復後：


  class Length implements Comparable<Length> {
    private int cm = 0;

    @Override
    public int compareTo(@NotNull Length o) {
      if (cm == o.cm) return 0;
      return cm < o.cm ? -1 : 1;
    }

    @Override
    public boolean equals(Object o) {
      return o instanceof Length && compareTo((Length) o) == 0;
    }
  }
