報告 public 嵌套類別。

範例：


  public class Outer {
    public static class Nested {} // 警告
    public class Inner {}         // 警告
    public enum Mode {}           // 警告取決於設定
    public interface I {}         // 警告取決於設定
  }

組態檢查：

  • 使用忽略 'public' 內部枚舉選項可忽略 public 內部枚舉。
  • 使用忽略 'public' 內部接口選項可忽略 public 內部接口。
