報告除了第一次迭代中產生的副作用外，在第二次和後續迭代中不產生其他副作用的迴圈，這可能表示編程錯誤。

此類別迴圈可能只迭代零次、一次或無限次。 如果不存在無限迴圈的情況，則可以將其取代為 if 語句。 否則，程序可能會卡住。

範例：


    public void foo(String baseName, String names) {
        int suffix = 1;
        String name = baseName;
        while (names.contains(name)) {
            // 錯誤：suffix 未更新，導致迴圈體冪等
            name = baseName + suffix;
        }
    }
