報告具有一個空參數子句且覆寫沒有參數子句的方法的方法。

推薦的約定是，只要沒有參數並且該方法沒有副作用，就使用沒有參數子句的方法。

該約定促進了統一存取原則，即客戶端程式碼不應受到將特性實作為欄位或方法的決定的影響。

根據里氏取代原則，當被覆寫的方法沒有參數子句時，覆寫方法也必須被宣告為沒有副作用的方法。

該快速修復會移除空參數子句。

範例：


  trait Base { def isOk: Boolean }

  class Impl extends Base {
    override def isOk(): Boolean = ???
  }

在套用快速修復後：


  trait Base { def isOk: Boolean }

  class Impl extends Base {
    override def isOk: Boolean = ???
  }
