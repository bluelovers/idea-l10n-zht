報告參照中的冗餘反引號。

一些 Kotlin 關鍵字在 Java 中是有效的關鍵字，例如：in、object、is。 如果 Java 庫將 Kotlin 關鍵字用於方法，您仍然可以呼叫該方法並使用反引號字元 (`) 對其進行轉義，例如 foo.`is`(bar)。 有時這種轉義是冗餘的，可以安全地省略。 該檢查會發現並報告此類別情況，並與“移除冗餘反引號”快速修復配對，允許您修改醒目提示的程式碼。

範例：


  fun `is`(x: String) {}
  fun foo() {
      `is`("bar") // 'is' 是一個關鍵字，反引號為必選項
  }

  fun `test that smth works as designed`() {} // 可以，複雜關鍵字有助於改進可讀性

  val `a` = 1  // 不需要反引號
