報告對包含 java.util.List 實參的 java.util.Set.removeAll() 的調用。

當實參大小大於或等於集合大小，並且集合是 java.util.AbstractSet 的子類時，此類調用可能會很慢。 在本例中，對集合中的每個元素調用了 List.contains()，將執行線性搜尋。

範例：

  public void check(String... ss) {
    // 可能的 O(n^2) 復雜度
    mySet.removeAll(List.of(ss));
  }

在套用快速修復後：

  public void check(String... ss) {
    // O(n) 復雜度
    List.of(ss).forEach(mySet::remove);
  }
