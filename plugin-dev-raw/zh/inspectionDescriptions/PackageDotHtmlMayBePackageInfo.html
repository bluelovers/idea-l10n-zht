報告用於記錄軟體套件的任何 package.html 檔案。 從 JDK 1.5 開始，建議您改用 package-info.java 檔案，因為這些檔案還可以包含軟體套件註解。 這樣，package-info.java 就變成了軟體套件級別註解和文檔的唯一儲存庫。

範例：package.html

<html>
  <body>
    Documentation example.
  </body>
</html>

套用快速修復後，結果如下所示: package-info.java

/**
 * Documentation example.
 */
package com.sample;
