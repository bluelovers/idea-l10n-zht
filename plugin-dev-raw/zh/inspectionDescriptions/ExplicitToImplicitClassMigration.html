報告可被轉換為緊湊原始檔的普通類別

範例：


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

在套用快速修復後：


  void main() {
      IO.println("Hello, world!");
  }

組態檢查：

  • 選擇將 'System.out.print()' 和 'System.out.println()' 取代為 'java.lang.IO' 類別中的方法以將這些呼叫轉換為 IO 方法。
