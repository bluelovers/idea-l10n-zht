報告 Reactor/RxJava 運算符程式碼中的 throw 表達式。

從 Reactor/RxJava 運算符拋出異常表明可能存在問題，因為您可以返回“類似於 Reactive”的錯誤：從 flatMap() 的 Mono.error() 或 Flowable.error()，或者從 Reactor 的 handle() 運算符呼叫 sink.error()。

此外，Reactor 工廠方法允許在沒有任何錯誤的情況下返回已檢查的異常，而在沒有 Exceptions 類別的情況下拋出此類別異常會導致編譯錯誤。

範例：


Flux.just(1, 2, 3).flatMap(i -> {
            throw new RuntimeException();
        })

在套用快速修復後：


Flux.just(1, 2, 3).flatMap(i -> {
            return Flux.error(new RuntimeException());
        })
