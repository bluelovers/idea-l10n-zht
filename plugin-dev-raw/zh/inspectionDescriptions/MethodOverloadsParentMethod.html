報告參數名稱和數量與超類別中的方法相同，但其中至少一個參數為不同的不相容類型的實例方法。

在本例中，子類別中的方法將多載而不是覆寫超類別中的方法。 如果不是有意為之，可能會導致潛在錯誤。

範例：


  public class Foo {
    void foo(int x) {}
  }

  public class Bar extends Foo {
    void foo(Number x) {} // 在覆寫可能是有意為之的情況下，
                          // 方法 'foo()' 多載超類別的相容方法
  }

使用此選項可選擇該檢查是否也應報告參數類型不相容的情況。
