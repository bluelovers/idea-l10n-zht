報告對 compareTo() 實作中的非 final 欄位的存取。

此類別存取可能導致 compareTo() 在物件生命周期的不同時刻返回不同的結果，進而可能在使用 java.util.TreeSet 等標準集合類別時導致問題。

只有在對該欄位不具備寫入存取權限時，才可以通過快速修復將該欄位設為 final，否則不建議進行修正。

範例：


  class Foo implements Comparable<Foo>{
    private int index;
    Foo(int idx) {
      index = idx;
    }
    @Override
    public int compareTo(Foo foo) {
     return Integer.compare(this.index, foo.index);
    }
  }

在套用快速修復後：


  class Foo implements Comparable<Foo>{
    private final int index;
    Foo(int idx) {
      index = idx;
    }
    @Override
    public int compareTo(Foo foo) {
      return Integer.compare(this.index, foo.index);
    }
  }
