報告可以通過提取 return 語句或賦值而轉換為表達式的 if、when 和 try 語句。

範例：


  fun foo(arg: Int): String {
      when (arg) {
          0 -> return "Zero"
          1 -> return "One"
          else -> return "Multiple"
      }
  }
在套用快速修復後： 

  fun foo(arg: Int): String {
      return when (arg) {
          0 -> "Zero"
          1 -> "One"
          else -> "Multiple"
      }
  }
