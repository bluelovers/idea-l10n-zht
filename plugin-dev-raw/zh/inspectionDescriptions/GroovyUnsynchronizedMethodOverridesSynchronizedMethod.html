報告覆寫 synchronized 方法的非 synchronized 方法。

範例：


  abstract class Base {
      synchronized void foo() {
          // ...
      }
  }
  class Derived extends Base {
      @Override
      void foo() {
          super.foo()
            // ...
        }
    }

在這裡，類別 Bar 中的非 synchronized 方法 foo() 會覆寫 synchronized 方法。
