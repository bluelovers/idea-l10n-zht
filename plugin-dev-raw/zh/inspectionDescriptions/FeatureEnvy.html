報告特性依戀程式碼異味。 當方法呼叫另一個類別的方法三次或更多次時，將拋出警告。 對庫類別、父類別、包含類別的呼叫不計入此檢查。 特性依戀通常表明該功能位於錯誤的類別中。

範例：


  class JobManager {
    // 警告：此方法呼叫 Job 類別的
    // 三個方法
    // 最好將此呼叫鏈
    // 移至 Job 類別本身。
    void performJob(Job job) {
      job.beforeStart();
      job.process();
      job.afterProcessing();
    }
  }
