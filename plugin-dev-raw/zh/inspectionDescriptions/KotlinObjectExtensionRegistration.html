報告由 IntelliJ 平台實例化但被宣告為 Kotlin 物件的擴展程序。

擴展程序生存期由 IntelliJ 平台管理。 使用 Kotlin 物件進行擴展程序註冊可能會導致建立不必要的擴展程序實例，並使延伸模組無法移除。

範例

擴展程序註冊：


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

擴展程序實作：


// 錯誤：
object MyAnnotator : Annotator {
  ...
}

// 正確：
class MyAnnotator : Annotator {
  ...
}
