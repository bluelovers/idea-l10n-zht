報告最近的不可序列化祖先沒有無實參建構函式的 Serializable 類別。 這種類無法反序列化，將失敗並拋出 InvalidClassException。

範例：


  class Ancestor {
    private String name;
    Ancestor(String name) {
      this.name = name;
    }
  }

  //對此類別發出警告，因為超類別並非
//可序列化，並且其建構函式接受實參
  class Descendant extends Ancestor implements Serializable {
    Descendant() {
      super("Bob");
    }
  }
