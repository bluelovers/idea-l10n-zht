報告沒有狀態和被覆寫的 equals() 的 sealed 類的直接繼承者。

強烈建議覆寫 equals() 以提供比較穩定性，或將 class 轉換為 object 以達到相同的效果。

範例:


  sealed class Receiver {
      class Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }

建議通過快速修復將 code 轉換為 object：


  sealed class Receiver {
      object Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }
