報告對沒有命名實參的資料類別的 copy() 方法的呼叫。

由於 copy() 函式的所有實參都是可選的，因此可能很難理解修改了哪些屬性。 顯式提供參數名稱使得程式碼容易理解，且無需導覽到 data class 宣告。

範例：


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy("John")
  }

該快速修復程序會為所有的 copy() 實參提供參數名稱：


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy(name = "John")
  }
