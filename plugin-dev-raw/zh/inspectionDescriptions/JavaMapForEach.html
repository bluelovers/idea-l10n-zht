報告可以取代為 Kotlin 的 forEach 的 Java Map.forEach 方法調用。

範例：


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}

該快速修復會移除圓括號：


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}
