報告目標變數或函數參數的名稱與分配給它的值的名稱不符合的賦值和函數調用。

範例 1：


  int x = 0;
  int y = x; // x 用作 y 坐標

範例 2：


  int x = 0, y = 0;
  // x 用作 y 坐標，y 用作 x 坐標
  Rectangle rc = new Rectangle(y, x, 20, 20);

設定檢查：

使用名稱組區域可以指定不應一起使用的名稱：如果參數名稱或賦值目標名稱包含一個組中的字詞，而賦值或傳遞的變數名稱包含另一個組中的字詞，就會報告錯誤。

使用忽略方法區域可以指定不應檢查但名稱可能存疑的方法。 例如，Integer.compare() 參數被命名為 x 和 y，但與坐標無關。
