報告對 Throwable.initCause() 的調用，其中異常構造函數還包含 Throwable cause 實參。

在本例中，可以移除 initCause() 調用，並將其實參添加到對異常構造函數的調用中。

範例：


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing");
    wrapper.initCause(ex); // 不必要調用 'Throwable.initCause()'
    throw wrapper;
  }

可通過快速修復將 cause 實參傳遞給構造函數。 在套用快速修復後：


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing", ex);
    throw wrapper;
  }
