報告僅包含 {@inheritDoc} 標記的 Javadoc 註釋。 由於 Javadoc 會在沒有註解的情況下複製超類的註解，因此僅包含 {@inheritDoc} 的註解不會添加任何內容。

此外，它還會報告無效位置（例如，欄位）中的 {@inheritDoc} 用法。

建議移除不必要的 Javadoc 註釋。

範例:

  class Example implements Comparable<Example> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }

在套用快速修復後：

  class Example implements Comparable<Example> {
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }
