報告 return 點數超過指定最大值的方法。 具有太多 return 點的方法可能令人困惑且難以重構。

一個 return 點可以是 return 語句，也可以是從 void 方法或構造函數底部的直落。

範例：

如果只允許兩個 return 語句，則報告以下方法：


  void doSmth(User[] users) {
    for (User user : users) {
      if (cond1(user)) {
        user.setId(getId());
        return;
      } else if (cond2(user)) {
        if (cond3(user)) {
          user.setId(getId());
          return;
        }
      }
    }
  }

請考慮覆寫該方法，使其更容易理解：


  void doSmth(User[] users) {
    for (User user : users) {
      if (cond1(user) || cond2(user) && cond3(user)) {
        user.setId(getId());
        return;
      }
    }
  }

設定檢查：

  • 使用 return 點限制欄位可指定方法允許的最大 return 點數。
  • 使用忽略臨界子句選項可忽略臨界子句。 臨界子句是只包含 return 語句的 if 語句
  • 使用對 'equals()' 方法忽略選項可忽略 equals() 方法中的 return 點。
