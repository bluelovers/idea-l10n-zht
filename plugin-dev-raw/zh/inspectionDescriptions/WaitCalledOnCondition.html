報告在 java.util.concurrent.locks.Condition 物件上進行的 wait() 呼叫。 這可能是編程錯誤，本應使用 await() 方法的某種變體。

範例：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

優良的程式碼應類似於：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
