報告枚舉類型中的非 final 欄位，因為很少需要它們，且提供全域可變狀態。

範例：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

在套用快速修復後：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
設定“在快速修復不可用時忽略欄位”複選框以僅高亮顯示可通過該快速修復設為 final 的欄位。
