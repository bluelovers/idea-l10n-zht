報告從屬性的類別內部對簡單屬性 setter 的呼叫。

簡單的屬性 setter 被定義為只將其參數的值分配給一個欄位，而不進行其他計算。 可以安全地內聯此類別簡單的 setter 呼叫。 為了保持程式碼清晰，某些編碼標準還建議不使用簡單的 setter。

範例：


  class Foo {
  private int index;
    public Foo(int idx) {
       setIndex(idx);
    }
    public void setIndex(int idx) {
       index = idx;
    }
  }

在套用快速修復後：


  class Foo {
    private int index;
    public Foo(int idx) {
      index = idx;
    }
    public void setIndex(int idx) {
      index = idx;
    }
  }

使用以下選項組態檢查：

  • 是否僅報告 this 上的 setter 呼叫，而不報告作為參數傳入的相同類型的物件。
  • 是否忽略非 private setter。
