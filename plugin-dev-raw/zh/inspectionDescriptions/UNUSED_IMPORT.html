報告冗餘的 import 語句。

如果不使用原始檔中匯入的類別和軟體套件，則不必使用一般 import 語句。 這同樣適用於原始檔中未使用但已匯入的 static 欄位和方法。

範例：


  import java.util.ArrayList;
  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }

在套用快速修復後： 


  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }
