報告包含 __slots__ 定義的類別的無效用法。

存取未定義特性的範例：


class Foo:
    __slots__ = ['foo', 'bar']

    def __init__(self):
        self.x = 3  # error: 'x' is not defined in __slots__

特性衝突的範例：


class A:
    __slots__ = ("x",)
    x = 42  # error: conflict with "x" listed in __slots__

範例 slots=True：


from dataclasses import dataclass

@dataclass(slots=True)  # 錯誤：__slots__ 也在 Foo 中定義
class Foo:
    __slots__ = ['a']
