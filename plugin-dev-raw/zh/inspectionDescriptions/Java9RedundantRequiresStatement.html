報告 Java Platform Module System module-info.java 檔案中的冗餘 requires 指令。 當模組 A 需要模組 B，但模組 A 中的程式碼不從 B 匯入任何軟體套件或類別時，則 requires 指令冗餘。 此外，所有模組都具有對 java.base 模組的隱式宣告相依，因此 requires java.base; 指令始終冗餘。

該快速修復會刪除冗餘的 requires 指令。 如果刪除的相依項重新匯出了實際使用的模組，則修復會為這些模組新增 requires 指令。

僅當專案或模組的語言級別為 9 或更高時，此檢查才會報告。
