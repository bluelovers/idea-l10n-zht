報告方法調用和欄位引用中不必要的 super 限定符。

當超類的欄位或方法在調用類中未被隱藏/覆寫時，不需要 super 限定符。

範例：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }

在套用快速修復後：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      foo();
    }
  }

使用檢查設定可以忽略有助於區分超類成員存取與外部類的同名成員的限定符。

另請參閱以下檢查：

  • Java | 可見性 | 繼承欄位的存取類似於存取周圍程式碼中的元素
  • Java | 可見性 | 對繼承方法的調用類似於對局部方法的調用
