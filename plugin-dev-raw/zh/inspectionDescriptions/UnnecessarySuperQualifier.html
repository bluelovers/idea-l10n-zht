報告方法呼叫和欄位參照中不必要的 super 限定符。

當超類別的欄位或方法在呼叫類別中未被隱藏/覆寫時，不需要 super 限定符。

範例：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }

在套用快速修復後：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      foo();
    }
  }

使用檢查設定可以忽略有助於區分超類別成員存取與外部類別的同名成員的限定符。

另請參閱以下檢查：

  • Java | 可見性 | 繼承欄位的存取類似於存取周圍程式碼中的元素
  • Java | 可見性 | 對繼承方法的呼叫類似於對局部方法的呼叫
