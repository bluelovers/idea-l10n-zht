報告內容類型為枚舉類別的 java.util.Set 物件的實例化。 此類別 Set 物件可以取代為 java.util.EnumSet 物件。

與其他集合相比，EnumSet 實作的效率更高，因為底層資料結構是位向量。 使用該快速修復可以將初始設定式取代為 EnumSet.noneOf() 呼叫。 當變數類型是 Set 的子類別時，此快速修復不可用。

範例：


  enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = new HashSet<MyEnum>();

在套用快速修復後：


  enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = EnumSet.noneOf(MyEnum.class);
