報告可以取代為 switch 語句的 if 語句。

取代結果通常更簡短、更清晰。

範例：


  void test(String str) {
    if (str.equals("1")) {
      System.out.println(1);
    } else if (str.equals("2")) {
      System.out.println(2);
    } else if (str.equals("3")) {
      System.out.println(3);
    } else {
      System.out.println(4);
    }
  }

在套用快速修復後：


  void test(String str) {
    switch (str) {
      case "1" -> System.out.println(1);
      case "2" -> System.out.println(2);
      case "3" -> System.out.println(3);
      default -> System.out.println(4);
    }
  }

僅當專案或模組的語言級別為 7 或更高時，此檢查才會報告。

使用 'if' 條件分支的最小數量欄位可指定必須報告 if 語句的最小 if 條件分支數。 注意，不計入終端 else 分支（無 if）。

使用對數字啟用 switch 建議選項，可以對基元和裝箱的數字及字符啟用 switch 語句建議。

使用對枚舉啟用 switch 建議選項，可以對 enum 常數啟用 switch 語句建議。

使用僅對 null-safe 表達式啟用建議選項，可僅建議不能引入 NullPointerException 的 switch 語句。
