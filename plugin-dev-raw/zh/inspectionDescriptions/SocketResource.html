報告未安全關閉的套接字資源。 此檢查報告的套接字資源包括 java.net.Socket、java.net.DatagramSocket 和 java.net.ServerSocket。

預設情況下，該檢查假定資源可以由名稱中帶有 'close' 或 'cleanup' 的任何方法關閉。

範例：


  byte[] getMessage(ServerSocket socket) throws IOException {
    Socket client = socket.accept(); //套接字未關閉
    return client.getInputStream().readAllBytes();
  }

使用以下選項組態檢查：

  • 是否允許在 try 塊內開啟套接字。 這種樣式不太合意，因為它比在 try 塊前面開啟資源更冗長。
  • 資源能否通過任何將資源作為實參傳遞的方法呼叫來關閉。
