報告用作格式字串實參的非常數字串串聯。

雖然偶爾是有意為之，但這通常是誤用格式化方法，串聯的字串中使用的變數包含 % 等特殊字元時，甚至可能導致安全問題。

此外，有時這可能是在本應鍵入 , 時鍵入了 + 而錯誤串聯字串格式實參所致。

範例：


  static String formatGreeting(String userName) {
    return String.format("Hello, " + userName);
  }

這裡的 userName 將解釋為格式字串的一部分，從而可能導致 IllegalFormatException（例如，如果 userName 為 "%"）或使用大量記憶體（例如，如果 userName 為 "%2000000000%"）。 該呼叫可能應取代為 String.format("Hello, %s", userName);。

此檢查將檢查對 java.util.Formatter、java.lang.String、java.io.PrintWriter 或 java.io.PrintStream 的格式化方法的呼叫。
