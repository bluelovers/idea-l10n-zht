報告比較 .length 或 .size，並建議將其取代為 .sizeIs。

範例：


  var seq: Seq[Int]
  seq.length == n
  seq.length != n
  seq.length < n
  seq.length > n

  seq.size == n
  seq.size != n
  seq.size < n
  seq.size > n

在套用快速修復後：


  var seq: Seq[Int]
  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n

  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n
