報告 TypedDict 無效的定義和用法。

範例：

from typing import TypedDict


class Movie(TypedDict):
    name: str
    year: int
    rate: int = 10  # 右側值不受支援

    def method(self): # TypedDict 中的語句無效
        pass


m = Movie(name="name", year=1000, rate=9)
print(m["director"])  # 'Movie' 中沒有 'director' 鍵
del m["name"]  # 無法刪除 'name' 鍵
m["year"] = "1001"  # 應為 'int'，得到的是 'str'
