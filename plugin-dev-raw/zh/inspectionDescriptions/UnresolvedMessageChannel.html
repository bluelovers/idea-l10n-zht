報告端點方法註解中不正確的通道特性： @Gateway、 @ServiceActivator、 @Filter 等。

範例：


  @Component
  public class MyComponent {}
     @Gateway(requestChannel = "requestChannel", replyChannel = "replyChannel")
     public void gateway() {...}

     @Gateway(
        requestChannel ="simpleBean",   // Bean 必須為以下一種類型：org.springframework.integration.MessageChannel,org.springframework.messaging.MessageChannel
        replyChannel = "unknownChannel")   // 找不到通道
     public void errors() {...}

     @Bean
     public MyBean simpleBean() {...}
  }
