報告可以安全移除的冗餘分號 (;)。

Kotlin 不要求在每個語句或表達式的末尾使用分號。 建議使用該快速修復移除冗餘的分號。

範例：


  val myMap = mapOf("one" to 1, "two" to 2);
  myMap.forEach { (key, value) ->  print("$key -> $value")};

在套用快速修復後：


  val myMap = mapOf("one" to 1, "two" to 2)
  myMap.forEach { (key, value) ->  print("$key -> $value")}

有兩種情況需要使用分號：

  1. 放在一個行裡的多個語句需要用分號隔開： 
    
  map.forEach { val (key, value) = it; println("$key -> $value") }
  2. 同時宣告了屬性或函式的 enum 類別要求在枚舉常數列表後加一個分號： 
    
  enum class Mode {
      SILENT, VERBOSE;

      fun isSilent(): Boolean = this == SILENT
  }
