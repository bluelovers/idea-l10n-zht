報告可以取代為 case 語句的 if 語句。

使用最小分支數欄位為得到的 case 語句指定最小 when 分支數。

範例：

fruit = gets

if fruit == 'apple' # Simplifiable 'if' statement
  puts 'This is apple!'
elsif fruit == 'peach'
  puts 'This is peach!'
elsif fruit == 'orange'
  puts 'This is orange!'
elsif fruit == 'banana'
  puts 'This is banana!'
end
