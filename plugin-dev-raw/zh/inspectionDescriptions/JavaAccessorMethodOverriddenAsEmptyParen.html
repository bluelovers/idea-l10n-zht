報告具有覆寫 Java 存取器方法的空參數子句的方法。

符合存取器 JavaBean 命名協定的方法預計不會有副作用。 推薦的約定是，在沒有參數且方法沒有副作用時使用無參數方法。 該約定促進了統一存取原則，即客戶端程式碼不應受到將特性實作為欄位或方法的決定的影響。

問題是 Java 沒有實作統一存取原則，不允許沒有參數子句的方法。 為了填補這一空白，Scala 允許您使用沒有參數子句的方法來覆寫帶有空參數子句的方法。

該快速修復會移除空參數子句。

範例：


  class TestException extends Exception {
    override def getMessage(): String = ???
  }

在套用快速修復後：


  class TestException extends Exception {
    override def getMessage: String = ???
  }
