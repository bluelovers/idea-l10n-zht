報告日期和時間文字中的錯誤。 此檢查在 MySQL、Oracle、Db2 和 H2 中可用。

示例（MySQL）：

SELECT TIME '10 -12:13:14' FROM dual;
SELECT TIME ' 12 : 13 : 14 ' FROM dual;
SELECT TIME '12 13 14' FROM dual;
SELECT TIME '12-13-14' FROM dual;
SELECT TIME '12.13.14' FROM dual;
SELECT TIME '12:13:' FROM dual;
SELECT TIME '12:13' FROM dual;
SELECT TIME '12:' FROM dual;

在此示例中，日期會忽略 MySQL 日期文字和時間文字標準。 因此，這些將高亮顯示。 有關 MySQL 中日期和時間文字的詳細資訊，請參閱 dev.mysql.com 上的“日期和時間文字”頁面。

以下日期和時間文字對 MySQL 有效。

SELECT TIME '12:13:14' FROM dual;
SELECT TIME '12:13:14.555' FROM dual;
SELECT TIME '12:13:14.' FROM dual;
SELECT TIME '-12:13:14' FROM dual;
SELECT TIME '10 12:13:14' FROM dual;
SELECT TIME '-10 12:13:14' FROM dual;
