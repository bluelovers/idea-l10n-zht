報告不正確的儲存庫方法參數類型。

範例：


  public class Person {
    private int id;
    @Column
    private String lastname;

    @Column
    private int age;
  }

  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // 應為 'String' 類型
    List<Person> deleteByAge(String str);   // 應為 'int' 類型
  }
