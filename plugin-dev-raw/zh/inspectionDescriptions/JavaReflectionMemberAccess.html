報告對不存在或不可見的欄位和方法的反射存取。

範例：


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getField("hash");
  }

在套用快速修復後：


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getDeclaredField("hash");
  }

對於 final 類別，很清楚該類中是否存在具有指定名稱的欄位或方法。

對於非 final 類別，子類別中可能有使用該名稱的欄位或方法，因此可能會出現誤報。 使用該項檢查的設定，可消除任何位置或特定類別的此類別誤報。
