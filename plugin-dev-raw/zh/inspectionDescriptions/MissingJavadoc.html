報告缺少 Javadoc 註釋和標記。

範例：


  /**
   * 報告缺少 "@param"（如果組態）。
   */
  public void sample(int param){
  }

該快速修復會新增缺少的標記或缺少的 Javadoc 註釋：


  /**
   * 報告缺少 "@param"（如果組態）。
   * @param param
   */
  public void sample(int param){
  }

可以將檢查組態為忽略棄用的元素或簡單的存取器方法，如 getField() 或 setField()。
