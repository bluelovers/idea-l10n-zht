報告在迴圈中調用的可以取代為 List.subList().clear() 的 List.remove(index)。

在刪除多個元素的情況下，此取代對於大多數 List 實作更有效。

範例:


  void removeRange(List<String> list, int from, int to) {
    for (int i = from; i < to; i++) {
        list.remove(from);
    }
  }

在套用快速修復後：

  void removeRange(List<String> list, int from, int to) {
    if (to > from) {
      list.subList(from, to).clear();
    }
  }

該快速修復會自動添加範圍檢查，以防止在最小值大於最大值時可能出現的 IndexOutOfBoundsException。 如果程式碼中不可能出現這種情況，則可將其移除。
