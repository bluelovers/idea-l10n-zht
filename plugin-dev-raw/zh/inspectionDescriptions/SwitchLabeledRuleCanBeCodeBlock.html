報告可以轉換為程式碼塊的 switch 表達式或增強的 switch 語句的規則。

範例:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

在套用快速修復後：


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };

僅當專案或模組的語言級別為 14 或更高時，此檢查才會報告。
