報告帶有手動遞增索引變數的 for 迴圈。

可以使用 withIndex() 函數簡化帶有手動遞增索引變數的 for 迴圈。

使用 withIndex() 代替手動索引遞增快速修復可用於自動修正程式碼。

範例：


  fun foo(list: List<String>): Int? {
      var index = 0
      for (s in list) { <== can be simplified
          val x = s.length * index
          index++
          if (x > 0) return x
      }
      return null
  }

在套用快速修復後：


  fun foo(list: List<String>): Int? {
      for ((index, s) in list.withIndex()) {
          val x = s.length * index
          if (x > 0) return x
      }
      return null
  }
