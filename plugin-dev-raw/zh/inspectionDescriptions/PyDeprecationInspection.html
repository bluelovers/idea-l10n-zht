報告標記為已棄用並引發 DeprecationWarning 或 PendingDeprecationWarning 警告的 Python 函數或方法的用法。

此外，該檢查還高亮顯示 abc.abstractstaticmethod、abc.abstractproperty 和 abc.abstractclassmethod 裝飾器的用法。

範例：

class Foo:
    @property
    def bar(self):
        import warnings
        warnings.warn("this is deprecated", DeprecationWarning, 2)
        return 5


foo = Foo()
print(foo.bar)
