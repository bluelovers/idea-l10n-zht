報告可以替換為 equals(..., ignoreCase = true) 的不區分大小寫的比較。

使用 equals()，無需分配具有 toLowerCase() 或 toUpperCase() 的額外字符串即可比較字符串。

該快速修復會將使用 toLowerCase() 或 toUpperCase() 的不區分大小寫的比較替換為 equals(..., ignoreCase = true)。

注意：可能會更改某些區域設置的語義。

示例：


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

在應用快速修復後：


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
