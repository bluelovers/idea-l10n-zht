報告可以取代為 equals(..., ignoreCase = true) 的不區分大小寫的比較。

使用 equals()，無需分配具有 toLowerCase() 或 toUpperCase() 的額外字串即可比較字串。

該快速修復會將使用 toLowerCase() 或 toUpperCase() 的不區分大小寫的比較取代為 equals(..., ignoreCase = true)。

注意：可能會更改某些區域設定的語意。

範例：


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

在套用快速修復後：


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
