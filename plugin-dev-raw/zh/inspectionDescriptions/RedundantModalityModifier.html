報告與元素的預設模態匹配的模態修飾符（對於大多數元素來說是 final，對於具有 override 的成員來說是 open）。

示例：


  final class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }

在應用快速修復後：


  class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }
