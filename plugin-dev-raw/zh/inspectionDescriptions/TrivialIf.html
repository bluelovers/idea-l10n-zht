報告可以簡化為單個賦值、return 或 assert 語句的 if 語句。

範例:


  if (foo()) {
    return true;
  } else {
    return false;
  }

在套用快速修復後：


  return foo();

設定檢查：

如果要隱藏鏈式 if 語句的警告，請使用忽略鏈式 'if' 語句選項。

例如，在以下程式碼中，警告將被隱藏，但快速修復仍然可用：


  if (condition1) return true;
  if (condition2) return false;
  return true;

如果要隱藏 if 語句在其主體中僅包含 assert 語句的警告，請使用 忽略 'assert' 語句選項。

例如，在以下程式碼中，警告將被隱藏：


  if (isTrue()) assert false;
