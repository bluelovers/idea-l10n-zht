報告包含 else 分支並且條件被否定的 if 語句。

翻轉 if 和 else 分支的順序通常會增加此類別語句的清晰度。

可以通過修正反轉當前 if 語句。

範例：


    void m(Object o1, Object o2) {
        if (o1 != o2) {
            System.out.println(1);
        }
        else {
            System.out.println(2);
        }
    }

套用快速修復後：


    void m(Object o1, Object o2) {
        if (o1 == o2) {
            System.out.println(2);
        } else {
            System.out.println(1);
        }
    }

使用忽略 '!= null' 比較選項可以忽略 != null 形式的比較。

使用忽略 '!= 0' 比較選項可以忽略 != 0 形式的比較。
