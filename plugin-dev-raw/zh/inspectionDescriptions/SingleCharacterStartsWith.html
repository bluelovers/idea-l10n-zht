報告對 String.startsWith() 和 String.endsWith() 的調用，其中單字符字符串文字會作為實參傳遞。

建議通過快速修復，使用更高效實現的 String.charAt() 替換此類調用。

然而，由於額外的非零長度檢查，此類更改的性能增益很小並且會降低代碼的可讀性，建議僅在緊密循環內應用該快速修復。

此檢查適用於 Java ME 和其他資源高度受限的環境。 不加考慮地應用此檢查的結果可能會對代碼清晰度和設計產生負面影響。

示例：


  boolean startsWithX(String s) {
    return s.startsWith("x");
  }

在應用快速修復後：


  boolean startsWithX(String s) {
    return !s.isEmpty() && s.charAt(0) == 'x';
  }
