報告在初始化後立即檢查變數是否為 null 的 if 表達式，該表達式可以在初始設定式中轉換為 elvis 運算符。

範例：


  fun test(foo: Int?, bar: Int): Int {
      var i = foo
      if (i == null) {
          return bar
      }
      return i
  }

該快速修復會將帶有初始設定式的 if 表達式轉換為 elvis 表達式：


  fun test(foo: Int?, bar: Int): Int {
      var i = foo ?: return bar
      return i
  }
