報告冗餘的修飾符並建議移除它們。 結果程式碼會更短，但其行為和含義將保持不變。

範例 1：


  // 在 Java 17 和更高版本下，所有程式碼都是隱式 strictfp
  strictfp class X {

    // 內部枚舉為隱式 static
    static enum Inner {
        A, B, C
    }

    // 內部記錄為隱式 static
    static record R() {
    }
  }

範例 2：


  final record R() {
    // 所有記錄均為隱式 final
  }

範例 3：


  // 所有接口均為隱式 abstract
  abstract interface Printer {

    // 所有接口成員均為隱式 public
    public int size();

    // 接口的所有內部類別均為隱式 static
    static class Inner {}
  }
