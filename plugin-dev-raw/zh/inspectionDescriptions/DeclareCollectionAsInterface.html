報告使用集合類別作為類型而不是使用相應接口進行的 Collection 變數的宣告。 如果削弱變數類型會導致編譯錯誤，則不發出警告。

範例：


  // 警告：使用了具體集合類別 ArrayList。
  int getTotalLength(ArrayList<String> list) {
    return list.stream().mapToInt(String::length).sum();
  }

  // 無警告，因為 trimToSize() 方法
  // 在 List 接口中不可用
  void addData(ArrayList<String> data) {
    data.add("Hello");
    data.add("World");
    data.trimToSize();
  }
建議通過快速修復使用相應集合接口（例如 Collection、Set 或 List）。
