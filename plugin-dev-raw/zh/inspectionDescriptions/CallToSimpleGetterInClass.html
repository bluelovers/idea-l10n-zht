報告從屬性的類內部對簡單屬性 getter 的調用。

簡單的屬性 getter 會被定義為只返回欄位的值，而不進行其他計算。 可以使用該快速修復安全地內聯此類簡單的 getter 調用。 為了保持程式碼清晰，某些編碼標準還建議不使用簡單的 getter。

範例：


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return getName();
    }
  }

在套用快速修復後：


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return name;
    }
  }

使用以下選項設定檢查：

  • 是否僅報告 this 上的 getter 調用，而不報告作為參數傳入的相同類型的物件。
  • 是否忽略非 private getter。
