報告從屬性的類內部對簡單屬性 getter 的調用。

簡單的屬性 getter 會被定義為只返回字段的值，而不進行其他計算。 可以使用該快速修復安全地內聯此類簡單的 getter 調用。 為了保持代碼清晰，某些編碼標准還建議不使用簡單的 getter。

示例：


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return getName();
    }
  }

在應用快速修復后：


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return name;
    }
  }

使用以下選項配置檢查：

  • 是否僅報告 this 上的 getter 調用，而不報告作為形參傳入的相同類型的對象。
  • 是否忽略非 private getter。
