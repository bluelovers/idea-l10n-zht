報告實作 java.io.Serializable 的類別中的不可序列化欄位。 如果物件被序列化，此類別欄位將導致執行時異常。

不報告被宣告為 transient 或 static 的欄位，也不報告定義了 writeObject 方法的類別的欄位。

此檢查假定 java.util.Collection 和 java.util.Map 類型的欄位為 Serializable，除非宣告它們的類型為非 Serializable。

範例：


  class NonSerializableClass {}

  public class SerializableClass implements Serializable {
      NonSerializableClass clazz; // //警告：可序列化類別中存在不可序列化欄位 'clazz'
      static NonSerializableClass staticClazz; // 無警告
  }

使用以下選項組態檢查：

  • 列出此檢查不應報告其繼承者的類別。 這適用於從超類別繼承 Serializable 但不用於序列化的類別。
  • 列出會使檢查忽略註解欄位的註解。
  • 是否忽略使用匿名類別初始化的欄位。
