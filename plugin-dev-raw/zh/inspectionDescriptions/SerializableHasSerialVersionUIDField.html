報告實現 Serializable 且不宣告 serialVersionUID 欄位的類。

如果沒有 serialVersionUID 欄位，對該類做出的任何更改都將導致以前序列化的版本不可讀。

範例：


  class Main implements Serializable {
  }

在應用快速修復後：


  class Main implements Serializable {
      private static final long serialVersionUID = -1446398935944895849L;
  }

使用以下選項配置檢查：

  • 列出此檢查不應報告其繼承者的類。 這適用於從超類繼承 Serializable 但不用於序列化的類。
  • 是否忽略 Serializable 匿名類。
