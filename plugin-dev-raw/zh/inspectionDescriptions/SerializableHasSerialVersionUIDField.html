報告實作 Serializable 且不宣告 serialVersionUID 欄位的類別。

如果沒有 serialVersionUID 欄位，對該類做出的任何更改都將導致以前序列化的版本不可讀。

範例：


  class Main implements Serializable {
  }

在套用快速修復後：


  class Main implements Serializable {
      private static final long serialVersionUID = -1446398935944895849L;
  }
當使用 JDK 14 或更高版本的語言級別時，該快速修復還將新增 java.io.Serial 註解。

使用以下選項組態檢查：

  • 列出此檢查不應報告其繼承者的類別。 這適用於從超類別繼承 Serializable 但不用於序列化的類別。
  • 是否忽略 Serializable 匿名類別。
