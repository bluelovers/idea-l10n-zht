在匿名函式中使用 return 關鍵字可以提早跳出該函式時報告。 非本地返回值通過拋出和捕獲 scala.runtime.NonLocalReturnException-s 來實作。 這通常不是程序員的本意。 由於拋出和捕獲異常的隱藏性能開銷，這可能會帶來問題。

此外，它是一個有漏洞的實作：一個捕獲所有異常的處理程序可以攔截 NonLocalReturnException。


def foo(numbers: Seq[Int]): Int =
  numbers.map { n =>
    if (n > 5)
      return 0 // highlighted
    n * 2
  }.sum
