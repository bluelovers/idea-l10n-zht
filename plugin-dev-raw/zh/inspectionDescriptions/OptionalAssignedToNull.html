報告 null 分配給 Optional 變數或從返回 Optional 的方法返回。

建議您使用 Optional.empty()（或 Optional.absent() 用於 Guava）來表示空值。

範例：


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : null;
  }

在套用快速修復後：


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : Optional.empty();
  }

設定檢查：

使用 報告 Optional 與 null 的比較 選項還可以報告optional == null 之類的比較。 雖然在極少數情況下（例如延遲初始化的可選欄位）這可能是正確的，但可選變數通常不會為 null，並且可能應為 optional.isPresent()。

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。
