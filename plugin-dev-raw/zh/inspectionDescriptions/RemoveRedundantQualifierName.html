報告關於類別名、函式和屬性的冗餘限定符（或其部分）。

完全限定名稱是一種明確的關鍵字，用於指定呼叫所參照的物件、函式或屬性。 在可以縮短名稱的上下文中，檢查會告知機會，關聯的“移除冗餘限定符名稱”快速修復可以修改程式碼。

範例：


  package my.simple.name
  import kotlin.Int.Companion.MAX_VALUE

  class Foo

  fun main() {
      val a = my.simple.name.Foo()    // 'Foo' 位於宣告的 'my.simple.name' 軟體套件中，限定符冗餘
      val b = kotlin.Int.MAX_VALUE    // 可以取代為 'MAX_VALUE'，因為它已被匯入
      val c = kotlin.Double.MAX_VALUE // 可以取代為 'Double.MAX_VALUE'，因為內建類型會被自動匯入
  }

在套用快速修復後：


  package my.simple.name
  import kotlin.Int.Companion.MAX_VALUE

  class Foo

  fun main() {
      val a = Foo()
      val b = MAX_VALUE
      val c = Double.MAX_VALUE
  }
