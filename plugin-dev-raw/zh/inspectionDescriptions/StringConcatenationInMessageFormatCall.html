報告用作對 MessageFormat.format() 的調用實參的非常量字符串串聯。

雖然偶爾是有意為之，但這通常是誤用格式化方法，串聯的字符串中包含 { 等特殊字符時，甚至可能導致異常。

此外，有時這可能是在本應鍵入 , 時鍵入了 + 而錯誤串聯字符串格式實參所致。

示例：


  String formatGreeting(String userName, int balance) {
    return MessageFormat.format("Hello, " + userName + "! Your balance is {0}.", balance);
  }

這裡的 userName 解讀為格式字符串的一部分，這可能導致 IllegalArgumentException（例如，如果 userName 為 "{"）。 此調用或許應當替換為 MessageFormat.format("Hello, {0}! Your balance is {1}.", userName, balance)。
