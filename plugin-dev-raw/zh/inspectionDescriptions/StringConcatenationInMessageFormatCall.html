報告用作對 MessageFormat.format() 的呼叫實參的非常數字串串聯。

雖然偶爾是有意為之，但這通常是誤用格式化方法，串聯的字串中包含 { 等特殊字元時，甚至可能導致異常。

此外，有時這可能是在本應鍵入 , 時鍵入了 + 而錯誤串聯字串格式實參所致。

範例：


  String formatGreeting(String userName, int balance) {
    return MessageFormat.format("Hello, " + userName + "! Your balance is {0}.", balance);
  }

這裡的 userName 解讀為格式字串的一部分，這可能導致 IllegalArgumentException（例如，如果 userName 為 "{"）。 此呼叫或許應當取代為 MessageFormat.format("Hello, {0}! Your balance is {1}.", userName, balance)。
