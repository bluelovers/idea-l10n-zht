報告 public 欄位。 不報告常數（標記 static 和 final 的欄位）。

範例：


  class Main {
    public String name;
  }

在套用快速修復後：


  class Main {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
  }

設定檢查：

  • 使用有註解時忽略列表可指定要忽略的註解。 該檢查將忽略包含其中任何註解的欄位。
  • 使用忽略枚舉的 'public final' 欄位選項，可以忽略 enum 類型的 public final 欄位。
