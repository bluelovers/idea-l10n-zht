報告 self 作為限定符但轉義閉包並無此要求的用法。

建議移除冗餘的 self。

範例： 


  struct URL {
      func loadAsync() {
          print(self.path)

          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

在套用快速修復後： 


  struct URL {
      func loadAsync() {
          print(path)

          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
