報告未安全關閉的 Channel 資源，包括任何通過在檔案或套接字資源上調用 getChannel() 而創建的實例。

預設情況下，該檢查假定資源可以由名稱中帶有 'close' 或 'cleanup' 的任何方法關閉。

範例：


  void send(Socket socket) throws IOException {
    SocketChannel channel = socket.getChannel(); //警告
    channel.write(ByteBuffer.wrap("message".getBytes()));
  }

使用以下選項設定檢查：

  • 是否允許在 try 塊內開啟 Channel 資源。 這種樣式不太合意，因為它比在 try 塊前面開啟 Channel 更複雜。
  • 資源能否通過任何將資源作為實參傳遞的方法調用來關閉。
