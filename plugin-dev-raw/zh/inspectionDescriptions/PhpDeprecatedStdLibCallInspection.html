報告對從 PHP 8.1 開始棄用的標準 PHP 庫函數的呼叫。

報告以下呼叫：

  • 將物件作為實參的 key()、current()、next()、prev() 和 reset( ) 呼叫。
  • 不包含實參的 mb_check_encoding() 呼叫。
  • 不包含實參的 get_class()、get_parent_class() 和 get_Called_class() 呼叫。
  • 將布林值作為 $value 實參傳遞的 IntlCalendar::roll() 呼叫。
  • 包含非字串實參的 ctype_*() 呼叫。

有關詳細資訊，請參閱 PHP RFC：PHP 8.1 棄用。
