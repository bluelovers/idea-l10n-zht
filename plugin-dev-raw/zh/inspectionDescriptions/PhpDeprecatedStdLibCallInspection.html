報告對從 PHP 8.1 開始棄用的標準 PHP 庫函數的調用。

報告以下調用：

  • 將物件作為實參的 key()、current()、next()、prev() 和 reset( ) 調用。
  • 不包含實參的 mb_check_encoding() 調用。
  • 不包含實參的 get_class()、get_parent_class() 和 get_Called_class() 調用。
  • 將布林值作為 $value 實參傳遞的 IntlCalendar::roll() 調用。
  • 包含非字串實參的 ctype_*() 調用。

有關詳細資訊，請參閱 PHP RFC：PHP 8.1 棄用。
