報告使用沒有 N 前綴的國家字元的字串文字。

如果沒有 N 前綴，字串將被轉換為資料庫的預設程式碼頁面。 此預設程式碼頁面可能無法識別某些字元。 有關更多資訊，請參閱 docs.microsoft.com 上的“nchar 和 nvarchar”(Transact-SQL) 頁面。

範例（Microsoft SQL 伺服器）：

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

SELECT 'абвгд' AS c; 沒有 N 前綴，'абвгд' 部分將被醒目提示。
