報告用於測試陣列相等性的運算符 == 和 !=。 在大多數情況下，如果原本就要測試陣列內容的相等性，可以通過 java.util.Arrays.equals() 方法來完成。

建議通過快速修復將 == 取代為 java.util.Arrays.equals()。

範例：


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

在套用快速修復後：


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
