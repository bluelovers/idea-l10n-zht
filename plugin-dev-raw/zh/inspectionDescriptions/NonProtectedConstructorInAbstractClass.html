報告 abstract 類的 public 構造函數。

abstract 類的構造函數只能從其子類的構造函數進行調用，將其宣告為 public 可能令人困惑。

該快速修復會將此類構造函數設為 protected。

範例：


  public abstract class Foo {
    public Foo () { // 警告：有 'public' 修飾符
        /* ... */
    }
  }

在套用快速修復後：


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

設定檢查：

使用下面的選項可以忽略非 public 類。
