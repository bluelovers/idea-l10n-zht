報告宣告為 public 的 abstract 類的構造函數。

abstract 類的構造函數只能在其子類的構造函數中調用。 因此，令 abstract 類的構造函數為 public 可能會使人混淆。

該快速修復會將此類構造函數設為 protected。

範例:


  public abstract class Foo {
    public Foo () { // 警告：有 'public' 修飾符
        /* ... */
    }
  }

在套用快速修復後：


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

設定檢查：

使用下面的選項可以忽略非 public 類。
