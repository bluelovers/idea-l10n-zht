報告目標是另一個方法調用的方法調用。 該快速修復會建議引入局部變數。

範例：


  class X {
    int foo(File f) {
      return f.getName().length();
    }
  }

在套用快速修復後：


  class X {
    int foo(File f) {
      final String name = f.getName();
      return name.length();
    }
  }

使用檢查選項來切換以下各項的警告：

  • 欄位初始設定式中的鏈式方法調用，例如 private final int i = new Random().nextInt();

  • 在相同的類型上運算的鏈式方法調用，例如，new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();。
