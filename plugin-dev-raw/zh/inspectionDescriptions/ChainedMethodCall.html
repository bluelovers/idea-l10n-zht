報告目標是另一個方法調用的方法調用。

範例:

  class X { Y foo() { return new Y(); } }
  class Y { X bar() { return new X(); } }

  void foo() {
    Y y = new X().foo(); // 鏈式方法調用
  }

使用檢查選項來切換以下各項的警告：

  • 欄位初始設定式中的鏈式方法調用，例如，private Y y=new X().foo();

  • 在相同的類型上運算的鏈式方法調用，例如，new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();。
