報告對其中的調用由宣告類的子類而不是由宣告類本身限定的 static 欄位的存取。

Java 允許此類限定，但是此類存取可能表明繼承和覆寫之間存在難以捉摸的混淆。

範例：


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Child.foo);
  }

套用快速修復後，結果如下所示：


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Parent.foo);
  }
