報告對其中的調用由聲明類的子類而不是由聲明類本身限定的 static 字段的訪問。

Java 允許此類限定，但是此類訪問可能表明繼承和覆寫之間存在難以捉摸的混淆。

示例：


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Child.foo);
  }

應用快速修復後，結果如下所示：


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Parent.foo);
  }
