如果類別沒有參照其外部類別的成員，則將類別上的 inner 修飾符報告為冗餘。

範例：


  class Foo {
      inner class InnerClass { // 冗餘 `inner` 修飾符
          fun hello() {
              println("Hi!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // 非冗餘 `inner` 修飾符
          fun next(): Any {
              return objects[0]
          }
      }
  }

在套用快速修復後：


  class Foo {
      class InnerClass { // 冗餘 `inner` 修飾符
          fun hello() {
              println("Hi!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // 非冗餘 `inner` 修飾符
          fun next(): Any {
              return objects[0]
          }
      }
  }
