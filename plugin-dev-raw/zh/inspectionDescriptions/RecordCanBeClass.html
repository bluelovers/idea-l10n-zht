報告記錄類並建議將它們轉換為普通類。

通過將快速修復應用到此記錄，此檢查可以將 Java 記錄移動到使用早期 Java 版本的代碼庫。

請注意，生成的類並不完全等同於原始記錄：

  • 結果的類不再擴展 java.lang.Record，因此 instanceof Record 返回 false。
  • Class.isRecord() 和 Class.getRecordComponents() 等反射方法會產生不同的結果。
  • 生成的 hashCode() 實現可能會產生不同的結果，因為故意沒有指定計算記錄 hashCode 的公式。
  • 記錄序列化機制不同於普通類。 有關詳細信息，請參閱 Java 對象序列化規範。

示例：


  record Point(int x, int y) {}

在應用快速修復後：


  final class Point {
    private final int x;
    private final int y;

    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }

    public int x() { return x; }

    public int y() { return y; }

    @Override
    public boolean equals(Object obj) {
      if (obj == this) return true;
      if (obj == null || obj.getClass() != this.getClass()) return false;
      var that = (Point)obj;
      return this.x == that.x &&
             this.y == that.y;
    }

    @Override
    public int hashCode() {
      return Objects.hash(x, y);
    }

    @Override
    public String function toString() { [native code] }() {
      return "Point[" +
             "x=" + x + ", " +
             "y=" + y + ']';
    }
  }

僅當項目或模塊的語言級別為 15（預覽）或更高時，此檢查才會報告。
