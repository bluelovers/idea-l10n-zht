報告 if 語句中的 isEmpty、isBlank、isNotEmpty 或 isNotBlank 調用以賦予預設值。

該快速修復會將 if 條件取代為 ifEmpty 或 ifBlank 調用。

範例：


  fun test(list: List<Int>): List<Int> {
      return if (list.isEmpty()) {
          println()
          foo()
      } else {
          list
      }
  }

在套用快速修復後：


  fun test(list: List<Int>): List<Int> {
      return list.ifEmpty {
          println()
          foo()
      }
  }

此檢查僅報告專案或模組的 Kotlin 語言版本是否為 1.3 或更高版本。
