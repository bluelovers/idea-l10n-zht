如果在類周圍的代碼中存在具有相同簽名的方法，則報告從匿名類、內部類或局部類對超類方法的調用。 在這種情況下，看似調用了來自周圍代碼的方法，而實際上調用的是來自超類的方法。

為了闡明代碼的意圖，建議對方法調用添加顯式的 super 限定符。

示例：


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        ambiguous(); //警告
      }
    }
  }

在應用快速修復后：


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
