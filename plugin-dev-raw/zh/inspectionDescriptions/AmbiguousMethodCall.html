如果包含類中存在簽名相同的方法，則報告對內部類或匿名類中任何 super 方法的調用。

在本例中，很容易缺失 super 方法，建議包含類中的方法引用此調用。 為闡明意圖，建議在方法調用中添加顯式的 super 限定符。

範例:


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        ambiguous(); //警告
      }
    }
  }

在套用快速修復後：


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
