報告可以安全地設為 final 的欄位。 所有 final 欄位都應具有一個不會更改的值，從而使程式碼更易於推理。

為避免分析開銷過大，此檢查僅報告欄位是否具有 private 修飾符或者是否在區域類別或匿名類別中定義。 如果滿足以下條件，欄位可以為 final：

  • 欄位為 static，並在其宣告或 static 初始設定式中進行了一次初始化。
  • 欄位為非 static，並在其宣告、實例初始設定式或每個建構函式中進行了一次初始化
並且欄位未在其他任何地方進行修改。

範例：


  public class Person {
    private String name; // 可以為 final

    Person(String name) {
      this.name = name;
    }

    public String getName() {
      return name;
    }
  }

在套用快速修復後：


  public class Person {
    private final String name;

    Person(String name) {
      this.name = name;
    }

    public String getName() {
      return name;
    }
  }

使用“註解”按鈕修改假定隱式欄位寫入的註解列表。
