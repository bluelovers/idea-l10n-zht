報告採用陣列類型參數的 JUnit assertEquals() 方法的調用。 此類方法會比較陣列的標識，而不是陣列的內容。

建議使用 其中一種 assertArrayEquals() 方法來比較陣列的內容。

範例:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

在套用快速修復後：


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
