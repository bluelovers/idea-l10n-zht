報告對 JSP 檔案中預設軟體套件中類的任何引用。

範例:


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // 錯誤
    new com.example.GoodClass(...); // 優良
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

注意： 在 JSP 編譯期間不會解析此類引用，因為 JSP 的編譯類通常會放在非預設包中（例如，Tomcat 的 “org.apache.jsp"，WebSphere 的 “com.ibm._jsp”），但在 Java 中不允許從預設包匯入。
