報告 CASE 和 COALESCE 調用可以互換的情況。 此檢查具有以下意圖操作：取代為 'COALESCE' 調用和相反的 取代為 CASE 表達式。

範例（MySQL）：

SELECT
  -- this CASE may be replaced by COALESCE
	CASE
		WHEN C1 IS NOT NULL THEN C1
		ELSE 0
		END
FROM dual;

在該範例中，CASE 語句可以取代為 SELECT COALESCE(C1, 0)，這會產生相同的結果。

如果您更喜歡使用 CASE 表達式，請在檢查頁面上選擇 CASE 表達式優先於 COALESCE 函數選項。
