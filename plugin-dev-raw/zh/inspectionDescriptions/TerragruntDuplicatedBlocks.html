報告在檔案根目錄下定義的共享相同標籤（名稱）的重複 Terragrunt 塊，例如 dependency、feature、generate、include、stack 或 unit。

該檢查會醒目提示多個使用相同標籤的相同類型的塊。 僅檢查頂層塊。

範例：


unit "vpc" {
  source = "git::git@github.com:acme/infrastructure-catalog.git//units/vpc?ref=v0.0.1"
  path   = "vpc"
}

# duplicate
unit "vpc" {
  source = "git::git@github.com:acme/infrastructure-catalog.git//units/database?ref=v0.0.1"
  path   = "database"
  values = {
    engine   = "postgres"
    version  = "13"
    vpc_path = "../vpc"
  }
}

可用的快速修復：

  • 導覽到重複項
  • 重新命名塊
  • 顯示所有重複項
  • 刪除塊
