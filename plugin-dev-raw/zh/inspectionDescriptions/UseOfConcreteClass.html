報告具體類而非接口的用法。 此類宣告可能代表抽象失敗，並可能使測試更加困難。

此檢查不會報告其類來自系統或第三方庫的宣告。 equals() 方法實作中不會報告轉換、instanceof 和局部變數。 此外，clone() 方法實作中不會報告轉換。

範例:


  interface Entity {}
  class EntityImpl implements Entity {}

  void processObject(Object obj) {
    // 警告：具體類的 instanceof
    if (obj instanceof EntityImpl) {
      // 警告：轉換為具體類，
      // 而不是接口
      processEntity((EntityImpl)obj);
    }
  }
  // 警告：具體類的參數
  void processEntity(EntityImpl obj) {
  }

使用下面的第一個複選框忽略對抽象類的強制轉換。

使用後續複選框來控制報告問題的上下文。
