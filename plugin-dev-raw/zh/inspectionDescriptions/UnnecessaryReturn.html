報告返回 void 的建構函式和方法結尾的 return 語句。 這些語句冗餘，可以安全移除。

此檢查不會在 JSP 檔案中報告。

範例：


  void message() {
    System.out.println("Hello World");
    return;
  }

在套用快速修復後： 


  void message() {
    System.out.println("Hello World");
  }

使用在具有 'else' 分支的 'if' 語句的 then 分支中忽略選項可以忽略也有 else 分支的 if 語句的 then 分支中的 return 語句。
