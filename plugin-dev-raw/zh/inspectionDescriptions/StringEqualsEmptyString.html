報告 equals() 被調用以將 String 與空字串進行比較的情況。 在這種情況下，使用 .isEmpty() 更好，因為它可以準確顯示您正在檢查的內容。

範例：


  void checkString(String s){
    if ("".equals(s)) throw new IllegalArgumentException();
  }

在套用快速修復後：


  void checkString(String s){
    if (s != null && s.isEmpty()) throw new IllegalArgumentException();
  }

"".equals(str) 在 str 為空時返回 false。 為了安全起見，當 equals() 實參可以為空時，此檢查的快速修復會插入一個顯式 null 檢查。 使用該選項可使檢查忽略此類情況。
