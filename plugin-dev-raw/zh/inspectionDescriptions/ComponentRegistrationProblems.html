報告程式碼和 plugin.xml 描述符中未正確註冊的延伸模組組件（組件和動作）。

報告的問題如下：

  • 未解析的組件/動作實作類別 (plugin.xml)
  • 未解析的組件接口類別 (plugin.xml)
  • 無法將組件實作類別分配給其接口類別 (plugin.xml)
  • 組件註冊不正確，例如，將 ProjectComponent 註冊為應用程式組件，或者類別完全沒有實作任何組件接口（plugin.xml 和 Java 程式碼）
  • 類別修飾符不正確：動作類別必須為 public，並且必須有 public 無實參建構函式。 此外，動作和組件實作類別不得為 abstract（plugin.xml 和 Java 程式碼）
  • 接口類別重複：只能將類別用作接口類別一次 (plugin.xml)

設定檢查：

  • 使用檢查延伸模組描述符(plugin.xml) 選項可以停用 plugin.xml 描述符中的檢查。
  • 使用檢查 Java 動作選項可以停用動作程式碼中的檢查。
  • 使用檢查 Java 程式碼選項可以停用程式碼中的所有檢查。
