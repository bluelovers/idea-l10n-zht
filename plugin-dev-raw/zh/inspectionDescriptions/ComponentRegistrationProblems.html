報告程式碼和 plugin.xml 描述符中未正確註冊的套件組件（組件和操作）。

報告的問題如下：

  • 未解析的組件/操作實作類 (plugin.xml)
  • 未解析的組件接口類 (plugin.xml)
  • 無法將組件實作類分配給其接口類 (plugin.xml)
  • 組件註冊不正確，例如，將 ProjectComponent 註冊為應用程式組件，或者類完全沒有實作任何組件接口（plugin.xml 和 Java 程式碼）
  • 類修飾符不正確：操作類必須為 public，並且必須有 public 無實參構造函數。 此外，操作和組件實作類不得為 abstract（plugin.xml 和 Java 程式碼）
  • 接口類重複：只能將類用作接口類一次 (plugin.xml)

設定檢查：

  • 使用檢查套件描述符(plugin.xml) 選項可以禁用 plugin.xml 描述符中的檢查。
  • 使用檢查 Java 操作選項可以禁用操作程式碼中的檢查。
  • 使用檢查 Java 程式碼選項可以禁用程式碼中的所有檢查。
