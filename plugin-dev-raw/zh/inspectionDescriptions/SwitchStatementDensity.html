報告 switch 標籤與可執行語句的比率太低的 switch 語句或表達式。

此類別 switch 語句可能令人困惑，或許應對其進行重構。

範例：


    switch (i) { // 1 個 case，5 個可執行語句 -> 20% 的密度
        case 1:
            System.out.println("1");
            System.out.println("2");
            System.out.println("3");
            System.out.println("4");
            System.out.println("5");
            break;
    }

使用分支的最小密度欄位可以指定 switch 標籤與可執行語句的允許比率。
