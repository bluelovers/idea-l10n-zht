報告從方法參數到陣列或可變類型（如 Collection、Date、Map、Calendar 等）的返回或賦值

由於這種類型是可變的，此結構可能會導致來自所屬類別之外的物件狀態發生意外修改。 儘管出於性能原因，此結構可能很有用，但它本質上很容易出現錯誤。

報告了以下可變類型：

  • java.util.Date
  • java.util.Calendar
  • java.util.Collection
  • java.util.Map
  • com.google.common.collect.Multimap
  • com.google.common.collect.Table

該快速修復會新增對欄位的 .clone() 方法的呼叫。

範例：


  class Log {
    String[] messages;
    ...

    String[] getMessages() {
      return messages; // 警告：返回 String[] 欄位 'messages'
    }
  }

在套用快速修復後：


  class Log {
    String[] messages;
    ...

    String[] getMessages() {
      return messages.clone();
    }
  }
