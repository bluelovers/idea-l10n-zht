報告出現在具有無效簽名或無效類型欄位的類或方法上的 Seam 註解。

  • @Name 在接口和 abstract 類上無效。 在註解的類中要求一個無實參構造函數。

  • @In 必須出現在欄位或 setter 方法上，例如 void setUser(User user)。

    • void 返回類型
    • 一個實參
    • 非 static
    • 名稱必須以 set 開頭

  • @Out 必須出現在欄位或 getter 方法上，例如 User getUser()。

    • 非 void 返回類型
    • 無實參
    • 非 static
    • 名稱必須以 get 開頭，如果返回類型為 boolean，則可選 is

  • @DataModel 必須出現在欄位或 getter 方法上（與 @Out 註解相同）。 欄位類型或返回類型必須是以下之一：

    • java.util.List
    • java.util.Set
    • java.util.Map

  • @DataModelSelection 必須出現在 setter 方法上。

  • @DataModelSelectionIndex 必須出現在 setter 方法上。

  • @Factory 和 @Unwrap 必須出現在具有以下簽名的方法上：

    • 非 void 返回類型
    • 沒有參數或單個 org.jboss.seam.Component 參數（或其超類型）
    • 非 static

  • @Create 必須出現在具有以下簽名的方法上：

    • 沒有參數或單個 org.jboss.seam.Component 參數（或其超類型）
    • 非 static

  • @Destroy 必須出現在具有以下簽名的方法上：

    • void 返回類型
    • 無形參
    • 非 static
