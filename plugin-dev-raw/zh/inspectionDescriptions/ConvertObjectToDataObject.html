報告可轉換為 data object 的 object

如果 object 使用 java.io.Serializable 進行註解，則 data object 會自動生成 toString、equals、hashCode 和 readResolve

主要在兩種情況下，您應該考慮將 object 轉換為 data object。 第一種情況是當自訂 toString 返回類的名稱時。 第二種情況是當 object 實作 java.io.Serializable 時

範例：


    class Foo {
        override fun toString(): String = "Foo"
    }

在套用快速修復後：


    data class Foo

僅當專案或模組的 Kotlin 語言級別為 1.8 或更高時，此檢查才會報告
