報告使用 instanceof 表達式來測試 catch 塊中參數類型的情況。

測試 catch 參數的類型通常最好使用單獨的 catch 塊，而不是使用 instanceof。

範例：


  void foo(Runnable runnable) {
    try {
        runnable.run();
    } catch (Throwable throwable) {
        if (throwable instanceof NoClassDefFoundError) { // 警告：'catch' 參數 'throwable' 上的 'instanceof'
            System.out.println("Class not found!");
        }
    }
  }
