報告由頂級 JSP Scriptlet 拋出的異常。

範例：


  <%!
  String getFormattedDate() {
    ...
    throw new IllegalArgumentException(...); // 錯誤
  }
  %>
  <html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title>Hello, JSP!</title>
  </head>
  <body>
  <i>Today is <%= getFormattedDate() %></i>
  </body>
  </html>

注意：此類別 JSP 頁面會編譯，因為 servlet 容器將所有 JSP 程式碼包裝在 try {} catch() {} 塊中，但它們在伺服器上部署時會產生執行時異常。
