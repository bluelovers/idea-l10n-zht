報告 YAML 檔案中已棄用的鍵。

僅當存在與相應 YAML 檔案關聯的 JSON 架構時，才會檢查是否棄用。

請注意，JSON 架構規範中尚未定義棄用機制，並且此檢查使用非標準的 deprecationMessage 擴展。

方案棄用示例：


  {
    "properties": {
      "SomeDeprecatedProperty": {
        "deprecationMessage": "Baz",
        "description": "Foo bar"
      }
    }
  }

以下是帶有相應警告的範例：


  SomeDeprecatedProperty: some value
