報告可以析構的宣告。

範例：


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { my ->
          println(my.second)
          println(my.third)
      }
  }

該快速修復會析構宣告並引入具有來自相應類的名稱的新變數：


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { (_, second, third) ->
          println(second)
          println(third)
      }
  }
