報告 Serializable 和 Externalizable 類別中不適合使用 java.io.Serial 註解進行註解的方法和欄位。

範例：


class Test implements Serializable {
  @Serial // 註解的欄位不是序列化機制的一部分，因為它並非 final
  private static long serialVersionUID = 7874493593505141603L;

  @Serial // 註解的方法不是序列化機制的一部分，因為它並非 private
  void writeObject(ObjectOutputStream out) throws IOException {
  }
}


class Test implements Externalizable {
  @Serial // 該註解方法不是序列化機制的一部分，因為它在 Externalizable 類別內部
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

有關所有有效情況的資訊，請參閱 java.io.Serial 的文檔。

僅當專案或模組的語言級別為 14 或更高時，此檢查才會報告。
