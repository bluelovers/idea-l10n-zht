報告陣列元素類型的類別中有常數時已知長度為零的陣列分配。 由於零長度陣列不可變，您可以重用相同的陣列實例來節省記憶體。

範例：


  class Item {
    // 可重用的 public 零長度陣列常數 
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // 不必要的零長度陣列建立
      return new Item[0];
    }
  }

在套用快速修復後：


  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }
