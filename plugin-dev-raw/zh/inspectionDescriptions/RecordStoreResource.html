報告未在 try 塊前開啟並在相應的 finally 塊中關閉的 Java ME javax.microedition.rms.RecordStore 資源。

如果在資源關閉之前拋出異常，此類別資源可能會被無意泄漏。

此檢查適用於 Java ME 和其他資源高度受限的環境。 不加考慮地套用此檢查的結果可能會對程式碼清晰度和設計產生負面影響。

範例：


  void foo1() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // 警告
  }
  void foo2() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // 無警告
      try {
          /* ... */
      } finally {
          rs.closeRecordStore();
      }
  }
