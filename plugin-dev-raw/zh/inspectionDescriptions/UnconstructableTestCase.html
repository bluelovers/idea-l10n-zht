報告無法由標準的 JUnit 測試執行程序建置的 JUnit 測試類。

JUnit 4 測試類需要為 public 並包含一個 public 無實參構造函數或完全不包含構造函數（隱式的預設構造函數），且沒有其他 public 構造函數。 JUnit 3 測試類需要為 public，並且需要包含一個 public 無實參構造函數 或是帶有單個 String 類型參數的 public 構造函數，以調用符合的 super 構造函數。 否則，測試類無法由標準的 JUnit 測試執行程序執行。

範例：


public class MyTest {

  private MyTest() {} // no-arg 構造函數為 private

  @Test
  public void testSomething() {
    assertEquals(1, 1);
  }
}
