報告派生類中與超類的欄位同名的欄位。 Java 欄位無法在派生類中覆寫，因此派生類中的欄位將對超類隱藏該欄位。

由於進行此類命名，在基類有意使用同名欄位的情況下，您可能會誤用派生類的欄位。

建議通過快速修復重命名派生類中的欄位。

範例：

class Parent {
  Parent parent;
}
class Child extends Parent {
  Child parent;
}

您可以為此檢查設定以下選項：

  1. 忽略不可存取欄位 - 指示此檢查應報告所有名稱衝突，還是只報告與子類可見的欄位的衝突。
  2. 忽略隱藏 static 欄位的 static 欄位 - 忽略隱藏基類中的 static 欄位的 static 欄位。
