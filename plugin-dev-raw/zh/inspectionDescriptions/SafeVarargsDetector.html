報告所有包含可變 arity 且可以註解為 @SafeVarargs 的方法。 @SafeVarargs 註解禁止有關在調用點創建參數化陣列的未檢查的警告。

範例：


  public class Foo<T> {
    private List<T> list = new ArrayList<>();

    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

在套用快速修復後：


  public class Foo<T> {
    private List<T> list = new ArrayList<>();

    @SafeVarargs
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Java 1.6 或更早版本的 JVM 不支持此註解。
