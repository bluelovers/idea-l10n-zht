報告可將動態調用取代為構造函數調用的情況。

範例：


  import akka.actor._
  
  class Foo(foo: String, bar: Int) extends AbstractActor
  
  object Foo {
    def props(foo: String, bar: Int): Props = Props(classOf[Foo], foo, bar)
  }
  
  Foo.props("foo", 42)

在套用快速修復後：


  import akka.actor._
  
  class Foo(foo: String, bar: Int) extends AbstractActor
  
  object Foo {
    def props(foo: String, bar: Int): Props = Props(new Foo(foo, bar))
  }
  
  Foo.props("foo", 42)
