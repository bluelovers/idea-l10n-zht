報告實作兩個或多個特徵（包含具有相同簽名的方法）的類別。

呼叫此類別方法可能產生意外的結果。

該快速修復會新增顯式覆寫方法。

範例：


  trait T1 {
      def foo() {}
  }

  trait T2 {
      def foo() {}
  }

  class X implements T1, T2 {}

  // 將呼叫 T2.foo()
  new X().foo()

在套用快速修復後：


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }
