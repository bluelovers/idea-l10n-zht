報告實作兩個或多個特徵（包含具有相同簽名的方法）的類。

調用此類方法可能產生意外的結果。

快速修復將添加顯式改寫方法。

範例:


  trait T1 {
      def foo() {}
  }

  trait T2 {
      def foo() {}
  }

  class X implements T1, T2 {}

  // T2.foo() will be called
  new X().foo()

在套用快速修復後：


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }
