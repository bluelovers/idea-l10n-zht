報告以下 JPA 規範錯誤：
  • 為一個方法或欄位組態了多個特性
  • Embeddable 物件中使用了非法特性 (JPA 1.0)
  • 持久特性元資料不在欄位或屬性 getter 上指定
  • 沒有基於屬性存取的 setter 或 getter

範例：


  @Entity
  @Table(name = "TABLE")
  public class JavaEntity {

    private Integer id; // 錯誤：getter 和 setter 都應該存在

    public Integer getId() { ... }



    private String data;

    public String getData() { ... }

    @Basic // 錯誤：持久註解應該放在欄位或 getter 方法上
    public void setData(String data) { ... }
  }
