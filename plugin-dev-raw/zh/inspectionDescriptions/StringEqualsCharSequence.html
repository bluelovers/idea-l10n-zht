報告對 CharSequence 作為實參的 String.equals() 的調用。

對於 String 實參，String.equals() 只能返回 true。 要比較包含非 String CharSequence 實參的 String 的內容，請使用 contentEquals() 方法。

範例:

  boolean equals(String s, CharSequence ch) {
    return s.equals(ch);
  }

在套用快速修復後：

  boolean equals(String s, CharSequence ch) {
    return s.contentEquals(ch);
  }
