報告 && 和 || 表達式以及 if 語句的分支中的重複條件。 這些重複的條件表達式有時是故意使用的，但通常因疏忽大意所致。

範例:


  boolean result = digit1 != digit2 || digit1 != digit2;

要忽略可能產生副作用的條件，請使用忽略有副作用的條件 選項。 禁用此選項可能會導致誤報，例如，當同一方法在後續調用中返回不同的值時。

範例:


  if (iterator.next() != null || iterator.next() != null)
    System.out.println("Got it");
  }

由於 iterator.next() 可能存在副作用（在範例中），只有在禁用 忽略有副作用的條件 選項時才會觸發警告。
