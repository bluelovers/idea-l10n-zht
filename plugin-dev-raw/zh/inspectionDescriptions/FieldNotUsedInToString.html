報告在類別的 toString() 方法中未使用的任何欄位。

此檢查對於尋找在 toString() 方法建立後新增且 toString() 方法未更新的欄位很有用。 該快速修復會重新生成 toString() 方法。

在 生成 | toString() 對話框中，可以從此檢查中排除欄位。 如果在程式碼生成中啟用 getter 選項，則此檢查還將檢查 getter 方法是否存在問題。

範例：


  public class Relevant {
    private String name; // 未在 toString() 中使用
    private int index;
    private int length;

    @Override
    public String toString() {
        return "Relevant{" + "index=" + index +
          ", length=" + length + '}';
    }
  }

在套用快速修復後：


  public class Relevant {
    private String name;
    private int index;
    private int length;

    @Override
    public String toString() {
        return "Relevant{" + "name='" + name + '\'' +
          ", index=" + index + ", length=" + length + '}';
    }
  }
