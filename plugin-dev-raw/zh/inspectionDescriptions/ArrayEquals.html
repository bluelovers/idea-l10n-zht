報告對 equals() 的調用以比較兩個陣列。

對陣列調用 equals() 可以比較同一性，相當於使用 ==。 使用 Arrays.equals() 比較兩個陣列的內容，或使用 Arrays.deepEquals() 比較多維陣列的內容。

範例:

  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

在套用快速修復後：

  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }
