報告任何不必要的括號。

如果刪除括號後表達式的評估順序保持不變，則不需要括號。

範例:


  int n = 3 + (9 * 8);

在套用快速修復後：


  int n = 3 + 9 * 8;

設定檢查：

  • 使用 忽略澄清括號 選項可忽略有助於明確二元表達式的括號。 括號澄清被括起來的表達式是一個 instanceof 更大表達式一部分的表達式或包含與父表達式不同運算符。
  • 使用 忽略條件表達式條件周圍的括號 選項以忽略條件表達式條件周圍的任何括號。 一些編碼標准規定所有這些條件都必須用括號括起來。
  • 使用忽略單個非正式類型 lambda 參數周圍的括號 選項可忽略 lambda 表達式中單個 lambda 參數周圍的括號。
