報告任何不必要的圓括號。

如果移除圓括號後表達式的評估順序保持不變，則不需要圓括號。

範例：


  int n = 3 + (9 * 8);

在套用快速修復後：


  int n = 3 + 9 * 8;

設定檢查：

  • 使用忽略澄清圓括號選項可忽略有助於明確二元表達式的圓括號。 圓括號澄清被括起來的表達式是一個 instanceof 更大表達式一部分的表達式或包含與父表達式不同運算符。
  • 使用忽略條件表達式條件周圍的圓括號選項以忽略條件表達式條件周圍的任何圓括號。 一些編碼標准規定所有這些條件都必須用圓括號括起來。
  • 使用忽略單個非正式類型 lambda 參數周圍的圓括號選項可忽略 lambda 表達式中單個 lambda 參數周圍的圓括號。
