報告字串模板中用於括簡單關鍵字的大括號的用法。

使用“移除大括號”快速修復可移除冗餘的大括號。

範例：


  fun redundant() {
     val x = 4
     val y = "${x}" // <== 冗餘
  }

  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== 可以
  }

在套用快速修復後：


  fun redundant() {
     val x = 4
     val y = "$x"
  }

  fun correctUsage() {
      val x = "x" <== 已更新
      val y = "${x.length}"
  }
