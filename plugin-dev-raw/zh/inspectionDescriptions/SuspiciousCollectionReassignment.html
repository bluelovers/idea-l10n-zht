報告唯讀 Collection 上的增強型賦值 (+=) 表達式。

唯讀 Collection 上的增強型賦值 (+=) 表達式不會修改目標集合，它會在背景創建一個新集合，這可能有誤導性並產生性能問題。

將類型更改為可變快速修復可用於自動修正程式碼。

範例：


  fun test() {
      var list = listOf(0)
      list += 42 // 創建新列表，變數 'list' 仍僅包含 '0'
  }

在應用快速修復後：


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
