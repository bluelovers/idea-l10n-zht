報告呼叫 ReactorDebugAgent.init() 時 Hooks.onOperatorDebug() 的用法。

如果 ReactorDebugAgent 在類別路徑中，並且 spring.reactor.debug-agent.enabled 為 true（預設值），啟動時自動呼叫 ReactorDebugAgent.init()。 同時使用 Hooks.onOperatorDebug() 和 ReactorDebugAgent.init() 會導致 Reactor 生成雙倍的偵錯堆疊幀。 此外，Hooks.onOperatorDebug() 會導致性能開銷。

該快速修復會移除對 Hooks.onOperatorDebug() 的呼叫。

範例：


  public void hook() {
    Hooks.onOperatorDebug(); // 報告 '初始化 ReactorDebugAgent 時呼叫 Hooks.onOperatorDebug()'
  }

在套用快速修復後：


  public void hook() {
  }

此檢查僅在專案或模組的 Spring Boot 版本為 2.2.0 或更高時觸發。
