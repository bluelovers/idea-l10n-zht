報告已知為常數的變數的所有用法。

如果（讀取）使用的變數被 if、while 或 for 語句包圍，且存在該變數與常數相比較的 == 條件時，就會發生這種情況。 在這種情況下，使用的已知為常數的變數可以取代為實際常數。

範例：


    private static void foo(double number) {
        if (number == 1.0) {
            f(number);
        }
    }
    private static void f(double number) {}

在套用快速修復後：


    private static void foo(double number) {
        if (number == 1.0) {
            f(1.0);
        }
    }
    private static void f(double number) {}
