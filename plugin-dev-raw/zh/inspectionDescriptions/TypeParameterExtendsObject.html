報告顯式宣告為擴展 java.lang.Object 的類型參數和萬用字元類型實參。

此類別 'extends' 子句是冗餘的，因為 java.lang.Object 是所有類別的父類型。

範例：

  class ClassA<T extends Object> {}

如果由於註解而需保留 'extends Object' 子句，請停用註解 java.lang.Object 時忽略選項。 這可能會有用，例如在您使用空值分析器並且 'extends Object' 子句包含 @Nullable/@NotNull 註解時。

範例：

  class MyClass<T extends @NotNull Object> {}
