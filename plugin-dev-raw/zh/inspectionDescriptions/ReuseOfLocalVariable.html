報告使用與其原始用途無關的新值覆蓋其值而進行“重用”的局部變量。

此類局部變量重用可能令人困惑，因為局部變量的預期語義可能隨每種用法而異。 如果由于代碼變更，視為被覆蓋的值實際看似處于活動狀態，它也可能容易出現錯誤。 保持變量生命周期盡可能短不失為一種好做法，不要為了簡潔而重用局部變量。

示例：


    void x() {
        String s = "one";
        System.out.println("s = " + s);
        s = "two"; //重用局部變量 's'
        System.out.println("s = " + s);
    }
