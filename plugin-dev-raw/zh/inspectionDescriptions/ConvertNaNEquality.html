報告使用 Float.NaN 或 Double.NaN 的相等檢查（應取代為 isNaN() 檢查）。

根據 IEEE 754，對照 NaN 進行的相等檢查始終返回 false，即便對 NaN == NaN 也如此。 因此，這種檢查很可能是錯誤的。

快速修復可將比較取代為使用不同比較技術的 isNaN() 檢查，並能夠正確處理 NaN 值。

範例：


  fun check(value: Double): Boolean {
      return Double.NaN == value
  }

套用修正後：


  fun check(value: Double): Boolean {
      return value.isNaN()
  }
