報告不正確的 Spring Shell 指令宣告：

  • 缺少指令描述
  • 指令不在 org.springframework.shell.standard.@ShellComponent 內

範例：


// 由於缺少 @ShellComponent，將報告所有方法
class MathCommands {

  @ShellMethod("")
  public int sum(int a, int b) { // 由於描述為空，將報告方法
    return a + b;
  }

  @ShellMethod
  public int mul(int a, int b) { // 由於缺少描述，將報告方法
    return a * b;
  }
}
