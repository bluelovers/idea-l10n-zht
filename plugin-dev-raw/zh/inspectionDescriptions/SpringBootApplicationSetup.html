報告預設軟體套件中的@SpringBootApplication 和冗餘的@EnableAutoConfiguration 或 @ComponentScan 註解。

該快速修復會移除冗餘註解。

範例：


@SpringBootApplication
@ComponentScan // 報告 '冗餘宣告: @SpringBootApplication 已暗示 @ComponentScan'
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

在套用快速修復後：


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }
