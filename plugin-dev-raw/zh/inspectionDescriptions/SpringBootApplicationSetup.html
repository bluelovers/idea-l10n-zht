報告預設包中的@SpringBootApplication 和冗餘的@EnableAutoConfiguration 或@ComponentScan 註解。

快速修復刪除了冗餘的註解。

範例:


@SpringBootApplication
@ComponentScan // 報告“冗餘宣告："@SpringBootApplication 已經暗示 @ComponentScan"
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

在套用快速修復後：


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }
