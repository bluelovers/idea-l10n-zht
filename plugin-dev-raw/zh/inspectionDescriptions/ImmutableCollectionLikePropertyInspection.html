報告在 JPA 實體中使用不可變集合。

檢測使用 @OneToMany 或 @ManyToMany 註釋的屬性，它們具有不可變集合類型（List， Set，Map）。 建議使用可變集合類型（MutableList，MutableSet，MutableMap）來確保與 Hibernate 的兼容性。

嵌入程式碼段： 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: List = listOf(),
)
套用快速修復後: 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: MutableList = mutableListOf(),
)
