報告 JPA 實體中不可變集合的用法。

檢測使用具有不可變集合類型（List、Set、Map）的 @OneToMany 或 @ManyToMany 註解的屬性。 建議使用可變集合類型（MutableList、MutableSet、MutableMap）來確保 Hibernate 兼容性。

嵌入程式碼段： 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: List = listOf(),
)
套用該快速修復後： 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: MutableList = mutableListOf(),
)
