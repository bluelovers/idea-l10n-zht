報告不從接收器存取任何內容的冗餘 with 函數調用。

範例：


  class MyClass {
      fun f(): String = ""
  }

  fun testRedundant() {
      with(c) { // <== 'with' 冗餘，因為 'c' 未使用
          println("1")
      }
  }

  fun testOk() {
      val c = MyClass()
      with(c) { // <== 可以，因為 'f()' 是有效的 'c.f()'
          println(f())
      }
  }
