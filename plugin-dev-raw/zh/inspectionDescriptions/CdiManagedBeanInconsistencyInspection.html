報告不正確的 Bean 定義規則。

頂級 Java 類別可以為簡單的 Bean，如果：

  • 它不是參數化類型。
  • 它不是一個非 static 內部類別。
  • 它是一個具體類別，或者使用 @Decorator 進行註解。
  • 擁有適當的建構函式：該類具有沒有參數的建構函式，或宣告使用 @Initializer 註解的建構函式。
  • 它不實現以下任何接口： javax.servlet.Servlet、 javax.servlet.Filter、 javax.servlet.ServletContextListener、 javax.servlet.http.HttpSessionListener、 javax.servlet.ServletRequestListener、 javax.ejb.EnterpriseBean。
  • 它不擴展 javax.faces.component.UIComponent。
