報告 kotlin.coroutines.suspendCoroutine 的用法，並建議在出現 kotlinx.coroutines 相依項時將其取代為 kotlinx.coroutines.suspendCancellableCoroutine。

suspendCancellableCoroutine 變體提供了內建的取消支援，這對於結構化並發和正確的資源管理非常重要。

範例：


    suspend fun usage(): String {
        return suspendCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }

在套用快速修復後：


    suspend fun usage(): String {
        return suspendCancellableCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }
