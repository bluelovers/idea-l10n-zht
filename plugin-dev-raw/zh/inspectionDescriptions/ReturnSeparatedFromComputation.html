報告返回區域變數的 return 語句，其中變數的值在同一方法中的其他位置計算。

快速修復通過將 return 語句移動到計算變量值的位置來內聯返回的變數。 當返回值無法內聯到 return 語句中時，快速修復嘗試將 return 語句移動到盡可能接近計算的返回值的位置。

範例：


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      n = i;
      break;
    }
  }
  return n;

在套用快速修復後：


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      return i;
    }
  }
  return n;
