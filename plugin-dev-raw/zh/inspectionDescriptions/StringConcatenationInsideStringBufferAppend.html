報告用作 StringBuffer.append()、StringBuilder.append() 或 Appendable.append() 的實參的 String 串聯。

此類別呼叫可以有利地轉變為現有 StringBuffer/Builder/Appendable 中的鏈式追加呼叫，從而節省額外的 StringBuffer/Builder 分配成本。 此檢查將忽略編譯時求值的 String 串聯，在這種情況下，轉換只會降低性能。

範例：


  void bar(StringBuilder builder, String name) {
    builder.append("Hello," + name);    //警告
    builder.append("Hello," + "world"); //無警告
  }
