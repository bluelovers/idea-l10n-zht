根據不同的樣式報告 Collection.toArray() 調用的用法，並建議套用首選樣式。

將集合轉換為數組有兩種樣式:

  • 預先確定大小的陣列，例如 c.toArray(new String[c.size()])
  • 空陣列，例如 c.toArray(new String[0])
    •

    在舊的 Java 版本中，建議使用預先確定大小的陣列，因為創建適當大小的陣列所需的反射調用非常慢。

    不過，自 OpenJDK 6 的最新更新以來，該調用內在化，使得空陣列版本的性能與預先確定大小的版本相同，有時甚至更高一籌。 此外，傳遞預先確定大小的陣列對於並發集合或同步集合很危險，因為 size 和 toArray 調用之間可能會發生資料競爭。 如果在操作過程中並發收縮該集合，可能導致陣列末尾出現額外的 null。

    使用檢查選項選擇首選樣式。
