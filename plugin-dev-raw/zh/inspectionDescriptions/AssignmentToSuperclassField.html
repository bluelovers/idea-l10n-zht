報告對超類中從子類構造函數中宣告的欄位的賦值或修改。

最好在其自己的構造函數中初始化超類的欄位，並在子類中委託給該構造函數。 如果它在構造後沒有改變，這還將允許宣告一個欄位 final。

範例：


  class Super {
    int x;
  }
  class Sub extends Super {
    Sub(int _x) {
      // Warning: x is declared in a superclass
      x = _x;
    }
  }

要避免此問題，請宣告超類構造函數：


  class Super {
    final int x;

    Super(int _x) {
      x = _x;
    }
  }
  class Sub extends Super {
    Sub(int _x) {
      super(_x);
    }
  }
