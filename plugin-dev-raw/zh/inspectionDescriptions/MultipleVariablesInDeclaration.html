報告在單個宣告中宣告的多個變數，並建議為每個變數建立單獨的宣告。

一些編碼標準禁止此類別宣告。

範例：


  int x = 1, y = 2;

在套用快速修復後：


  int x = 1;
  int y = 2;

組態檢查：

  • 使用 忽略 'for' 迴圈宣告 選項可忽略在 'for' 迴圈語句的初始化中宣告的多個變數，例如： 
    
  for (int i = 0, max = list.size(); i > max; i++) {}
  • 使用 僅在單個宣告中對不同的陣列維度發出警告 選項，以僅在單個宣告中宣告具有不同陣列維度的變數時發出警告，例如： 
    
  String s = "", array[];
