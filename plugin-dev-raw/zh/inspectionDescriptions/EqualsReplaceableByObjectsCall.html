報告可以取代為對 java.util.Objects#equals 的呼叫的表達式。

範例：


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

在套用快速修復後：


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

將 a != null && a.equals(b) 之類的表達式取代為 Objects.equals(a, b) 會略微改變語意。 使用醒目提示與 'a != null && a.equals(b)' 類似的表達式選項，可啟用或停用此行為。

僅當專案或模組的語言級別為 7 或更高時，此檢查才會報告。
