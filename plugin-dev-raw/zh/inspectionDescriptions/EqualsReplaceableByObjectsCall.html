報告可以替換為對 java.util.Objects#equals 的調用的表達式。

示例：


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

在應用快速修復後：


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

將 a != null && a.equals(b) 之類的表達式替換為 Objects.equals(a, b) 會略微改變語義。 使用高亮顯示與 'a != null && a.equals(b)' 類似的表達式選項，可啟用或禁用此行為。

僅當項目或模塊的語言級別為 7 或更高時，此檢查才會報告。
