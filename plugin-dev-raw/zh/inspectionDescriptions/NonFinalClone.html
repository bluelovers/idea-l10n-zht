報告沒有 final 修飾符的 clone() 方法。

clone() 可以用於在不使用建構函式的情況下實例化物件，因此允許覆寫 clone() 方法可能導致物件損壞，甚至導致安全漏洞。 將 clone() 方法或封閉類別本身設為 final 可以防止這種情況。

範例：


  class Main implements Cloneable {
      @Override
      protected Object clone() throws CloneNotSupportedException {
        return super.clone();
      }
  }
