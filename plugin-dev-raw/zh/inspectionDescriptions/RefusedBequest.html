報告覆寫特定方法而不調用 super 的方法。

這也稱作被拒絕的遺贈。 此類方法可能表明抽象失敗，導致出現難以跟踪的錯誤。

該檢查不報告預設方法和從 java.lang.Object 覆寫的方法，但對 clone() 除外。 clone() 方法預計會調用其 super，從而自動返回正確類型的物件。

範例:

  • 
  class A {
    @Override
    public Object clone() { // 已報告，因為它不調用 'super.clone()'
      return new A();
    }
  }
  
  • 
  interface I {
    default void foo() {}
  }

  class A implements I {
    // 禁用“忽略 'default' super 方法”和“忽略註解”選項時對方法發出警告
    @Override
    public void foo(){}
  }

設定檢查：

  • 使用僅在通過以下方式註解 super 方法時報告選項，可忽略使用所提供列表中的註解標記的 super 方法。 您可以手動將註解添加到此列表中。
  • 使用忽略空 super 方法選項，可忽略僅拋出異常或空的 super 方法。
  • 使用忽略 'default' super 方法選項，可忽略包含 default 關鍵字的 super 方法。
