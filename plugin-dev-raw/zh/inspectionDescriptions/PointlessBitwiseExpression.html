報告無意義的按位表達式。

此類別表達式包括對給定類型的最大值套用 & 運算符，對零以及移零套用 or 運算符。 出現此類別表達式，可能是因為自動重構沒有一直持續到完成，而且不太可能是出於原本的意圖。

範例：


  // 警告：運算毫無意義，可以取代為 `flags`
  // 0xFFFF_FFFF 是一個整數的最大值，並且這兩個文字都被視為
  // 32 位整數文字。
  int bits = flags & 0xFFFF_FFFF;

  // 警告：運算毫無意義，可以取代為 `bits`
  // 包含 0 的 OR 運算始終輸出另一個動作數。
  int or = bits | 0x0;

  // 警告：運算毫無意義，結果始終為 0
  int xor = or ^ or;
