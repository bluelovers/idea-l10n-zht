報告不以調用 "super" 構造函數或同一類的另一個構造函數開頭的構造函數。

可以認為是此類構造函數隱式地從調用 super() 開始。 某些編碼標準傾向於顯式地調用 super()。

示例：


  class Foo {
    Foo() {}
  }

在應用快速修復後：


  class Foo {
    Foo() {
      super();
    }
  }

使用檢查設定可以忽略直接從 Object 擴展的類。 例如： 


  class Foo {
    Foo() {} // 未報告
  }

  class Bar extends Foo {
    Bar() {} // 對 'super()' 的隱式調用
  }
