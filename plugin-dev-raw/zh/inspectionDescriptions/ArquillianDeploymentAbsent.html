報告沒有部署的 Arquillian 測試用例類別。

根據 Arquillian 文檔，Arquillian 測試類別應該使用具有特定簽名的 public static 方法定義部署歸檔，並使用 @Deployment 註解。

該快速修復會根據使用者可編輯的 Arquillian 部署方法程式碼模板新增所需方法。

範例：


  // 此測試用例無法被 Arquillian 啟動
  @RunWith(Arquillian.class)
  public class SomeTest {
    @Test
    public void testSomething() {
      Assert.fail("To be implemented");
    }
  }

在套用快速修復後：


  @RunWith(Arquillian.class)
  public class SomeTest {
    // 通過快速修復新增
    @Deployment
    public static JavaArchive ${DEPLOYMENT_NAME}() {
      return ShrinkWrap.create(JavaArchive.class)
                       .addAsManifestResource(EmptyAsset.INSTANCE, "beans.xml");
    }

    @Test
    public void testSomething() {
      Assert.fail("To be implemented");
    }
  }
