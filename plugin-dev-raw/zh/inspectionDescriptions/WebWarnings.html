報告檔案描述子中用於將 Web 模組部署到伺服器的重複 welcome-file。

範例：



  <?xml version="1.0" encoding="UTF-8"?>
  <!DOCTYPE web-app PUBLIC "-//Sun Microsystems, Inc.//DTD Web Application 2.3//EN" "http://java.sun.com/dtd/web-app_2_3.dtd">
  <web-app>
    <welcome-file-list>
      <welcome-file>
        Welcome1.jsp
      </welcome-file>
      <welcome-file>
        /Welcome1.jsp // 錯誤：重複的 welcome-file
      </welcome-file>
      <welcome-file>
        Welcome3.jsp
      </welcome-file>
    </welcome-file-list>
  </web-app>
