報告三元條件運算符的用法並建議將它們轉換為 if/else 語句。

一些程式碼標準禁止使用條件運算符。

範例：


  Object result = (condition) ? foo() : bar();

在套用快速修復後：


  Object result;
  if (condition) {
    comp = foo();
  }
  else {
    comp = bar();
  }

組態檢查：

使用 對簡單賦值和返回值忽略 選項忽略簡單的賦值和返回並允許以下結構：


  String s = (foo == null) ? "" : foo.toString();

在不可能用 if 語句自動取代的上下文中（例如，當條件表達式用作 super() 建構函式呼叫），使用 忽略無法使用 if 語句的位置 選項來忽略條件表達式。
