報告增強型 switch 語句和表達式。 建議將它們取代為一般的 switch 語句。

範例：


  boolean even = switch (condition) {
    case 1, 3, 5, 7, 9 -> false;
    default -> true;
  };

在套用快速修復後：


  boolean even;
  switch (condition) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 9:
      even = false;
      break;
    default:
      even = true;
      break;
}
