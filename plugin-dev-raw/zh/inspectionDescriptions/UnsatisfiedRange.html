報告從不符合所宣告方法返回範圍的方法返回的數值。 您可以使用許多註解宣告方法的返回範圍：
  • JetBrains 註解軟體套件中的 org.jetbrains.annotations.Range（指定 'from' 和 'to'）
  • Checker Framework 註解軟體套件中的 org.checkerframework.common.value.qual.IntRange（指定 'from' 和 'to'）
  • Checker Framework 註解軟體套件中的 org.checkerframework.checker.index.qual.GTENegativeOne（範圍為 '>= -1'）
  • Checker Framework 註解軟體套件中的 org.checkerframework.checker.index.qual.NonNegative（範圍為 '>= 0'）
  • Checker Framework 註解軟體套件中的 org.checkerframework.checker.index.qual.Positive（範圍為 '> 0'）
  • JSR 305 註解軟體套件中的 javax.annotation.Nonnegative（範圍為 '>= 0'）
  • javax.validation.constraints.Min（指定最小值）
  • javax.validation.constraints.Max（指定最大值）

範例：


  @Range(from = 0, to = Integer.MAX_VALUE) int getValue() {
    // 警告：-1 超出宣告的範圍
    return -1;
  }
