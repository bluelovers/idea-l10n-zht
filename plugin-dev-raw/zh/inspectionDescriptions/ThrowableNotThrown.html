當創建的 Throwable 從未實際拋出時，報告 Throwable 或其子類的實例化。

這些報告指出了測試中難以捕獲的錯誤。

此外，當方法調用的結果未拋出時，此檢查會報告返回 Throwable 或其子類實例的方法調用。

不會報告對使用 Error Prone 或 AssertJ 的 @CanIgnoreReturnValue 註解進行批注的方法的調用。

範例:


    void leftBehind() throws Throwable {
        RuntimeException e = new RuntimeException(""throw me"");
        // ...
    }
