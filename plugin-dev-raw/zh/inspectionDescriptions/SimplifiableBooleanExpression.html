報告可以簡化的布爾表達式。

範例：


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

在套用快速修復後：


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

範例：


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

在套用快速修復後：


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }
