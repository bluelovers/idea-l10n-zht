報告可以取代為 sum、product、max 或 min 的 fold 和 reduce 方法。

範例：


  List(1, 2, 3).foldLeft(1){(x,y) => x * y}
  List(1, 2, 3).fold(0)(_ + _)
  List(1, 2, 3).reduce(_ + _)
  List(1, 2, 3).reduceLeft(_ min _)
  List(1, 2, 3).reduce((x, y) => math.max(x, y))

在套用快速修復後：


  List(1, 2, 3).product
  List(1, 2, 3).sum
  List(1, 2, 3).sum
  List(1, 2, 3).min
  List(1, 2, 3).max
