報告 ISNULL 函數中字串實參的截斷。

ISNULL 語法是 ISNULL(check_expression, replacement_value)。

根據 docs.microsoft.com 上的 ISNULL，如果 replacement_value 比 check_expression 長，replacement_value 將被截斷。

範例（Microsoft SQL 伺服器）：

DECLARE @name1 VARCHAR(2) = NULL;
DECLARE @name2 VARCHAR(10) = 'Example';
DECLARE @name3 VARCHAR(2) = 'Hi';

  -- `@name2` 是 VARCHAR(10) 並且將被截斷
SELECT ISNULL(@name1, @name2);

  -- `@name3` 是 VARCHAR(2)（作為 `@name1`）並且不會被截斷
SELECT ISNULL(@name1, @name3);
