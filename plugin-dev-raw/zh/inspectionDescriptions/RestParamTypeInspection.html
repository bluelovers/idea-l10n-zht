報告具有不受支援類型的資源方法參數（使用 @PathParam、@QueryParam 等註解）。

註解的參數、欄位或屬性的類型必須滿足以下要求之一：

  1. 為基元類型
  2. 有一個接受單個 String 實參的構造函數
  3. 有一個名為 valueOf() 或 fromString() 的 static 方法，它接受單個 String 實參，例如 Integer.valueOf(String)。
  4. 擁有 ParamConverterProvider JAX-RS 擴展 SPI 的註冊實作，它返回一個能夠將字串轉換為特定類型的 ParamConverter 實例。
  5. 為 List<T>、Set<T> 或 SortedSet<T>，其中 T 為 2、3 或 4 以上。 生成的集合為唯讀。
