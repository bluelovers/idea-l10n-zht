報告與那些從 Java get 或 set 方法自動生成的合成屬性相衝突的擴展屬性。

應移除或重命名此類屬性，以避免將來在編譯器中進行更改時破壞程式碼。

該快速修復會刪除擴展屬性。

範例：


  val File.name: String
      get() = getName()

該快速修復會添加 @Deprecated 註解：


  @Deprecated("Is replaced with automatic synthetic extension", ReplaceWith("name"), level = DeprecationLevel.HIDDEN)
  val File.name: String
      get() = getName()
