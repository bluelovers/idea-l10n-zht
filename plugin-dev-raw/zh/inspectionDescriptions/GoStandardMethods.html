在以下情況下報告具有特定名稱的方法：
  • 方法的名稱與標準庫中幾個眾所周知的接口方法的名稱相符合
  • 簽名與對應接口方法的簽名不符合

此類別方法可能表明接收器類型旨在滿足標準庫中的接口，但由於方法簽名中的錯誤而無法滿足。

範例：

type MyReader []byte

func (r MyReader) ReadByte(data []byte) (byte, error) {
}

這種用法很可疑，因為它看起來像是試圖實作 io.ByteReader，但簽名不正確。 下面這樣寫比較合適：

type MyReader []byte

func (r MyReader) ReadByte() (byte, error) {
}
