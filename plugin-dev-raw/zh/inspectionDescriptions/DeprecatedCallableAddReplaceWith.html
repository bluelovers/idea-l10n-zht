報告在其 kotlin.deprecated 注解中沒有 kotlin.ReplaceWith 實參的已棄用函數和屬性，IDE 會建議根據它們的主體進行添加。

Kotlin 提供 ReplaceWith 實參來自動替換已棄用的聲明。 建議使用實參來修正代碼中的棄用問題。

示例：


  @Deprecated("Use refined() instead.")
  fun deprecated() = refined()

  fun refined() = 42

該快速修復會添加 ReplaceWith() 實參：


  @Deprecated("Use refined() instead.", ReplaceWith("refined()"))
  fun deprecated() = refined()

  fun refined() = 42
