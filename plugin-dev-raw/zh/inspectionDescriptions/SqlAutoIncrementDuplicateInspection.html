報告包含兩個具有自動增量的列的表。 在 MySQL、Microsoft SQL Server 和 Db2 方言中，一個表只能有一個帶有自動增量選項的欄位，並且此欄位必須是鍵。

範例（MySQL）：

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT,
    c2 INT AUTO_INCREMENT,
);

c2 的 AUTO_INCREMENT 約束將被醒目提示，因為 c1 已經具有此約束。 要修正警告，您可以將 id 設為主鍵並刪除 c2 的 AUTO_INCREMENT。

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT PRIMARY KEY,
    c2 INT,
);
