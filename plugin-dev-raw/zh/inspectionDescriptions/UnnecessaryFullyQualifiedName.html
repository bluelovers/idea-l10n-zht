報告可以縮短的完全限定類別名。

快速修復會縮短完全限定名稱並在必要時新增匯入語句。

範例：


  class ListWrapper {
    java.util.List<Integer> l;
  }

在套用快速修復後：


  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

組態檢查：

使用 在 Java 9 模組語句中忽略 選項忽略 Java 9 provides 和 uses 模組語句中的完全限定名稱。

在設定| 編輯器 | 程式碼樣式 | Java | 匯入 中，請使用以下選項組態檢查：

  • 如果應使用外部類別限定對內部類的參照，請使用 為內部類別插入匯入 選項。
  • 使用在 JavaDoc 中使用完全限定類別名 選項允許在 Javadoc 中使用完全限定名稱。
