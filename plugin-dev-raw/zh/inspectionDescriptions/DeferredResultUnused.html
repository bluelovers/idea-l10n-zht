如果未使用返回值，則報告具有 Deferred 結果類型的函式呼叫。

如果未使用 Deferred 返回值，則呼叫站點不會等待補全此函式。

範例：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }

該快速修復會提供具有 Deferred 初始設定式的變數：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      val answer = calcEverythingAsync()
  }
