如果未使用返回值，則報告具有 Deferred 結果類型的函數調用。

如果未使用 Deferred 返回值，則調用站點不會等待補全此函數。

範例：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }

該快速修復會提供具有 Deferred 初始值設定項的變數：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      val answer = calcEverythingAsync()
  }
