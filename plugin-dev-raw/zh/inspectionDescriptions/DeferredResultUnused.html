如果未使用返回值，則報告具有 Deferred 結果類型的調用。

如果有返回值，函數將不會等到它完成。

範例:


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }

建議通過快速修復引入具有 Deferred 初始設定式的變數：


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      val answer = calcEverythingAsync()
  }
