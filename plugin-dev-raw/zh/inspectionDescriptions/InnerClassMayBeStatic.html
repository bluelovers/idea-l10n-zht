報告可以設定為 static 的內部類。

static 內部類不保留對其封閉實例的隱式引用。 這可以防止常見的內存泄漏，減少類的每個實例使用的內存。

範例：


  public class Outer {
    class Inner { // 非 static
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }

在應用快速修復後：


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }
