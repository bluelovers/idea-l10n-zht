報告對 junit.framework.Assert 類別中方法的任何調用。 此類別已廢棄，這些調用可以取代為對 org.junit.Assert 類別中方法的調用。

範例：


  import org.junit.*;
  public class NecessaryTest {
    @Test
    public void testIt() {
      junit.framework.Assert.assertEquals("expected", "actual");
    }
  }

套用快速修復後，結果如下所示：


  import org.junit;
  public class NecessaryTest {

    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }
