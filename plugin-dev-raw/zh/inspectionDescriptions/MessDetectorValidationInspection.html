報告由 PHP Mess Detector 檢測到的編碼風格問題。

該項檢查要求正確安裝 PHP Mess Detector，並在 IDE中的 設置/偏好設置 | PHP | 質量工具 | Mess Detector 下正確設置 。

要了解有關安裝 PHP Mess Detector 的更多信息，請參見 PHP Mess Detector installation (phpmd.org)。

您可以使用預定義規則集，或通過指定有效的 ruleset.xml 文件添加您自己的規則。 有關詳細信息，請參閱 rules index (phpmd.org) 和 How to create a custom rule set (phpmd.org)。
