報告由 PHP Mess Detector 檢測到的編碼風格問題。

該項檢查要求正確安裝 PHP Mess Detector，並在 IDE 中的設定 | PHP | 品質工具 | Mess Detector 下正確設定 。

要了解有關安裝 PHP Mess Detector 的更多資訊，請參見 PHP Mess Detector installation (phpmd.org)。

您可以使用預定義規則集，或通過指定有效的 ruleset.xml 檔案新增您自己的規則。 有關詳細資訊，請參閱 rules index (phpmd.org) 和 How to create a custom rule set (phpmd.org)。
