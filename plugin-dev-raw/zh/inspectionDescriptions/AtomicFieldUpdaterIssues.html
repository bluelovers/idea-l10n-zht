報告 AtomicLongFieldUpdater、AtomicIntegerFieldUpdater 或 AtomicReferenceFieldUpdater 欄位（java.util.concurrent.atomic 軟體套件）的問題。

報告的問題與原子欄位更新程序可能發生的執行時問題相同：找不到指定欄位、指定欄位不可存取、指定欄位類型錯誤等。

範例：

  • 
  class A {
    private static volatile int value = 0;
    private static final AtomicIntegerFieldUpdater updater =
    AtomicIntegerFieldUpdater.newUpdater((A.class), "value"); // 警告：欄位 'value' 具有 'static' 修飾符
  }
  
  • 
  class B {
    private static final AtomicIntegerFieldUpdater updater =
      AtomicIntegerFieldUpdater.newUpdater(B.class, "value"); // 警告：在類別 'B' 中找不到名為 'Value' 的欄位
  }
