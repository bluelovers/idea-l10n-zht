報告不必要地宣告為 final 的區域變數或參數。

出於簡潔的原因，某些編碼標準不贊成宣告為 final 的變數。

範例：


  class Foo {
    Foo(Object o) {}

    void bar(final Object o) {
      new Foo(o);
    }
  }

在套用快速修復後：


  class Foo {
    Foo(Object o) {}

    void bar(Object o) {
      new Foo(o);
    }
  }

使用檢查選項可以切換以下各項的報告：

  • 區域變數
  • 參數（包括 catch 塊的參數和增強型 for 語句）

此外，您可以將檢查組態為僅報告 abstract 或接口方法的 final 參數，這可能被認為是多餘的，因為此類別標記不會影響這些方法的實作。
