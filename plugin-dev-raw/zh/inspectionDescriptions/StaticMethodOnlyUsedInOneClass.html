報告僅從包含類別以外的類別使用的 static 方法和欄位。 此類別成員可以移動到所使用的類別。 此檢查將忽略從成員的類別內部的匿名類別存取的工廠方法和成員。 呼叫同一個類別中名稱相同但參數較少的方法的方便多載也會被忽略。

僅在從測試類別使用 static 成員時，可以使用第一個複選框來禁止此檢查。

使用下面的第二個複選框可以忽略匿名類別、本地類別或非 static 內部類別中的成員用法。

使用下面的第三個複選框，可以不對沒有問題的情況下也無法移動的成員發出警告，例如，由於目標類別中已存在簽名相同的方法，或者由於移動此方法時將無法存取方法內部使用的欄位或方法。

使用第四個複選框可以忽略位於實用程序類別中的成員。
