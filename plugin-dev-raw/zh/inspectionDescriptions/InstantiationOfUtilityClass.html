報告使用 new 關鍵字的實用程序類別的實例化。

在實用程序類別中，所有欄位和方法均為 static。 這種類的實例化很可能不必要，並且表明存在錯誤。

範例：


  class MyUtils {
    public static double cube(double x) {
      return x * x * x;
    }
  }
  class Main {
    public static void main(String[] args) {
      // 實用類別的實例化
      MyUtils utils = new MyUtils();
    }
  }

為防止實用程序類別被實例化，建議使用 private 建構函式。
