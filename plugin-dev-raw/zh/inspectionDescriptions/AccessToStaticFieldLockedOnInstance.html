報告對在 this 或 this 的實例欄位上鎖定的非常數 static 欄位的存取。

在實例資料上鎖定 static 欄位並不能防止其他實例修改該欄位，因此可能會導致意外的競爭條件。

範例：


  static String test;
  public void foo() {
      synchronized (this) {
          System.out.println(test); // 警告
      }
  }

通過快速修復可以忽略特定類型的 static 欄位。 可以在檢查選項中管理被忽略的類型。

使用檢查選項可以指定應忽略哪些用於 static 欄位的類別。
