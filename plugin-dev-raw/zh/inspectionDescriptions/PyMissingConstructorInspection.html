報告類中缺少對 super 構造函數進行調用的情況。

範例：

class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        pass

Pear 類在 __init__ 方法中應有 super 調用。

套用快速修復後，程式碼變為：

class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        super().__init__()
