報告 java.util.List 上對 containsAll() 的呼叫。

此方法呼叫的時間復雜度為 O(n·m)，其中 n 是呼叫該方法的列表中的元素數，m 是作為參數傳遞至該方法的集合中的元素數。 當列表很大時，這可能是一項開銷巨大的動作。

該快速修復會將列表包裝在 new java.util.HashSet<>() 中，因為從 java.util.List 建立 java.util.HashSet 並在 java.util.HashSet 上執行 containsAll() 的所需時間為 O(n+m)。

範例：

  public boolean check(List<String> list, Collection<String> collection) {
    // O(n·m) 復雜度
    return list.containsAll(collection);
  }

在套用快速修復後：

  public boolean check(List<String> list, Collection<String> collection) {
    // O(n+m) 復雜度
    return new HashSet<>(list).containsAll(collection);
  }
