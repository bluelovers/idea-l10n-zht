報告 @Bean 註解的 initMethod 和 destroyMethod 參數上的未解析方法引用。

範例：

如果 MyBean 類不包含 'public void 'doInit'(){...}' 方法，將在 'doInit' 表達式上報告 "無法解析方法" 

  public class MyBean {...}

  @Component
  public class MyComponent {
     @Bean(initMethod="doInit" )
     public MyBean myBean() {...}
  }

在此範例中，如果 MyBean 不定義 doInit() 方法，該檢查將報告未解析的方法引用。
