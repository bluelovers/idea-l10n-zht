報告在 Java 記錄中宣告的冗餘構造函數。

範例 1:


  record Point(int x, int y) {
    public Point {} // 可以移除
  }
  
  record Point(int x, int y) {
    public Point(int x, int y) { // 可以移除
      this.x = x;
      this.y = y;
    }
  }

快速修復移除了冗餘的構造函數。

範例 2:


   // 可以轉換為壓縮構造函數
  record Range(int from, int to) {
    public Range(int from, int to) {
      if (from > to) throw new IllegalArgumentException();
      this.from = from;
      this.to = to;
    }
  }

快速修復將此程式碼轉換為壓縮構造函數。

僅當所設定的語言級別為 14 級或 15 級(預覽)時，該檢查才會進行報告。
