報告 Spring Security 註解中參照的未解析 Spring Bean。

範例：

  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "unknownUserDetailsService")
      // 如果 Spring 模型中沒有定義 'unknownUserDetailsService'，將報告 "無法解析 'unknownUserDetailsService' Bean"
    
  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "jpaUserDetailsService")
      //"Bean 必須為 'org.springframework.security.core.userdetails.UserDetailsService' 類型"
      // 如果 "jpaUserDetailsService" Bean 不是 'UserDetailsService' 類型，將報告
