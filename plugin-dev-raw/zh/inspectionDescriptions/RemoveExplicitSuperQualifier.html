報告具有冗餘父類型限定的 super 成員呼叫。

派生類別中的程式碼可以使用 super 關鍵字呼叫其父類別函式和屬性存取器實作。 要指定從中獲取繼承實作的父類型，可以通過尖括號中的父類型名稱來限定 super ，例如 super<Base>。 有時這種限定是冗餘的，可以省略。 使用“移除顯式父類型限定”快速修復可清理程式碼。

範例：


  open class B {
      open fun foo(){}
  }

  class A : B() {
      override fun foo() {
         super<B>.foo() // <== 冗餘，因為 'B' 是唯一父類型
      }
  }

  interface I {
      fun foo() {}
  }

  class C : B(), I {
      override fun foo() {
          super<B>.foo() // <== 在這裡，需要 <B> 限定符以將 'B.foo()' 與 'I.foo()' 區分
      }
  }

在套用快速修復後：


  open class B {
      open fun foo(){}
  }

  class A : B() {
      override fun foo() {
         super.foo() // <== 已更新
      }
  }

  interface I {
      fun foo() {}
  }

  class C : B(), I {
      override fun foo() {
          super<B>.foo()
      }
  }
