報告與使用的匯入重疊的變數、參數或函數的宣告。

這樣的宣告雖然本身有效，但可能會使包的匯出關鍵字不可用或造在讀取程式碼時成混亂。

範例:

import "fmt"
import _ "fmt"
import iio "io"

func _() {
 fmt.Println("demo")
 demo := true
 _, _ = iio.EOF, demo
}

func demo() (int, int) {
 return 1, 2
}

func _() {
 _, _ = iio.EOF, demo
 fmt := "demo"
 iio := 1
 _, _ = iio, fmt
 a, _ := demo()
 _ = a
}

變數名 fmt 和 iio 與匯入包的名稱衝突。 為了之後不在程式碼中混淆他們，最好重命名這些變數。
