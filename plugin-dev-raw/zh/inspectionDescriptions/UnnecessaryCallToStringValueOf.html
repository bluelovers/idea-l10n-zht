報告在不需要顯式字串轉換的字串串聯中使用或作為庫方法實參使用的靜態方法調用，例如 String.function valueOf() { [native code] }() 或 Integer.function toString() { [native code] }()。

範例：


  System.out.println("Number: " + Integer.function toString() { [native code] }(count));

在應用快速修復後：


  System.out.println("Number: " + count);

其中顯式字串轉換被認為是冗餘的庫方法：

  • 類 java.io.PrintWriter, java.io.PrintStream
    • print(), println()
  • 類 java.lang.StringBuilder, java.lang.StringBuffer
    • append()
  • 類 org.slf4j.Logger
    • trace(), debug(), info(), warn(), error()
