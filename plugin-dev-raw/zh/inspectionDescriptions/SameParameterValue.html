報告始終具有相同常量值的方法參數。

範例：


  static void printPoint(int x, int y) { // x 始終為 0
    System.out.println(x + ", " + y);
  }

  public static void main(String[] args) {
    printPoint(0, 1);
    printPoint(0, 2);
  }

該快速修復會內聯常量值。 這可能會簡化方法實作。

在以下情況下，使用內聯參數初始設定式不成功時忽略選項來禁止檢查：

  • 參數在方法內部被修改。
  • 傳遞的參數值引用了不可存取的欄位（僅在 Java 中）。
  • 參數為 vararg（僅在 Java 中）。

使用報告方法最大可見性選項可以控制要報告的方法的最大可見性。

使用報告方法最小使用計數欄位可以指定具有相同參數值的最小方法使用數。
