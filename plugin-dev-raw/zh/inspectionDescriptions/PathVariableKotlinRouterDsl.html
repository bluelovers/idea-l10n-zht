報告 Kotlin 路由器 DSL 中存在未使用的路徑變數，以及使用未宣告的路徑變數。

路徑變數必須先在在路由路徑中宣告，然後才能存取。 嘗試存取未宣告的路徑變數將導致執行時錯誤。

範例：


@Configuration
class RouterConfiguration {
    @Bean
    fun myRouter() = router {
        GET("/test/{var}") { ServerResponse.ok().body("${it.pathVariable("bar")}") }
    }
}

套用修正後：


@Configuration
class RouterConfiguration {
    @Bean
    fun myRouter() = router {
        GET("/test/{var}") { ServerResponse.ok().body("${it.pathVariable("var")}") }
    }
}
