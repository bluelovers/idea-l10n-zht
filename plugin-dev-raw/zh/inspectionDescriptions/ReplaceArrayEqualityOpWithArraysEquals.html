報告 == 或 != 運算符的用法，用於應取代為 contentEquals() 的陣列。

== 和 != 運算符比較陣列參照而不是它們的內容。

範例：

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // 參考比較
  }

在套用快速修復後：

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
