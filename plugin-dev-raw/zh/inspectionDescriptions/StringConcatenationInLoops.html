報告迴圈中的 String 串聯。

由於每個 String 串聯都會複製整個字串，因此通常最好將其取代為對 StringBuilder.append() 或 StringBuffer.append() 的顯式呼叫。

範例：


  String str = "";
  for(int i=0; i<10; i++) {
    str += i;
  }

在套用快速修復後：


  String str = "";
  StringBuilder strBuilder = new StringBuilder(str);
  for(int i = 0; i<10; i++) {
    strBuilder.append(i);
  }
  str = strBuilder.toString();

有時，通過該快速修復可將 String 變數轉換為 StringBuilder 或引入新的 StringBuilder。 原始程式碼專門處理 null 值時應小心，因為取代可能會改變語意。 如果可能為 null，建議進行 null 安全修正，生成必要的 null 檢查。 此外，也不能保證自動取代總是性能更好。
