報告不帶 @Pattern 註解的方法覆寫帶 @Pattern 註解方法的情況。 這不會阻止在編輯器中醒目提示錯誤，但覆寫方法在執行時將不會被檢查。

提供了快速修復，用於新增與超類別方法中註解相符合的 @Pattern 註解。 這確保 執行時檢查檢測正常工作。

範例：


  abstract class Parent {
    abstract @Pattern("\\d\\d-\\d\\d\\d") String getId();
  }
  class Child extends Parent {
    @Override String getId() { // 此處為警告
      return "12-345";
    }
  }
