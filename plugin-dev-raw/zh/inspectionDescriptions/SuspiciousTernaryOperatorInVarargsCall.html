報告使用具有混合陣列分支和非陣列分支的三元運算符的 vararg 方法呼叫。

編譯時，兩個分支都被包裝在陣列中。 結果，陣列分支變成了二維陣列，這可能說明有問題。

該快速修復會將非陣列分支包裝在陣列中以防止編譯器執行轉換。

範例：


    static void bar(boolean flag) {
        Object[] a = {1, 2};
        Object b = "hello";
        foo(flag ? a : b);
    }
    static void foo(Object... obj) {
    }

套用該快速修復後： 


    static void bar(boolean flag) {
        Object[] a = {1, 2};
        Object b = "hello";
        foo(flag ? a : new Object[]{b});
    }
    static void foo(Object... obj) {
    }
