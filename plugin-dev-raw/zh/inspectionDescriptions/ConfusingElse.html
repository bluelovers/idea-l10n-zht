在 if—else 語句和語句鏈中報告冗餘的 else 關鍵字。

如果 else 關鍵字因為之前分支中的程式碼以 return、throw、break 或 continue 語句結尾而無法到達，則它是冗餘的。 在這種情況下，來自 else 分支的語句可以放在 if 語句之後，並且 else 關鍵字可以刪除。

範例：


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

在應用快速修復後：


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

禁用 當 'if' 語句之後沒有其他語句時報告 選項以忽略 if—else 語句是程式碼塊中最後一條語句的情況。
