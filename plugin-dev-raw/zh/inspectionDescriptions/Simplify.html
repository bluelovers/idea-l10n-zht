報告可以簡化的語句，例如常數條件、相同的 if 分支、冗餘布爾表達式等。 使用下面的複選框可以控制分析選項：
  • 簡化 "if (true)", "while(false)"：建議將 if/while 語句取代為布爾文字。
  • 簡化 "condition == true"：建議將相等語句取代為布爾文字。
  • 簡化 "condition1 ? true : condition2;"：建議簡化其中一個表達式是布爾文字的三元運算符。
  • 簡化 "if (condition1) return true; return condition2;"：建議簡化符合以下條件的 if：如果條件為 true 則返回布爾文字，否則返回另一個條件。
