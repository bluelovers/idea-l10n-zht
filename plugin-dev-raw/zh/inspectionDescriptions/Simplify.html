報告可以簡化的語句，例如常數條件、相同的 if 分支、冗餘布爾表達式等。 使用下面的複選框來控制分析選項：
  • 簡化 "if (true)", "while(false)": 建議用布爾文字取代 if/while 語句。
  • 簡化 "condition == true": 建議用布爾文字取代相等語句。
  • 簡化 "condition1 ? true : condition2;": 建議簡化三元運算符，其中表達式之一是布爾文字。
  • 簡化 "if (condition1) return true; return condition2;": 建議簡化 if，其中如果條件為 true 則返回布爾文字，否則返回另一個條件。
