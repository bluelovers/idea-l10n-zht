報告對象與自身進行相等比較的 equals() 或 compareto() 調用。

根據方法約定，對於 equals()，這些運算始終返回 true；對於 compareTo()，始終返回 0。 該檢查還可以檢查對 Objects.equals()、Objects.deepEquals()、Arrays.equals()、Comparator.compare 等的調用。

範例：


  class Foo {
    boolean foo(Object o) {
        return o.equals(o); // 警告
    }

    boolean bar(String[] ss) {
        return Arrays.equals(ss, ss); // 警告
    }
}
