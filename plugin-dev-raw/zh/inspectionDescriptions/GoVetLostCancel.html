報告未呼叫由 context.WithCancel 和類似的函式返回 cancel 函式的執行路徑。

WithCancel、WithDeadline 和 WithTimeout 函式接收一個 Context（父項）並返回一個派生的 Context（子項）和一個 CancelFunc。 呼叫 CancelFunc 可以取消子項及其子項，刪除父項對該子項的參照，並停止任何 相關的定時器。 如果不呼叫 CancelFunc，就會泄漏子項和其子項，直到父項被取消或定時器啟動。

有關 context 軟體套件的更多資訊，請參閱 go.dev 上的 Package context。

範例：

func _(ctx context.Context, cancel func()) {
 var ctx2 context.Context
 ctx2, cancel = context.WithCancel(ctx)
 _ = ctx2
}
