報告 val 或 var 可被移除的主構造函數參數。

在構造函數中宣告的類屬性會消耗額外的記憶體。 如果參數值僅在構造函數中使用，通常沒有理由存儲它們。

請注意，引用的物件可能是之前被垃圾回收。

範例:


  class Task(val name: String) {
      init {
          print("Task created: $name")
      }
  }

建議通過快速修復移除額外的 val 或 var 關鍵字：


  class Task(name: String) {
      init {
          print("Task created: $name")
      }
  }
