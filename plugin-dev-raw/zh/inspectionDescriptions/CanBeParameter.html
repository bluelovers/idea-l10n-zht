報告 val 或 var 可被移除的主構造函數參數。

在構造函數中宣告的類屬性會增加內存消耗。 如果參數值僅在構造函數中使用，則可以省略它們。

請注意，引用的物件可能是之前被垃圾回收。

範例：


  class Task(val name: String) {
      init {
          print("Task created: $name")
      }
  }

該快速修復會移除額外的 val 或 var 關鍵字：


  class Task(name: String) {
      init {
          print("Task created: $name")
      }
  }
