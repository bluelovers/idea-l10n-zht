報告在物件構造期間對當前類的覆寫方法的任何調用。 如果物件構造在內部，則會發生這種情況：
  • 構造函數
  • 非 static 實例初始設定式
  • 非 static 欄位初始設定式
  • clone()
  • readObject()
  • readObjectNoData()

此類調用可能會導致難以捉摸的錯誤，因為不能保證物件在方法調用發生之前被初始化。

範例：


  abstract class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }

    @Override
    void someMethod() { }
  }

此檢查與以下各項共享其功能：

  • 物件構造期間調用的 abstract 方法檢查
  • 物件構造期間調用的可覆寫方法檢查

一次只應啟用一個檢查，以防止重複警告。
