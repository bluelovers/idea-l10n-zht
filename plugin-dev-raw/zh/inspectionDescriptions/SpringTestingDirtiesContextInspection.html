報告 @DirtiesContext 注解中不正確的 'mode' 配置。

示例：


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // 在注解的測試類上設置方法模式沒有意義。 對於類級別控制，請改用 classMode。
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // 通過 @ContextHierarchy 將上下文配置為層次結構的一部分時，應使用 hierarchyMode
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // 在注解的測試方法上設置類模式沒有意義。 對於方法級別控制，請改用 methodMode。
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // 通過 @ContextHierarchy 將上下文配置為層次結構的一部分時，應使用 hierarchyMode
    public void testFoo() {...}
  }
