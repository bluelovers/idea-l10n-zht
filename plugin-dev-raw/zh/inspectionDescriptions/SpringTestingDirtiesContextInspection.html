報告 @DirtiesContext 註解中不正確的 'mode' 組態。

範例：


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // 在註解的測試類別上設定方法模式沒有意義。 對於類別級別控制，請改用 classMode。
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // 通過 @ContextHierarchy 將上下文組態為層次結構的一部分時，應使用 hierarchyMode
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // 在註解的測試方法上設定類別模式沒有意義。 對於方法級別控制，請改用 methodMode。
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // 通過 @ContextHierarchy 將上下文組態為層次結構的一部分時，應使用 hierarchyMode
    public void testFoo() {...}
  }
