報告返回類型不正確的 Spring Data CRUD 儲存庫方法。

支援以下返回類型：

  • void
  • 基元類型
  • 包裝器類型
  • T（當查詢方法應最多返回一個結果時）。 如果沒有結果，則返回 null，如果有多個結果，則拋出 IncorrectResultSizeDataAccessException。
  • java.util.List 或任何常見的 Iterable 類型
  • java.util.stream.Stream
  • java.util.Optional 或 scala.Option（當查詢方法應最多返回一個結果時）。 如果沒有結果，則返回 Optional.empty() 或 Optional.absent()，如果有多個結果，則拋出 IncorrectResultSizeDataAccessException。
  • io.micronaut.data.model.Page
  • io.micronaut.data.model.Slice
  • java.util.concurrent.Future、java.util.concurrent.CompletableFuture 或 ListenableFuture（如果該方法使用 @Async 進行註解並啟用 Spring 異步方法執行）
  • GeoResults<T>、GeoResult<T> 或 GeoPage<T>
