報告任何不必要的分號，包括在類別成員之間、塊語句內部或類別定義之後使用的分號。

儘管這些分號在 Java 中有效，但它們是冗餘的，可以被刪除。

範例：


  class C {
    ;
    void m() throws Exception {
        try (AutoCloseable r1 = createAutoCloseable();) {
          ;
        }
    }
    ;
  }

在套用快速修復後：


  class C {
    void m() throws Exception {
      try (AutoCloseable r1 = createAutoCloseable()) {
      }
    }
  }
