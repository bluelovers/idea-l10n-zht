在物件構造期間使用 this 報告不安全的操作，包括：
  • 在類初始化期間存取非 final 屬性：從構造函數或屬性初始化
  • 在類初始化期間調用非 final 函數
  • 在非 final 類的構造函數中使用 this 作為函數實參

如果其他類繼承自給定的類，則在執行不安全操作時，可能無法完全初始化它們。

範例：


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }

  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }

  fun testIt() {
      println(Derived(42).code) // 預期值：42，實際值：0
  }
