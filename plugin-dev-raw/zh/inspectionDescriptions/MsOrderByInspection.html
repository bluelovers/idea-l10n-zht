報告在檢視、內聯函數、派生表、子查詢和通用表表達式中使用 ORDER BY 子句而不使用 TOP、OFFSET 或 FOR XML 的用法。

有關 ORDER BY 用法的詳細資訊，請參閱 docs.microsoft.com 上的 SELECT - ORDER BY 子句 (Transact-SQL) 頁面。

範例（Microsoft SQL Server）：

CREATE TABLE foo (a INT NOT NULL, b  INT NOT NULL);

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89
ORDER BY b) ALIAS;

在子查詢中，ORDER BY 將被高亮顯示為錯誤。 您可以將 TOP、OFFSET 或 FOR XML 添加到子查詢。 或者，使用刪除元素快速修復來刪除 ORDER BY 部分。

在套用快速修復後：

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89) ALIAS;
