報告內建函式的簽名問題。

檢查將報告錯誤數量的實參、無效的關鍵字、錯誤的資料類型和其他問題。

範例（MySQL）：

CREATE TABLE foo (a INT, b INT, c INT)

SELECT IFNULL() FROM foo; -- 錯誤
SELECT IFNULL(a) FROM foo; -- 錯誤
SELECT IFNULL(a, b) FROM foo; -- 正確
SELECT IFNULL(a, b, c) FROM foo; -- 錯誤

在 MySQL 中，IFNULL() 函式嚴格接受兩個實參。 因此，只有 SELECT IFNULL(a, b) FROM foo; 查詢是正確的。
