如果包含類別沒有將 java.lang.Object 作為其實參類型的其他 equals() 多載，則報告採用 java.lang.Object 以外的實參類型的 equals() 方法。

equals() 的協變版本不會覆寫 Object.equals(Object) 方法。 它可能會在執行時導致意外行為。 例如，如果該類用於建構一個標準集合類別，該集合類別預期會覆寫 Object.equals(Object) 方法。

範例：


  class Foo {
      public boolean equals(Foo foo) {  // 警告
          return false;
      }
  }
  class Bar {
      public boolean equals(Bar bar) {  // 此處無警告
          return false;
      }
      @Override
      public boolean equals(Object obj) {
          return false;
      }
  }
