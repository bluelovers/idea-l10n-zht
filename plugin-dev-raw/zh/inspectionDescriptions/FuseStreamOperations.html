檢測 Stream API 鏈之外的可以合併到其中的轉換。

範例：


  List<String> list = stream.collect(Collectors.toList());
  list.sort(null);
  return list.toArray(new String[list.size()]);

轉換後：


  return stream.sorted().toArray(String[]::new);

請注意，有時轉換後的 Stream 鏈可能會將顯式 ArrayList 取代為 Collectors.toList() 或將顯式 HashSet 取代為 Collectors.toSet()。 當前庫實作會在內部使用這些集合。 但是，這種方法不是很可靠，將來可能會更改程式碼的語意。

如果您對此感到擔心，請使用不建議使用 'toList()' 或 'toSet()' 收集器選項來建議使用 Collectors.toCollection() 而不是 toList 和 toSet 收集器。
