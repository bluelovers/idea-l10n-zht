報告 x = atomic.AddUint64(&x, 1) 形式的賦值語句。

此類操作為非原子操作，這是 sync/atomic API 中的常見誤用。 要使其原子化，請刪除賦值並直接調用它，如 atomic.AddUint64(& x, 1) 。 在這種情況下，x 的值由地址以原子方式更新。

範例：

import (
 "sync/atomic"
)

type Counter uint64

func AtomicTests() {
 x := uint64(1)
 x = atomic.AddUint64(&x, 1)         // 錯誤 “直接賦值給原子值”
 _, x = 10, atomic.AddUint64(&x, 1)  // 錯誤 “直接賦值給原子值”
 x, _ = atomic.AddUint64(&x, 1), 10  // 錯誤 “直接賦值給原子值”
}
