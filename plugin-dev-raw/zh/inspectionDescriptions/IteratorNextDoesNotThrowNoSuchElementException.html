報告不能拋出 java.util.NoSuchElementException 的 Iterator.next() 的實作。

此類別實作違反了 java.util.iterator 的約定，如果以非標準方式使用迭代器，可能會導致難以捉摸的錯誤。

範例：


  class Numbers implements Iterator<Integer> {
    @Override
    public Integer next() { //warning
        if (hasNext()) {
            return generateNext();
        } else {
            return null; //而是拋出 NoSuchElementException
        }
    }

    ...
  }
