報告被宣告為 java.lang.String 且重複追加的變數。 此類別變數可以更有效地被宣告為 java.lang.StringBuffer 或 java.lang.StringBuilder。

範例：


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

此類別迴圈可以取代為：


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

或者甚至可以取代為：


  String s = String.join(" ", names);

使用該選項可使此檢查僅在變數被附加到迴圈中時報告。
