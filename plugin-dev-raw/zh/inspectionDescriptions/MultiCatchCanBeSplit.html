報告多個 catch 部分並建議將它們拆分為單獨的 catch 塊。

範例：


  try {
    int i = getIndex();
  } catch (NullPointerException|IndexOutOfBoundsException e) {
    e.printStackTrace();
  }

在套用快速修復後：


  try {
    int i = getIndex();
  } catch (NullPointerException e) {
    e.printStackTrace();
  } catch (IndexOutOfBoundsException e) {
    e.printStackTrace();
  }

僅當專案或模組的語言級別為 7 或更高時，此檢查才會報告。
