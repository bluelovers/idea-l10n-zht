如果在類周圍的程式碼中存在同名的局部變數、參數或欄位，則報告從匿名類、內部類或局部類對超類欄位的存取。 在這種情況下，看似存取了來自周圍程式碼的元素，而實際上存取的是來自超類的欄位。

為闡明程式碼的意圖，建議對欄位存取添加顯式 super 限定符。

範例：


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(ambiguous); // 存取的是欄位，而不是參數
          }
        };
      }
    }

在套用快速修復後：


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(super.ambiguous);
          }
        };
      }
    }
