報告對匿名類中的超類欄位的模糊存取，其中環繞的程式碼中有同名的局部變數、參數或欄位。

在本例中，看似存取了環繞的程式碼中的一個元素，實則存取了匿名類擴展的超類中的一個欄位。

為闡明程式碼的意圖，建議對欄位存取添加顯式的 super 限定符。

範例:


    class First {
      protected String test;
    }
    class Second {
      void foo(String test) {
        new First() {
          {
            System.out.println(test); // 存取了此欄位，而不是此參數
          }
        };
      }
    }

在套用快速修復後：


    class First {
      protected String test;
    }
    class Second {
      void foo(String test) {
        new First() {
          {
            System.out.println(super.test);
          }
        };
      }
    }
