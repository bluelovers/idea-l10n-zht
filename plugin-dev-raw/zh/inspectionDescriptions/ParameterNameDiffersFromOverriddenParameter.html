報告名稱與其覆寫或多載方法的相應參數不同的參數。 雖然在 Java 中是合法的，但這種不一致的名稱可能令人困惑，並減少具有良好命名做法的文檔優勢。

範例：


  class Person {
    Person(String fullName) {}
  }
  class Child extends Person {
    Child(String name) { super(name); }
  }

在套用快速修復後：


  class Person {
    Person(String fullName) {}
  }
  class Child extends Person {
    Child(String fullName) { super(fullName); }
  }

使用選項可以指示是否忽略只有單個字符長度或來自庫方法的覆寫的參數名稱。 如果您不希望受到庫中使用的可疑命名慣例的約束，那麼兩者都會很有用。
