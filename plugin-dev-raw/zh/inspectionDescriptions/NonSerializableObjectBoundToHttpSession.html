報告未實作 java.io.Serializable 來用作 javax.servlet.http.HttpSession.setAttribute() 或 javax.servlet.http.HttpSession.putValue() 參數的物件。

如果 HttpSession 被鈍化或遷移，則不會序列化這樣的物件，並且可能導致難以診斷的錯誤。

此檢查假定 java.util.Collection 和 java.util.Map 類型的物件為 Serializable，除非類型參數為非 Serializable。

範例:


  void foo(HttpSession session) {
      session.setAttribute("foo", new NonSerializable());
  }
  static class NonSerializable {}
