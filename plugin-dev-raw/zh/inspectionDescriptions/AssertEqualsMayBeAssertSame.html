報告可以取代為等效的 assertSame() 方法的 JUnit 的 assertEquals() 方法的調用。 當參數是不覆寫 Object.equals() 方法並顯式比較物件標識的 final 類的實例時，這是可能的。

建議用 assertEquals() 取代

assertSame()。

範例:


  @Test
  public void testObjectType() {
    Object o = getObject();
    Assert.assertEquals(String.class, o.getClass());
  }

在套用快速修復後：


  @Test
  public void testSort() {
    Object o = getObject();
    Assert.assertSame(String.class, o.getClass());
  }
