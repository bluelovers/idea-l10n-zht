報告可以取代為等效 assertSame() 調用的 JUnit assertEquals() 調用。 當實參是不覆寫 Object.equals() 方法並顯式比較物件標識的 final 類的實例時，可能發生這種情況。

建議將 assertEquals() 取代為 assertSame()。

範例：


  @Test
  public void testObjectType() {
    Object o = getObject();
    Assert.assertEquals(String.class, o.getClass());
  }

在套用快速修復後：


  @Test
  public void testSort() {
    Object o = getObject();
    Assert.assertSame(String.class, o.getClass());
  }
