報告實作 java.io.Serializable 且其中的 readResolve() 或 writeReplace() 方法未被宣告為 protected 的類別。

將 readResolve() 和 writeReplace() 方法宣告為 private 可以強制子類別無提示地忽略它們，而宣告為 public 則允許不受信任的程式碼呼叫它們。

如果包含類別被宣告為 final，這些方法就可以被宣告為 private。

範例：


  class ClassWithSerialization implements Serializable {
    public Object writeReplace() { // 警告：'writeReplace()' 未被宣告為 protected
        ...
    }
  }
