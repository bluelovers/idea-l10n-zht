報告具有 = { ... } 和推斷返回類型的函式。

範例：


  fun sum(a: Int, b: Int) = { a + b } // 此函式的返回類型為 '() -> Int'。

有幾個快速修復。 套用快速修復後：


    fun sum(a: Int, b: Int) = a + b // 移除大括號
    fun sum(a: Int, b: Int): () -> Int = { a + b } // 顯式指定返回值類型
    fun sum(a: Int, b: Int) = run { a + b } // 轉換為 run { ... }
    fun sum(a: Int, b: Int) = { -> a + b } // 顯式指定 lambda 簽名
