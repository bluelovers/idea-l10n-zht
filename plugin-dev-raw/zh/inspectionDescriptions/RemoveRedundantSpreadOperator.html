報告針對一系列 arrayOf 函數調用而使用冗餘展開運算符的情況。

使用“移除冗餘展開運算符”快速修復可清理程式碼。

示例：


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo(*arrayOf("abc"))       // 對於 'foo' 的兩次調用，數組創建
      foo(*arrayOf(*ss, "zzz"))  // 及其後續分配均冗餘
  }

在應用快速修復後：


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo("abc")
      foo(*ss, "zzz")
  }
