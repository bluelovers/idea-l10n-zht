報告在以下情況下使用的 toString() 調用：
  • 在字串串聯中
  • 在 java.lang.StringBuilder#append() 或 java.lang.StringBuffer#append() 方法中
  • 在 java.io.PrintWriter 或 java.io.PrintStream 的方法中
  • 在 org.slf4j.Logger 的方法中

在這些情況下，轉換為字串將由底層庫方法處理，不需要顯式調用 toString()。

範例：


  System.out.println(this.toString())

在套用快速修復後：


  System.out.println(this)

請注意，沒有 toString() 調用，程式碼語意可能會有所不同：如果表達式為 null，則將使用 null 字串，而不是拋出 NullPointerException。

使用 僅在已知限定符為非 null 時報告 選項以避免對可能為 null 的值發出警告。
