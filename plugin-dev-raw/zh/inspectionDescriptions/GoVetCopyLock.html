報告鎖錯誤地通過值傳遞。

不小心複製包含鎖的值可能會導致兩個副本均無法正常工作。 通常，此類別值應通過指針參照。 這裡的鎖表示實作 sync.Locker 的類型，例如 sync.Mutex 或 sync.WaitGroup。

範例：

type SafeInt struct {
  m sync.Mutex
  i int
}

func (s SafeInt) Inc() { // 互斥被複製
  s.m.Lock()
  s.i++
  s.m.Unlock()
}

在套用新增指針快速修復後：

type SafeInt struct {
  m sync.Mutex
  i int
}

func (s *SafeInt) Inc() {
  s.m.Lock()
  s.i++
  s.m.Unlock()
}
