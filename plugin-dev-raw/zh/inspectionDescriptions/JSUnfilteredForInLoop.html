報告未篩選的 for-in 循環。

使用此結構不僅會處理對象自身的屬性，還會處理對象原型的屬性。 在某些特定情況下可能出人意料，例如，在復制或修改所有屬性的實用程序方法中 或者 Object 的原型可能被錯誤修改時。 例如，以下代碼將打印 42 和 myMethod： 


Object.prototype.myMethod = function myMethod() {};
let a = { foo: 42 };
for (let i in a) {
  console.log(a[i]);
}

建議將整個循環替換為 Object.keys() 方法，或添加 hasOwnProperty() 檢查。 應用快速修復后，代碼如下所示：


for (let i in a) {
  if (a.hasOwnProperty(i)) {
    console.log(a[i]);
  }
}
