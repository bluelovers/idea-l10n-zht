報告 Python 程式碼中未使用的 import 語句。

此檢查會檢測程式碼中未使用並且可以安全移除的 import 語句。 移除未使用的 import 有助於保持程式碼整潔，並降低名稱衝突的風險。

範例：


import os           # Unused import
import sys         # Used import
from math import pi # Unused import

print(sys.version)

該檢查提供了一項最佳化 import 的快速修復，可以移除所有未使用的 import 語句。

請注意，某些 import 可能會被間接使用（例如，用於副作用），不應被移除。 如果某些 import 是為了副作用而存在，您可以對其禁止此檢查。
