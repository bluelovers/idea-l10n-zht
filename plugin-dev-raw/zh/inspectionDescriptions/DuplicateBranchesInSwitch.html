報告在不同分支中包含相同程式碼的 switch 語句或表達式，並建議合併重複的分支。

範例：


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

在套用快速修復後：


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
