報告建立沒有 DEFAULT 值的 NOT NULL 列的嘗試。

範例（Microsoft SQL 伺服器）：

CREATE TABLE foo (a INT, b  INT)

ALTER TABLE foo ADD c INT NOT NULL;

預設情況下，列包含 NULL 值。 在範例中，我們使用 NOT NULL 約束強制列不接受 NULL 值。 如果我們禁止使用 NULL 值，則必須設定 SQL 在建立新記錄時可以使用的 DEFAULT 值。 

ALTER TABLE foo ADD c INT NOT NULL DEFAULT 42;

您可以使用新增 DEFAULT 值快速修復快速新增 DEFAULT 值。
