報告方法 throws 子句中未檢查的異常的宣告（java.lang.RuntimeException 或其子類之一）。

未檢查的異常的宣告不是必需的，可以刪除或移動到 Javadoc @throws 標記。

範例：


  public class InvalidDataException extends RuntimeException {}

  class TextEditor {
    void readSettings() throws InvalidDataException {} // 警告：在 'throws' 子句中宣告了未檢查的異常 'InvalidDataException'
  }
