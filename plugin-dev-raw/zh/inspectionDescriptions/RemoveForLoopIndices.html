報告使用 withIndex() 函數而非索引變數迭代集合的 for 迴圈。

使用“移除 'for' 迴圈中的索引”快速修復可清理程式碼。

範例：


  fun foo(bar: List<String>) {
     for ((index : Int, value: String) in bar.withIndex()) { // <== 'index' 未使用
         println(value)
     }
  }

在套用快速修復後：


  fun foo(bar: List<String>) {
      for (value: String in bar) { // <== '.withIndex()' 和 'index' 已被移除
          println(value)
      }
  }
