報告被周圍的 catch 塊禁止的 assert 語句和測試框架斷言。 此類別斷言永遠不會失敗，因為將捕獲並無提示地忽略拋出的 AssertionError。

範例 1：


  void javaAssertion() {
    try {
      ...
      assert 1 == 2;
    } catch (AssertionError e) {
      // 該斷言被無提示地忽略
    }
  }

範例 2：


  @Test
  void testWithAssertJ() {
    try {
      ...
      assertThat(1).as("test").isEqualTo(2);
    } catch (AssertionError e) {
      // 該斷言被無提示地忽略
    }
  }

範例 3：


  @Test
  void testWithJunit() {
    try {
      ...
      assertEquals(1, 2);
    } catch (AssertionError e) {
      // 該斷言被無提示地忽略
    }
  }
