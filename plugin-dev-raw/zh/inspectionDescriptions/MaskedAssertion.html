報告使用 catch 塊屏蔽的斷言語句和測試框架斷言。 此類斷言永遠不會失敗，因為將捕獲並無提示地忽略 AssertionError。

範例 1：


void javaAssertion() {
  try {
    ...
    assert 1 == 2; // 斷言被無提示忽略
  } catch (AssertionError e) {}
}

範例 2：


@Test
void testWithAssertJ() {
  try {
    ...
    assertThat(1).as("test").isEqualTo(2); // 斷言被無提示忽略
  } catch (AssertionError e) {}
}

範例 3：


@Test
void testWithJunit() {
  try {
    ...
    assertEquals(1, 2); // 斷言被無提示忽略
  } catch (AssertionError e) {}
}
