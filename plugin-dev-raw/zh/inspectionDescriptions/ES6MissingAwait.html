報告 async 函數中沒有預期的 await 前綴的 async 函數調用。 此類調用返回 Promise，並且控制流會立即繼續。

示例：


async function bar() { /* ... */ }
async function foo() {
    bar(); // 不良
}

應用快速修復後，將添加 await 前綴：


async function bar() { /* ... */ }
async function foo() {
    await bar(); // 優良
}

在選中“報告 return 語句中的 promise”復選框的情況下，還建議在 return 語句中添加 await。
雖然一般不必這樣做，但它有兩大好處。
  • 使用 try-catch 環繞代碼時，您不會忘記去添加 await。
  • 顯式的 await 有助於 V8 運行時提供異步堆棧跟踪。
