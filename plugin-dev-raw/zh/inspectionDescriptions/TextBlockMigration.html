報告可通過取代為文本塊來簡化的 String 串聯。

要求：

  • \n 發生兩次或多次。
  • 文本塊未串聯。

使用應用於單字串文字選項，可建議修正包含換行符的單一文字。

範例：


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

在套用快速修復後：


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;

僅當專案或模組的語言級別為 15 或更高時，此檢查才會報告。
