報告一個返回類型為 boolean 的方法，該方法僅在否定上下文中使用。

您可以使用快速修復來重命名方法和反轉方法。 由於性能原因，某些方法可能不會在編輯器中高亮顯示。

範例：


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

在套用快速修復後：


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
