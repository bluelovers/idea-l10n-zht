報告不正確的 @EventListener 方法。

範例：


  @Configuration @ComponentScan
  open class Config

  data class MyEvent(val string: String)

  @Component
  class LogComponent {
    @EventListener // 使用 @EventListener 註解的方法必須為 public
    private fun logCommands(commandName: MyEvent) {}

    @EventListener // 方法必須最多只有一個參數
    fun processCommand(commandName: MyEvent, text: String) {}
  }
