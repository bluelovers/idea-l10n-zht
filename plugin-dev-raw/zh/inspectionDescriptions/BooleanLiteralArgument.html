報告類型為 Boolean 但未指定顯式參數名稱的呼叫實參。

按順序傳遞多個布爾文字時，很容易忘記參數順序，這可能會導致錯誤。 顯式參數名稱讓閱讀和理解程式碼變得更容易。

範例：


  fun check(checkName: Boolean, checkAddress: Boolean, checkPhone: Boolean) {}

  fun usage() {
      check(true, false, true) //這是什麼意思？
  }

該快速修復會新增缺少的參數名稱：


  fun check(checkName: Boolean, checkAddress: Boolean, checkPhone: Boolean) {}

  fun usage() {
      check(checkName = true, checkAddress = false, checkPhone = true)
  }
