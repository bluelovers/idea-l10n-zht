報告可以安全地取代為 static 內部類別的匿名類別。 如果某個匿名類別沒有從周圍方法顯式引用其封閉實例或局部類別，則它可能是 static 內部類別。

static 內部類別不保留對其封閉實例的隱式引用。 這可以防止常見的記憶體泄漏，並減少每個類別實例使用的記憶體。

該快速修復會將匿名類別提取到命名的 static 內部類別中。

範例：


  void sample(){
    Thread thread = new Thread(new Runnable() {
      @Override
      public void run() {
      }
    });
  }

在套用快速修復後：


  void sample() {
    Thread thread = new Thread(new Task());
  }

  private static class Task implements Runnable {
    @Override
    public void run() {
    }
  }
