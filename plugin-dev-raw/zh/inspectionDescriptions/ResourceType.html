此檢查會檢視使用各種支援註解（如 RequiresPermission 或 UiThread）進行註解的 Android API 呼叫，並標記未按照註解指定的方式正確使用 API 的任何呼叫。

由此檢查標記的錯誤範例：
  • 將錯誤類型的資源整數（如 R.string）傳遞給需要不同類型（如 R.dimen）的 API。
  • 忘記在需要它的方法中呼叫被覆寫的方法（通過 super）
  • 呼叫需要權限但未在清單中宣告該權限的方法
  • 將資源顏色參照傳遞給需要 RGB 整數值的方法。
…以及更多。
