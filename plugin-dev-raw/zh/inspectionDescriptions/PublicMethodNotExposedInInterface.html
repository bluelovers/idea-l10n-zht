報告未在接口中公開的類別中的 public 方法。

通過接口公開所有 public 方法對於維護鬆散耦合非常重要，某些基於組件的編程風格可能需要這樣做。

範例：


interface Person {
  String getName();
}

class PersonImpl implements Person {
  private String name;

  // 正常：方法在接口中公開
  @Override
  public String getName() {
    return name;
  }

  // 警告：方法為 public
  // 但未在接口中公開
  public void setName() {
    this.name = name;
  }
}

使用有註解時忽略列表可指定特殊註解。 此檢查將忽略使用其中一種註解進行註解的方法。

使用如果包含類別未實作非庫接口，則忽略選項可以忽略未實作專案中任何接口的類別中的方法。
