報告異常始終可以通過包含 try 語句捕獲的 throw 語句。

使用 throw 語句作為 "goto" 來更改區域控制流會令人困惑並導致性能不佳。

範例：


  try {
    if (!Files.isDirectory(PROJECTS)) {
        throw new IllegalStateException("Directory not found."); // 警告：'throw' 被所包含的 'try' 語句捕獲
    }
    ...
  } catch (Exception e) {
      LOG.error("run failed");
  }

使用忽略重新拋出的異常選項可忽略重新拋出的異常。
