報告在多個目標上使用 @targetName 註解的情況。

該快速修復會移除 @targetName 註解。

範例：


  // 1
  val list: List[Int] = (1 to 5).toList
  @targetName("head_and_tail")
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    @targetName("b_and_c")
    case B, C
    case D

在套用快速修復後：


  // 1
  val list: List[Int] = (1 to 5).toList
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    case B, C
    case D
