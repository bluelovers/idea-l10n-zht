報告對使用 java.lang.Object 中預設實作的 toString() 的呼叫。

該預設實作很少是有意為之，可能是誤用。

此檢查將忽略對包含 java.lang.Object、接口或 abstract 類別類型的物件的 toString() 呼叫。

範例：


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // 警告
          /* ... */
      }

      void foo2(Object obj) {
          String s = obj.toString(); // 此處無警告
          /* ... */
      }
  }
