報告 Spring Shell org.springframework.shell.standard.ShellMethodAvailability 註解內的無效引用。

範例：


@ShellComponent
public class MyCommands {

    @ShellMethod("sum")
    @ShellMethodAvailability("sumAvailability") // 註解值將高亮顯示為未解析的符號
    public int sum(int a, int b) {
        return a + b;
    }

    @ShellMethodAvailability("sumMethod") // 註解值將高亮顯示為未解析的符號
    public Availability isSumAvailable() {
        return Availability.available();
    }
}
