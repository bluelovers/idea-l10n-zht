報告隱式轉換為 long 的整數乘法和左移。

範例：


  void f(int i) {
    long val = 65536 * i;
  }

套用快速修復後，程式碼變為：


  void x(int i) {
    long val = 65536L * i;
  }

範例：


  void f(int i) {
    long value = i << 24;
  }

套用快速修復後，程式碼變為：


  void f(int i) {
    long value = (long) i << 24;
  }

此類別乘法通常表明有錯誤，因為可能會意外發生溢出截斷。 將 int 文字轉換為 long 文字 (65536L) 可以修正該問題。
