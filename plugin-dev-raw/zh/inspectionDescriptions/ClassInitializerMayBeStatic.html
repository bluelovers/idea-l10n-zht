報告可以使之為 static 的實例初始設定式。

如果實例初始設定式不參照其類別的任何非 static 成員，則可以為 static。 static 初始設定式在類別解析後執行，而實例初始設定式對此類別的每個實例化執行。

此檢查不報告匿名類別中的實例空初始設定式和初始設定式。

範例：


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

在套用快速修復後：


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // 現在每個類別僅實例化一次
     }
  }
