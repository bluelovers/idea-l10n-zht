報告可以轉換為建構函式或帶有 lambda 表達式實參的工廠方法呼叫的匿名類別。

本次檢查報告以下類別：

  • 擴展 ThreadLocal 的匿名類別，其具有 initialValue() 方法（可以取代為 ThreadLocal.withInitial）
  • 擴展 Thread 的匿名類別，其具有 run() 方法（可以取代為 new Thread(Runnable)）

範例：


  new Thread() {
    @Override
    public void run() {
      System.out.println("Hello from thread!");
    }
  }.start();

在套用快速修復後：


  new Thread(() -> {
    System.out.println("Hello from thread!");
  }).start();
