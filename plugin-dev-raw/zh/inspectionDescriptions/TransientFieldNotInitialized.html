報告在正常物件建構過程中初始化，但其類別沒有 readObject 方法的 transient 欄位。

transient 欄位未被序列化，因此在反序列化期間，需要在 readObject() 方法中單獨初始化。

任何在正常物件建構過程中未被初始化的 transient 欄位都被視為使用預設初始化，並且不會被此檢查報告。

範例：


  class Person implements Serializable {
    transient String name = "Default"; //警告，反序列化後實際上可能為 null
    transient String surname; //null 被視為預設值，不予報告
  }
