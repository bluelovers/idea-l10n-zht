報告主要用作基元類型的包裝器類型的區域變數。

在某些情況下，裝箱可能會導致嚴重的性能損失，尤其是在迴圈中。

採用啟發估計裝箱動作次數。 例如，迴圈內的轉換數量視為更多。

範例：

  public void example() {
    Integer value = 12;
    needBox(value);
    for (int i = 0; i < 10; i++) {
      // 迴圈的用法正變得越來越頻繁
      needPrimitive(value);
    }
  }

  void needPrimitive(int value) {}
  void needBox(Integer value) {}

在套用快速修復後：

  public void example() {
    int value = 12;
    needBox(value);
    for (int i = 0; i < 10; i++) {
      // 迴圈的用法正變得越來越頻繁
      needPrimitive(value);
    }
  }

  void needPrimitive(int value) {}
  void needBox(Integer value) {}
