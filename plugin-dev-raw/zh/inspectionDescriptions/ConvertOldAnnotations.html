報告 TestNG org.testng.annotations.Configuration 註解。

最好使用現代 @BeforeXXX/@AfterXXX 註解來組態測試套件。

範例：


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

在套用快速修復後：


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
