報告不必要的 this 或 super 限定。

在 this 或 super 上使用限定符來消除程式碼參照的歧義可能很容易通過自動重構變得不必要，為了清楚起見應該刪除。

範例：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

在套用快速修復後：


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
