報告沒有在迴圈中呼叫的 java.util.concurrent.locks.Condition.await()。

await() 和相關方法通常用於掛起執行緒，直到某個條件變為 true 為止。 執行緒可能出於不同原因而被喚醒，因此應在 await() 呼叫返回後檢查條件。 使用迴圈很容易做到這一點。

範例：


  void acquire(Condition released) throws InterruptedException {
    released.await();
  }

優良的程式碼應類似於：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
