報告對可以取代為 associateBy() 或 associateWith() 的 associate() 和 associateTo() 的調用。

兩個函數都接受應用於給定序列或集合（作為接收器）的元素的轉換器函數。 然後該對函數被用於建置結果 Map。

鑑於轉換器引用 it，associate[To]() 調用可以取代為性能更高的 associateBy() 或 associateWith()。

範例：

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // 可取代的 'associate()'
      listOf(1).associate { it to getValue(it) } // 可取代的 'associate()'
  }

在套用快速修復後：

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }
