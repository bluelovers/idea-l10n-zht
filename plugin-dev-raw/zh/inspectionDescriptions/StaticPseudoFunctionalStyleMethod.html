如果 Java Stream API 可用，則報告偽函數程式碼的用法。

雖然 guava Iterable API 提供類似於 Java Streams API 的功能，但它略有不同，可能會缺少某些功能。 特別是，IntStream 等基元專門化的流變體比一般變體的性能更好。

範例：


List<String> transformedIterable = Iterables.transform(someList, someTransformFunction);//警告：偽函數樣式程式碼

在套用快速修復後：

List<String> transformedIterable = someList.stream().map(someTransformFunction).collect(Collectors.toList());

注意：程式碼語意可能改變：例如 guava 的 Iterable.transform 產生惰性求值的可迭代物件，但取代為及早求值。

使用轉換為 'Stream' API 的 static 方法調用選項，設定應轉換為 stream API 的 static 方法調用。

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。
