報告註解的 Seam 類的配置問題

示例：


@Name("authenticator")
class SeanComponent {
  @Create // 不允許使用 static 方法
  private static void create() {...}

   @Destroy // 不允許使用 static 方法
   private static void destroy() {...}

   @DataModel // 非 getter
   private Map method() {...}

   @Factory // 應為 public
   private Map getUniverse2() {...}

   @Unwrap // 必須沒有形參
   private static Map createUniverse(int days) { ... }
}
