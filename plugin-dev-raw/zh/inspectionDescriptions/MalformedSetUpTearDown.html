報告不聲明為 public、不返回 void 或接受實參的 JUnit 3 setUp() 或 tearDown() 方法。 此類 setUp() 或 tearDown() 方法很可能是無意中創建的，JUnit 測試執行程序不會執行這些方法。

示例：


  public class WeightyTest extends TestCase {
    private File file;
    private int setUp(String prefix) throws IOException {
      file = File.createTempFile(prefix, ".tmp");
      return 0;
    }

    // ... 測試轉到此處
  }
