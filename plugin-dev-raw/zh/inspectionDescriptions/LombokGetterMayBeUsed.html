報告可由 lombok @Getter 註解取代的標準 getter 方法。

範例：


  import java.util.Date;

  public class MyClass {
    /**
     * 處理日期。
     */
    private Date processDate;

    /**
     * 返回日期。
     *
     * @return The date
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

在套用快速修復/清理後：


  import lombok.Getter;
  import java.util.Date;

  @Getter
  public class MyClass {
    /**
     * 處理日期。
     * -- GETTER --
     * 返回日期。
     *
     * @return The date
     */
    private Date processDate;
  }

它僅在組態 lombok 庫時報告。 要移除 lombok，請參閱 lombok 網站。
