報告可以取代為字串模板的 String.format 呼叫。

使用字串模板可使程式碼更簡單。

該快速修復會將呼叫取代為字串模板。

範例：


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

在套用快速修復後：


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
