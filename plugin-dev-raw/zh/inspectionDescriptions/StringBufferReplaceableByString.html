報告可以用單個 String 串聯取代的 StringBuffer、StringBuilder 或 StringJoiner 的用法。

使用 String 串聯會使程式碼更短更簡單。

此檢查僅在建議的取代不會導致現代 JVM 的性能顯著下降時報告。 在許多情況下，String 串聯可能執行效果更好。

範例:

  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

在套用快速修復後：

  String result = "i = " + i + ";";
  return result;
