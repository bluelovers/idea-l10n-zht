報告數值類型之間的隱式轉換。

隱式數字轉換本身不是問題，但如果是意外，可能會在追蹤錯誤時造成困難。

範例：


  double m(int i) {
    return i * 10;
  }

在套用快速修復後：


  double m(int i) {
    return (double) (i * 10);
  }

組態檢查：

  • 使用忽略擴大轉換選項可以忽略不會導致資料丟失的隱式轉換（例如，int->long）。
  • 使用忽略 'char' 與其他類型之間的轉換選項可以忽略 char 與其他類型之間的轉換。 該檢查仍將報告浮點數與其他類型之間的轉換。
  • 使用忽略來自常數和文字的轉換可使檢查忽略來自文字和編譯時常量的轉換。
