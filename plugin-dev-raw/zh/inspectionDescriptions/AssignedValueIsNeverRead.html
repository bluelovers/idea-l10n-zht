報告賦值後從未在程式碼中使用所賦值的情況。

範例：


  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
      local = 42  // 所賦值從未被讀取
  }
  • 在快速修復後，該賦值被刪除：

  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
  }
  • 如果右側 (RHS) 表達式可能具有副作用，該快速修復還會建議保留 RHS 表達式以保留這些副作用： 

  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
      foo()  // 該函式呼叫因其副作用而保留
  }
