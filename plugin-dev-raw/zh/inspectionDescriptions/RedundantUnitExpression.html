報告冗餘的 Unit 表達式。

Kotlin 中的 Unit 可以用作不返回任何有意義的函式的返回類型。 Unit 類型只有一個可能的值，即 Unit 物件。

範例：


  fun redundantA(): Unit {
      return Unit // 冗餘，'Unit' 預設返回並與預期返回類型符合
  }

  fun requiredA(condition: Boolean): Any {
      if (condition) return "hello"
      return Unit // 顯式 'Unit' 為必選項，因為預期類型為 'Any'
  }

  fun redundantB(condition: Boolean): Any = if (condition) {
      fun ancillary(): Int = 1
      println("${ancillary()}")
      Unit // 冗餘，因為上一個表達式已經為 'Unit' 類型
  } else {
      println("else")
  }

  fun requiredB(condition: Boolean): Any = if (condition) {
      1024
      Unit // 必選項，否則 '1024' (Int) 將為返回值
  } else {
      println("else")
  }
