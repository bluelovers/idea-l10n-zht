報告對不存在或不可訪問的資料提供者方法的引用。

範例：


public class InstanceDataProviderFromForeignClass {
  // 方法 data() 在類 A 中不存在
  @Test(dataProvider = "data", dataProviderClass = A.class)
  public void test() {
  }
}
class A { }

在套用快速修復後：


//所需的 data() 方法在類 A 中創建
class A {
  @DataProvider
  public Object[][] data() {
    return new Object[][]{};
  }
}
