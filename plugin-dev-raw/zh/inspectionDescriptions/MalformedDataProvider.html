報告對不存在或不可存取的資料提供者方法的參照。

範例：


public class InstanceDataProviderFromForeignClass {
  // 方法 data() 在類別 A 中不存在
  @Test(dataProvider = "data", dataProviderClass = A.class)
  public void test() {
  }
}
class A { }

在套用快速修復後：


//所需的 data() 方法在類別 A 中建立
class A {
  @DataProvider
  public Object[][] data() {
    return new Object[][]{};
  }
}
