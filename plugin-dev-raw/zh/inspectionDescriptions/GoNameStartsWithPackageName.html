報告以軟體套件名稱開頭的匯出名稱。 此檢查不會在 main 軟體套件中報告此類別名稱。

範例：

package myPackage

func MyPackageGetIP() {

}

MyPackageGetIP 名稱將醒目提示，因為它以軟體套件名稱開頭。

根據 github.com/golang 上的 Package Names，對軟體套件中名稱的所有參照都將使用軟體套件名稱完成，因此可以在關鍵字中省略該名稱。 例如，如果您在軟體套件 foo 中，則不需要類型 FooFile，客戶端將其寫為 foo.FooFile。 相反，我們將類型命名為 File，客戶端將其寫為 foo.File。
