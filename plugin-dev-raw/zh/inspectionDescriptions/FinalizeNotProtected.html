報告被宣告為 public 的 Object.finalize() 方法的任何實作。

根據 Object.finalize() 的約定，只有垃圾回收器呼叫此方法。 將此方法設為 public 可能令人困惑，因為這意味著該方法可以從其他程式碼中使用。

可以通過快速修復將該方法設為 protected，以防止它被其他類別呼叫。

範例：


  class X {
    public void finalize() {
      /* ... */
    }
  }

在套用快速修復後：


  class X {
    protected void finalize() {
      /* ... */
    }
  }
