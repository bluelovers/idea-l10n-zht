報告被宣告為 public 的 Object.finalize() 方法的任何實作。

根據 Object.finalize() 的約定，只有垃圾回收器調用此方法。 將此方法設為 public 可能令人困惑，因為這意味著該方法可供使用者使用。

該快速修復會將方法設為 protected，防止它被其他類顯式調用。

範例：


  class X {
    public void finalize() {
      /* ... */
    }
  }

在套用快速修復後：


  class X {
    protected void finalize() {
      /* ... */
    }
  }
