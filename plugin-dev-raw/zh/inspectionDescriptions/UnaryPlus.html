報告使用 + 一元運算符的情況。 一元加號通常為 null 運算，它的存在可能代表編碼錯誤。 例如，與遞增運算符（如 +++ 中）或相等運算符（如 =+ 中）結合使用。

範例：


  void unaryPlus(int i) {
    int x = + +i;
  }

建議使用以下快速修復：

  • 移除 i 變數前的 + 運算符：

    
  void unaryPlus(int i) {
    int x = i;
  }

  • 使用前綴遞增運算符取代 + 運算符：

    
  void unaryPlus(int i) {
    int x = ++i;
  }

使用下面的複選框可以報告與二進制或另一個一元表達式一起使用的一元加號。 這意味著檢查將不會報告將一元加號表達式用於陣列初始設定式表達式或用作方法實參的情況。
