報告在 java.util.concurrent.locks.Condition 物件上對 notify() 或 notifyAll() 的呼叫。

這可能是編程錯誤，本應使用 signal() 或 signalAll() 方法的某些變體，否則可能出現 IllegalMonitorStateException。

範例：


  class C {
    final Lock l = new ReentrantLock();
    final Condition c = l.newCondition();

    void release() {
      l.lock();
      try {
        c.notifyAll(); // 此處可能本應使用 'signalAll()'
      } finally {
        l.unlock();
      }
    }
  }
