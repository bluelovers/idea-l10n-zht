報告迴圈中最後能夠到達的 continue 語句。 這些 continue 語句是不必要的語句，可以安全移除。

範例：


  for (String element: elements) {
    System.out.println();
    continue;
  }

在套用快速修復後：


  for (String element: elements) {
    System.out.println();
  }

該檢查不分析 JSP 檔案。

使用在具有 'else' 分支的 'if' 語句的 then 分支中忽略選項可以忽略位於完整 if-else 語句的 then 分支中的 continue語句。

範例：


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
