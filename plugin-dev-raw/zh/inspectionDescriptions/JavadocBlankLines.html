報告 Javadoc 註釋中的空行。

Javadoc 中的空行可能表示將文本拆分為不同段落的意圖。 但是，Javadoc 工具和 IntelliJ IDEA 在呈現文檔註釋時會忽略它們。

該快速修復會建議將空行取代為段落標記 (<p>)。

範例：


  class Main {
    /**
     * 不起作用。
     *
     * 完全不起作用
     */
    void foo() {}
  }

在套用快速修復後：


  class Main {
    /**
     * 不起作用。
     * <p>
     * 完全不起作用
     */
    void foo() {}
  }
