報告 PEP-544 中引入的協議的無效定義和用法。

範例：

from typing import Protocol


class MyProtocol(Protocol):
    def method(self, p: int) -> str:
        pass


class MyClass(MyProtocol):
    def method(self, p: str) -> int: # 'method' 的類型不相容 'MyProtocol'
        pass


class MyAnotherProtocol(MyClass, Protocol): # 協議的所有父類必須為協議
    pass
