報告具有三個或更多取反運算符 ! 或 != 的方法。

此類方法可能會引起混淆。

範例:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

沒有取反，該方法變得更容易理解：


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

設定檢查：

  • 使用忽略 'equals()' 方法中的否定選項可禁用 equals() 方法中的檢查。
  • 使用忽略 'assert' 語句中的否定可禁用 assert 語句中的檢查。
