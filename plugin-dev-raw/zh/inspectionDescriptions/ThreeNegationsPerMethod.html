報告有三個或更多否定的方法。 此類別方法可能令人困惑。

範例：


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

沒有否定，該方法變得更容易理解：


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

組態檢查：

  • 使用忽略 'equals()' 方法中的否定選項可停用 equals() 方法中的檢查。
  • 使用忽略 'assert' 語句中的否定可停用 assert 語句中的檢查。
