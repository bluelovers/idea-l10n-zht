報告錯誤字串的格式問題。

範例：

err := fmt.Errorf("Cannot read the file!")
log.Printf("Reading %s: %v", file, err)

根據 github.com/golang 上的 Error Strings，錯誤字串不應大寫或以標點符號結尾，因為它們可能會出現在其他上下文中。

要修正格式，可以套用修正錯誤字串格式快速修復。 套用快速修復後，錯誤字串將如下所示：

err := fmt.Errorf("cannot read the file")
log.Printf("Reading %s: %v", file, err)
