報告未宣告 public 和 static、不返回 void 或包含參數的 JUnit 4.0 @BeforeClass 或 @AfterClass 方法。

這樣的方法很容易無意間創建，但是 JUnit 測試執行程序不會執行這些方法。

範例:


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) {
    }
  }

套用快速修復後，該方法變為:


  class MainTest {
    @BeforeClass
    public static void beforeClass() {
    }
  }
