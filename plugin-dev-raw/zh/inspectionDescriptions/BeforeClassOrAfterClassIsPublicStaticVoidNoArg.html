報告非 public、非 static、返回類型為 void 或在沒有宣告參數解析器的情況下接受實參的 JUnit 4 @BeforeClass 與 @AfterClass 註解方法和 JUnit 5 @BeforeAll 與 @AfterAll 註解方法。 此類方法很容易無意間創建，但是 JUnit 測試執行程序不會執行這些方法。

範例：


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) { ... }
  }

在套用快速修復後：


  class MainTest {
    @BeforeClass
    public static void beforeClass() { ... }
  }
