在 Kotlin/JS 專案中報告擴展 React Props 或 State 類型但未標記為 external 的類別、接口和物件。

React 組件類型應當為外部接口，以便正確地與 JavaScript 執行時互動作。

範例：

interface MyProps : Props { // 警告：應為 external 接口
    var title: String
}

class MyState : State { // 警告：應為 external 接口
    var count: Int = 0
}

在套用快速修復後：

external interface MyProps : Props {
    var title: String
}
