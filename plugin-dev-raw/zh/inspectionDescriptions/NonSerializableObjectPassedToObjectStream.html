報告用作 java.io.ObjectOutputStream.write() 實參的非 Serializable 物件。 此類別呼叫將導致執行時異常。

此檢查假定 java.util.Collection 和 java.util.Map 類型的物件為 Serializable，除非宣告它們的類型為非 Serializable。

範例：


  public class IWantToSerializeThis {
    public static void main(String[] args) throws IOException {
      try(var stream = new ObjectOutputStream(Files.newOutputStream(Paths.get("output")))) {
        // 警告 -- 將會失敗並拋出 NotSerializableException
        stream.writeObject(new IWantToSerializeThis());
      }
    }
  }
