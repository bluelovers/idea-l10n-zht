報告帶有太少 case 標籤的 switch 語句和表達式，並建議將它們覆寫為 if 和 else if 語句。

範例 (minimum branches == 3)：


  switch (expression) {
    case "foo" -> foo();
    case "bar" -> bar();
  }

在套用快速修復後：


  if ("foo".equals(expression)) {
    foo();
  } else if ("bar".equals(expression)) {
    bar();
  }

設定檢查：

使用最小分支數 欄位指定 case 標籤的最小預期數量。
