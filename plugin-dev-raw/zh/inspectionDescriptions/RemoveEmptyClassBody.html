報告具有空主體的類別和物件的宣告。

使用“移除冗餘的空類別主體”快速修復可清理程式碼。

範例：


  class EmptyA() {} // <== 空主體

  class EmptyB {
      companion object {} // <== 空主體
  }

  fun emptyC() {
     object {} // <== 匿名物件，可以（未報告）
  }

在套用快速修復後：


  class EmptyA()

  class EmptyB {
      companion object
  }

  fun emptyC() {
     object {}
  }
