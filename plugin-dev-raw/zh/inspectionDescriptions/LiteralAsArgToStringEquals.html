報告包含字串文字實參的 String.equals() 或 String.equalsIgnoreCase() 調用。

某些編碼標準指定字串文字應當為 equals() 的限定符而不是實參，從而最大限度減少 NullPointerException。

可以通過快速修復交換文字和表達式。

範例：


  boolean isFoo(String value) {
    return value.equals("foo");
  }

在套用快速修復後：


  boolean isFoo(String value) {
    return "foo".equals(value);
  }
