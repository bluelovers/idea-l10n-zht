報告任何對 Serializable lambda、區域類別和匿名類別外部的非 Serializable 區域變數的參照。

從匿名類別參照區域變數時，它的值存儲在該類的隱式欄位中。 對於區域類別和 lambda 也是如此。 如果變數類型為非 Serializable，序列化將失敗。

範例：


  interface A extends Serializable {
      abstract void foo();
  }
  class B {}
  class C {
      void foo() {
          B b = new B();
          A a = new A() {
              @Override
              public void foo() {
                  System.out.println(b); // 警告
              }
          };
      }
  }
