報告任何無需創建類實例並且可以設置為 static 的方法。

示例：

class MyClass(object):
    def my_method(self, x):
        print(x)

如果應用從方法中生成函數快速修復，代碼將變為：

def my_method(x):
    print(x)


class MyClass(object):
    pass

如果選擇將方法設為 static 快速修復，將添加 @staticmethod 裝飾器：

class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
