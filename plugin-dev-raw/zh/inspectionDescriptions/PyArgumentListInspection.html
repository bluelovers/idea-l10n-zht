報告聲明的參數和實際實參之間的差異，也報告不正確的實參（例如重複的命名實參）和不正確的實參順序。

示例：

class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__() # 參數未填
bar(5, "#") # 非預期實參

正確的程式碼段如下所示：

class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__(5)
bar(5, p2="#")
