報告可以取代為 Kotlin stdlib 的 Java Collections static 方法調用。

範例：


  import java.util.Collections

  fun test() {
      val mutableList = mutableListOf(1, 2)
      Collections.fill(mutableList, 3)
  }

該快速修復會將 Java Collections static 方法調用取代為相應的 Kotlin stdlib 方法調用：


  import java.util.Collections

  fun test() {
      val mutableList = mutableListOf(1, 2)
      mutableList.fill(3)
  }
