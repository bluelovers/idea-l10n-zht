報告不將 `infix` 修飾符作為中綴運算符的字母數字定義的用法。

  • “用反引號包裝”快速修復會在調用周圍添加 ` 符號。
  • “從中綴表達式轉換”快速修復會將中綴調用轉換為方法語法 .method(...)。
  • “從中綴類型轉換”快速修復會將中綴類型轉換為前綴語法 Type[...]。
  • “從中綴模式轉換”快速修復會將中綴模式轉換為前綴語法 pattern(...)。

範例：


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c op 2
  val or: Int Or String = ???
  val _ Pair _ = p

套用“用反引號包裝”快速修復後：


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c `op` 2
  val or: Int `Or` String = ???
  val _ `Pair` _ = p

在套用“從中綴表達式轉換”快速修復後：


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c.op(2)
  val or: Int Or String = ???
  val _ Pair _ = p

套用“從中綴類型轉換”快速修復後：


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c op 2
  val or: Or[Int, String] = ???
  val _ Pair _ = p

套用“從中綴模式轉換”快速修復後：


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c op 2
  val or: Int Or String = ???
  val Pair(_, _) = p
