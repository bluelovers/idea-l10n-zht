報告使用 @RunWith(Parameterized.class) 註解，但要麼不包括使用 @Parameterized.Parameters 註解的資料提供程序方法，要麼此方法的簽名不正確的 JUnit 4 參數化測試類。 這種測試類無法執行。 資料提供程序方法應為 public 和 static 並具有 Iterable 或 Object[] 的返回類型。

建議創建一個空的參數提供程序方法或更改不正確的資料提供程序方法的簽名。

範例：


  @RunWith(Parameterized.class)
  public class ImportantTest {
    private int input;
    private int expected;

    ImportantTest(int input, int expected) {
      this.input = input;
      this.expected = expected;
    }

    // …測試用例
  }

在套用快速修復後：


  @RunWith(Parameterized.class)
  public class ImportantTest {
    private int input;
    private int expected;

    ImportantTest(int input, int expected) {
      this.input = input;
      this.expected = expected;
    }

    @Parameters
    public static Iterable<Object[]> parameters() {
      return null;
    }

    // …測試用例
  }
