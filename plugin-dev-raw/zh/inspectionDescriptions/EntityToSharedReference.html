報告從 Entity 到 SharedEntity 的無效引用。

不能從 SharedEntity 引用 Entity。 僅當屬性為可 null 或具有 @CascadeDeleteBy 註解時，才能從 Entity 引用 SharedEntity。 否則，可能會導致執行時異常。

範例：


  interface A: Entity {
    var x: SharedEntity
    var y: SharedEntity
  }

可以通過快速修復來生成註解或使類型為可 null


  interface A: Entity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
