建議使用欄位初始設定式取代通過賦值的欄位初始化。

僅當欄位初始設定式位於初始設定式中時才會高亮顯示程式碼，並且將其與欄位宣告結合可能是安全的。 在其他情況下，檢查提供沒有程式碼高亮顯示的快速修復，因為修復可能會更改程式碼語意。

範例：


  class MyClass {
    static final int intConstant;
    
    static {
      intConstant = 10;
    }
  }
快速修復將賦予的值移動到欄位初始設定式，並移除類初始設定式部分（如果可能）： 

  class MyClass {
    static final int intConstant = 10;
  }
