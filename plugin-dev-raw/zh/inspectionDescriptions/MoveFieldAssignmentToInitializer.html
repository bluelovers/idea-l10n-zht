建議使用欄位初始值設定項取代通過賦值的欄位初始化。

僅當欄位初始值設定項位於初始值設定項中時才會高亮顯示程式碼，並且將其與欄位宣告結合可能是安全的。 在其他情況下，檢查提供沒有程式碼高亮顯示的快速修復，因為修復可能會更改程式碼語義。

範例：


  class MyClass {
    static final int intConstant;
    
    static {
      intConstant = 10;
    }
  }
快速修復將賦予的值移動到欄位初始值設定項，並移除類初始值設定項部分（如果可能）： 

  class MyClass {
    static final int intConstant = 10;
  }
