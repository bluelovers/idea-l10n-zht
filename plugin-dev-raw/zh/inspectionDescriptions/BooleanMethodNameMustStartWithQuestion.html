報告名稱不以疑問詞開頭的布爾方法。

此檢查將忽略覆寫庫方法的布爾方法。

範例：

boolean empty(List<String> list) {
  return list.isEmpty();
}

只能在編輯器中使用重命名此類方法的快速修復。

設定檢查：

  • 使用布爾方法名稱前綴列表可指定布爾方法名稱開頭可以接受的疑問詞。
  • 使用忽略具有 'java.lang.Boolean' 返回類型的方法選項，可忽略返回類型為 java.lang.Boolean 的方法。
  • 使用忽略 @&interface 中的布爾方法選項，可忽略註解返回類型 (@interface) 的布爾方法。
  • 使用忽略覆寫/實作 super 方法的方法選項，可忽略具有 super 方法的方法。
