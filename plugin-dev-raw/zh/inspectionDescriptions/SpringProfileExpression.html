報告不正確的 @Profile 表達式：

  • Spring 設定檔案不得為空
  • 在 Spring 設定檔案表達式中，'&' 和 '|' 運算符不得在沒有圓括號的情況下混用

範例：


  
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  
  @Configuration
  @Profile() // 報告 "設定檔案表達式必須包含文本"
  public class MyConfiguration {...}

  
  @Configuration
  @Profile("a & b | c") // 報告 "格式不正確的設定檔案表達式"
  public class MyConfiguration {...}
