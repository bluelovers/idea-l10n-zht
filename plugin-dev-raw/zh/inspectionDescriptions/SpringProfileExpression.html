報告不正確的 @Profile 表達式：

  • Spring 設定檔案不能為空
  • '&' 和 '|' 在 Spring 設定檔案表達式中，不得在沒有括號的情況下混合使用運算符

範例:


  
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  
  @Configuration
  @Profile() // 報告 "設定檔案表達式必須包含文本"
  public class MyConfiguration {...}

  
  @Configuration
  @Profile("a & b | c") // 報告 "格式不正確的設定檔案表達式"
  public class MyConfiguration {...}
