如果沒有為模組組態 API 版本控制，則會醒目提示端點的 version 屬性。

可以通過兩種方式組態 API 版本控制：

  • 在 Spring Boot 專案的組態檔案中組態
  • 或提供 org.springframework.web.servlet.config.annotation.WebMvcConfigurer 的實作並覆寫 configureApiVersioning 方法。

例如： 


  # 對於 Spring Boot 專案
  spring.mvc.apiversion.use.header=Api-Version
  spring.mvc.apiversion.default=1.0
  spring.mvc.apiversion.detect-supported=false

或： 


  // Spring Boot 和 Spring Framework 專案均支援使用這種方式
  @Configuration
  public class MyWebConfig implements WebMvcConfigurer {

      @Override
      public void configureApiVersioning(ApiVersionConfigurer configurer) {
          configurer
                  .useRequestHeader("Api-Version")
                  .setDefaultVersion("1.0")
                  .detectSupportedVersions(false);
      }
  }
