使用未實作所需接口的特定 JSF 註解的報告類別：

  • @FacesComponent 必須擴展 <javax/jakarta>.faces.component.FacesComponent
  • @FacesRenderer 必須擴展 <javax/jakarta>.faces.render.Renderer
  • @FacesConverter 必須擴展 <javax/jakarta>.faces.convert.Converter
  • @FacesValidator 必須擴展 <javax/jakarta>.faces.validator.Validator

範例：



@FacesConverter(value="...")  // 類別必須擴展 'javax.faces.convert.Converter'
public class FooConverter {
}

@FacesRenderer(rendererType="..", componentFamily="..")  // 類別必須擴展 'javax.faces.render.Renderer'
public class FooConverter {
}

@FacesRenderer(rendererType="..", componentFamily="..")  // 類別必須擴展 'javax.faces.validator.Validator'
public class FooValidator {
}
