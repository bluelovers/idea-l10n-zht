報告常數宣告內 iota 的無規律用法。

iota 關鍵字會針對每個常數宣告重設，並針對每個常數規範自動遞增。 在一個常數規範內，關鍵字會保留其值。 顯式參照 iota 不會重設計數器。

有關詳細資訊，請參閱 Go 規範中的 Iota。

如果兩個常數規範具有文本相同的表達式列表，其中至少包含一個對 iota 的參照，並且兩個常數規範之間只有常數規範，而沒有表達式列表，則會觸發此檢查。

考慮省略冗餘的表達式列表，或者每次都寫出表達式列表。

範例：

const (
  a = iota  // 0
  b         // 1
  c = iota  // 2
)

由於 c 的定義中的 iota 冗餘，觸發檢查。

範例：

const (
  a, aa = iota, iota       // 0, 0
  b, bb                    // 1, 1
  c, cc = iota + 40, iota  // 42, 2
)

不會觸發檢查，因為沒有一個表達式列表是冗餘的。
