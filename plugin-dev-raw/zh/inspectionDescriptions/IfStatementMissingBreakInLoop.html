報告包含 if 語句並且以 break 結尾但又不改變語意的迴圈。

這可以防止冗餘的迴圈迭代。

範例:

  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

在套用快速修復後：

  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
