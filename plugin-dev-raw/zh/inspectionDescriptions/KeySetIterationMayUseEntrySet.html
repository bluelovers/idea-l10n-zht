報告對 java.util.Map 實例的 keySet() 的迭代，其中迭代的鍵用於從映射中檢索值。

取代為 entrySet() 或 values() 的迭代時，此類別迭代可能會更有效（如果實際上沒有使用鍵）。

同理，如果在 lambda 中檢索值，則 keySet().forEach(key -> ...) 可以取代為 forEach((key, value) -> ...)。

範例：


  for (Object key : map.keySet()) {
   Object val = map.get(key);
  }

在套用快速修復後：


  for (Object val : map.values()) {}
