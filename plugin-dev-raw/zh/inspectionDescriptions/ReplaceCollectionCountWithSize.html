報告對 Collection<T>.count() 的調用。

此函數調用可以取代為 .size。

.size 形式可以確保運算是 O(1) 並且不會分配額外的對象，而 count() 可能與 Iterable<T>.count() 混淆，後者是 O(n) 和分配。

示例：

  fun foo() {
      var list = listOf(1,2,3)
      list.count() // 可取代的 'count()'
  }

在應用快速修復後：

  fun foo() {
      var list = listOf(1,2,3)
      list.size
  }
