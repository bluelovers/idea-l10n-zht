報告 protected 嵌套類。

範例：


  public class Outer {
    protected static class Nested {} // 警告
    protected class Inner {}         // 警告
    protected enum Mode {}           // 警告取決於設定
    protected interface I {}         // 警告取決於設定
  }

設定檢查：

  • 使用忽略 'protected' 內部枚舉選項可忽略 protected 內部枚舉。
  • 使用忽略 'protected' 內部接口選項可忽略 protected 內部接口。
