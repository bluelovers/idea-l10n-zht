報告構造函數中未初始化的註解為非 null 的欄位。

範例：

public class MyClass {
  private @NotNull String value;

  public void setValue(@NotNull String value) {
    this.value = value;
  }

  public @NotNull String getValue() {
    return value;
  }
  }

此類欄位可能違反非 null 約束。 在上例中，setValue 參數註解為非 null，但是如果沒有調用 setter，getValue 可能返回 null。

設定檢查：

  • 使用忽略可以隱式初始化的欄位選項，可控制該欄位能夠隱式初始化（例如通過依賴項注入）時是否應發出警告。
  • 使用忽略在 setUp() 方法中初始化的欄位選項，可控制在測試用例 setUp() 方法中寫入該欄位時是否應發出警告。
