報告對無法為之找到相應 Condition.await() 調用的 Condition.signal() 或 Condition.signalAll() 的調用。

此檢查僅報告以當前類的目標欄位為目標的調用。

範例：


  class Queue {
    private final Condition isEmpty = ...;

    void add(Object elem) {
      // ...
      isEmpty.signal(); // 警告：在沒有相應 'await()' 的情況下調用 'signal()'
      // ...
    }

    void remove(Object elem) throws InterruptedException {
      // ...
      //      isEmpty.await();
      // ...
    }
  }
