報告對無法為之找到相應 Condition.await() 呼叫的 Condition.signal() 或 Condition.signalAll() 的呼叫。

此檢查僅報告以當前類別的目標欄位為目標的呼叫。

範例：


  class Queue {
    private final Condition isEmpty = ...;

    void add(Object elem) {
      // ...
      isEmpty.signal(); // 警告：在沒有相應 'await()' 的情況下呼叫 'signal()'
      // ...
    }

    void remove(Object elem) throws InterruptedException {
      // ...
      //      isEmpty.await();
      // ...
    }
  }
