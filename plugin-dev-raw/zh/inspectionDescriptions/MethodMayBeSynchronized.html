報告其主體只包含一條 synchronized 語句的方法。 對於實例方法，此 synchronized 語句的鎖定表達式必須等於 this；對於 static 方法，必須等於 [ClassName].class。

為改善此類別方法的可讀性，可以移除 synchronized 包裝器，並將方法標記為 synchronized。

範例：


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

在套用快速修復後：


  public synchronized int generateInt(int x) {
    return 1;
  }
