報告不必要的、不會增加方法的可理解性的局部變數，包括：
  • 立即返回的局部變數。
  • 立刻賦值給另一個變數並且不再使用的局部變數。
  • 始終與另一個局部變數或參數具有相同值的局部變數。

範例：


  boolean yes() {
    boolean b = true;
    return b;
  }

在套用快速修復後：


  boolean yes() {
    return true;
  }

設定檢查：

  • 使用忽略立即返回或拋出的變數選項可以忽略立即返回或拋出的變數。 為清晰起見和易於偵錯，某些編碼樣式建議使用此類變數。
  • 使用忽略具有註解的變數選項可以忽略註解的變數。
