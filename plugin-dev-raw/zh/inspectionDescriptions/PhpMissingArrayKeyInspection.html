報告根據預定義陣列形狀建立陣列但未提供形狀定義的所有必需鍵/條目的情況。

此檢查幫助您識別由於缺少必需條目而可能導致執行時問題的陣列實例化場景，從而確保更好的程式碼一致性和對定義結構的遵守。

嵌入程式碼段： 


      function connect(#[ArrayShape([
          'path' => 'string',
          'host' => [
              'name' => 'string',
              'port' => 'int'
          ]
      ])] array $options = []) { };
      connect(['host' => ['name' => 'localhost', 'port' => 1234]]); // 缺少 'path' 鍵
