報告未宣告為 static 和 final 的類的記錄器欄位。 確保每個類記錄器都是有效的常數並綁定到該類可以簡化為應用程式提供統一日誌記錄實作的任務。 此檢查不報告接口、枚舉、註解和內部類。

提供了一個快速修復來將記錄器修飾符更改為 static final。

範例:


  public class Significant {
    private Logger LOG = Logger.getLogger(Critical.class);
  }

在套用快速修復後：


  public class Significant {
    private static final Logger LOG = Logger.getLogger(Critical.class);
  }

設定檢查：

  • 使用記錄器類名表可以指定記錄器類名。 該檢查將報告不是 static 和 final 並且類型等同於指定類名之一的欄位。
