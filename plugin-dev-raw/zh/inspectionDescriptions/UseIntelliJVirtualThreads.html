報告使用 Thread.ofVirtual 而不是 IntelliJVirtualThreads.ofVirtual 的情況。

Thread.ofVirtual 是一個用於建立虛擬執行緒的通用 Java API，不適合 IntelliJ 平台的需求。

快速修復將使用 IntelliJVirtualThreads.ofVirtual 執行取代。

範例： 


// 錯誤：
Thread thread = Thread.ofVirtual().start();

// 正確：
Thread thread = IntelliJVirtualThreads.ofVirtual().start();
