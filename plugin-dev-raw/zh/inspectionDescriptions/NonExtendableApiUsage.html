報告擴展、實作或覆寫使用 @ApiStatus.NonExtendable 標記的 API 元素的類別、接口和方法。

@ApiStatus.NonExtendable 註解表示類別、接口或方法不得擴展、實作或覆寫。 將此類別接口和類別轉換為內部庫實作很常見，因此，如果客戶端提供不同的實作，將獲得 ClassCastException。 將新的 abstract 方法新增到這種類和接口將破壞與客戶端實作的兼容性。

範例：


  // 在上游庫程式碼中
  @ApiStatus.NonExtendable
  public interface MyService {
    public void perform();
  }

  // 在下游客戶程式碼中
  public class MyServiceImpl implements MyService {
    @Override
    public void perform() {
      // ...
    }
  }
此檢查還可以檢測以下與錯誤套用註解相關的問題：
  • 為不能擴展/覆寫的類別/方法新增註解
  • 為已經使用 @ApiStatus.NonExtendable 註解的類別中的方法冗餘新增註解
