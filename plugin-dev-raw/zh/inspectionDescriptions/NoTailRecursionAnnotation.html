報告沒有 @tailrec 註解的尾遞迴方法，該註解驗證方法將使用尾呼叫最佳化進行編譯。

請注意，方法必須是有效的 final，此檢查才會被顯示。

範例：


  def factorial(n: Long, acc: Long = 1): Long =
    if (n <= 1) acc else factorial(n - 1, acc * n)

在套用快速修復後：


  @tailrec
  def factorial(n: Long, acc: Long = 1): Long =
    if (n <= 1) acc else factorial(n - 1, acc * n)
