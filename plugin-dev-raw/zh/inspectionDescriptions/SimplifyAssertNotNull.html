報告檢查已宣告變數非 null 值的 assert 呼叫。

使用 !! or ?: 可使程式碼更簡單。

該快速修復會將變數初始設定式中的 assert 取代為 !! 或 ?: 運算符。

範例：


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

在套用快速修復後：


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }
