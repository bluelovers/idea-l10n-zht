報告 serialVersionUID 欄位未被宣告為 private static final long 的 Serializable 類別。

範例：


  class SampleClass implements Serializable {
    private long serialVersionUID = 1; // Serializable 類別的欄位未宣告為 'private static final long'

    public SampleClass() {
        System.out.println(serialVersionUID);
    }
  }
