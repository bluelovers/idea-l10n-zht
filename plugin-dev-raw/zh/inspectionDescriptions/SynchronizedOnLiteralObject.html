報告在用文字初始化的物件上鎖定的 synchronized 塊。

字串文字是暫存的，Character、Boolean 和 Number 文字可以從快取中分配。 正因為如此，可能系統的其他某些部分（使用通過相同文字初始化的物件）實際上會包含對完全相同的物件的引用。 如果鎖定物件被認為是 private 物件，這可能會產生意外的死鎖情況。

範例：


  class Main {
    final String mutex = "Mutex";
    void method() {
      synchronized (mutex) {
      }
    }
  }

使用警告所有可能的文字選項可以報告 String、Character、Boolean 和 Number 物件的任何同步。
