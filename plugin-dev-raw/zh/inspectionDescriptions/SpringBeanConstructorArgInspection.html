報告使用基於構造函數的依賴項注入的 Spring <bean>。

當使用設定的 <constructor-arg> 標記和定義的 autowire 策略找不到 <bean> 的合適構造函數或工廠方法時，將觸發檢查。

範例：


    public class MyComponent {
    // constructor
    public MyComponent(MyBean bean) {}
    // factory method
    public static MyComponent getInstance(String name, int port) {
    }
    <beans>
      <bean class="beans.MyComponent" id="foo"> // 報告 "在類 'MyComponent' 中找不到符合的構造函數"
         <constructor-arg ref="myBean"/>
         <constructor-arg value="123"/>
      </bean>

      <bean class="beans.MyComponent" factory-method="getInstance"> // 報告 "在類中找不到符合的工廠方法"
         <constructor-arg value="123"/>
      </bean>
    </beans>
