報告未用類別名稱限定的 static 成員的用法。

如果 static 成員在同一個類別中，是合法的，但可能令人困惑。

範例：


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      foo();
      System.out.println(x);
    }

    static void baz() { foo(); }
  }

在套用快速修復後：


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      Foo.foo();
      System.out.println(Foo.x);
    }

    static void baz() { Foo.foo(); }
  }

使用檢查設定可以切換以下各項的報告：

  • static 欄位存取
    void bar() { System.out.println(x); }

  • 對 static 方法的呼叫
    void bar() { foo(); }
    static void baz() { foo(); }

您還可以將檢查組態為僅報告來自非 static 上下文的 static 成員用法。 在上面的範例中，不會報告 static void baz() { foo(); }。
