報告可以使用 Math.max() 或 Math.min() 呼叫而不是手動計算兩個數字的最小值或最大值的情況。

範例：


  public int min(int a, int b) {
    return b < a ? b : a;
  }

在套用快速修復後：


  public int min(int a, int b) {
    return Math.min(a, b);
  }

使用停用 float 和 double 選項，可對 double 和 float 類型停用此檢查。 由於該快速修復在處理 NaN 時可能會略微更改 float/ double 類型的語意，因此這很有用。 然而，如果不考慮 NaN，此快速修復在大多數情況下其實會修正難以捉摸的錯誤。
