報告可以使用 Math.max() 或 Math.min() 調用而不是手動計算兩個數字的最小值或最大值的情況。

示例：


  public int min(int a, int b) {
    return b < a ? b : a;
  }

在應用快速修復後：


  public int min(int a, int b) {
    return Math.min(a, b);
  }

使用禁用 float 和 double 選項，可對 double 和 float 禁用此檢查。 由於該快速修復在處理 NaN 時可能會略微破壞 float/ double 的語義，因此這很有用。 不過，如果使用者不在意 NaN，那麼在大多數情況下它其實會修正難以捉摸的錯誤。
