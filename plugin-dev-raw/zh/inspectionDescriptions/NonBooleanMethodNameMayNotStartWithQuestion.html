報告名稱以疑問詞開頭的非布爾方法。 此類方法名稱可能令人困惑。

此檢查將忽略覆寫庫方法的非布爾方法。

範例： 


  public void hasName(String name) {
    assert names.contains(name);
  }

只能在編輯器中使用重命名此類方法的快速修復。

設定檢查：

  • 使用布爾方法名稱前綴列表可指定應僅用於布爾方法的疑問詞。
  • 使用忽略具有 'java.lang.Boolean' 返回類型的方法選項可忽略返回類型為 java.lang.Boolean 的方法。
  • 使用忽略覆寫/實作 super 方法的方法選項可忽略具有 super 方法的方法。
