報告當整個 if/when 表達式本身被返回時，在 if 或 when 的分支最後一個表達式中出現的冗餘 return 關鍵字。

這些關鍵字不會增加任何價值，只會導致程式碼混亂。

該快速修復可以移除冗餘的 return 關鍵字，只保留表達式。

範例：


fun foo(flag: Boolean): String {
  return if (flag) {
    "foo"
  } else {
    return "bar"  // 冗餘的 'return' 關鍵字
  }
}

在套用快速修復後：


fun foo(flag: Boolean): String {
  return if (flag) {
    "foo"
  } else {
    "bar"
  }
}
