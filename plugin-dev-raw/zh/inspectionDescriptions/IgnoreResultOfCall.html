報告其結果被忽略的方法調用。

忽略結果對於許多方法完全合法，但對於某些方法，幾乎肯定是錯誤。 忽略結果可能是錯誤之舉的方法範例包括返回實際讀取字節數的 java.io.inputStream.read()，以及 java.lang.String 或 java.math.BigInteger 上的任何方法。 這些方法不會產生副作用，因此忽略它們的結果將毫無意義。

將檢查對以下方法的調用：

  • 簡單 getter（除返回欄位外別無他用）
  • 在此檢查的設定中指定的方法
  • 使用 org.jetbrains.annotations.Contract(pure=true) 註解的方法
  • 使用 .*.CheckReturnValue 註解的方法
  • 使用 javax.annotation.CheckReturnValue 註解的類別或軟體套件中的方法
  • 所有非庫方法（可選）

不會報告對使用 Error Prone 或 AssertJ 的 @CanIgnoreReturnValue 註解進行註解的方法的調用。

使用檢查設定指定要檢查的類別。 方法採用 Java 正則表達式語法按名稱或名稱模式進行符合。 對於類別，請使用完全限定名稱。 每個條目都應用於該類及其所有繼承者。
