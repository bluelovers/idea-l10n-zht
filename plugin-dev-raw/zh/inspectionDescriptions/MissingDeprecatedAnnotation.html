報告具有 @deprecated Javadoc 標記但沒有 @java.lang.Deprecated 註解的模組宣告、類、欄位或方法。

建議添加缺失的 @Deprecated 註解。

範例:

  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

在套用快速修復後：

  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

此檢查僅在專案或模組的語言版本為 5 或更高中有效。

使用下面的複選框，對用 @deprecated 註解而在 @Deprecated Javadoc 標記中沒有解釋的符號發出警告。
