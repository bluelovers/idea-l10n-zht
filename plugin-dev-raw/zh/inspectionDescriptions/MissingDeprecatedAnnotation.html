報告具有 @deprecated Javadoc 標記但沒有 @java.lang.Deprecated 註解的模組宣告、類、欄位或方法。

範例：


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

在套用快速修復後：


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

此檢查僅在專案或模組的語言級別為 5 或更高時有效。

使用下面的複選框，對使用 @Deprecated 註解但在 @deprecated Javadoc 標記中沒有解釋的符號發出警告。
