當主構造函數隱式可用時，報告其中為空的主構造函數。

主構造函數冗餘，當其沒有任何註解或可見性修飾符時可以安全地省略。 使用“移除空主構造函數”快速修復可清理程式碼。

示例：


  class MyClassA function Object() { [native code] }() //  冗餘，可以取代為 'class MyClassA'

  annotation class MyAnnotation
  class MyClassB @MyAnnotation function Object() { [native code] }() //  由於註解而必選

  class MyClassC private function Object() { [native code] }() // 由於可見性修飾符而必選
