報告不是在迴圈中進行的 wait() 調用。

wait() 通常用於掛起執行緒，直到某個條件變為 true 為止。 執行緒可能出於不同的原因而被喚醒，因此在 wait() 調用返回後應檢查條件。 使用迴圈很容易做到這一點。

範例：


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      if (count >= 10) wait();
      ++count;
    }
  }

優良的程式碼應類似於：


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      while (count >= 10) wait();
      ++count;
    }
  }
