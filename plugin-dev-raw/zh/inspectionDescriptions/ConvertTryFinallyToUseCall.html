報告 finally 中帶有 resource.close() 並可轉換為 resource.use() 調用的 try-finally 塊。

use() 更容易讀取，且不易出錯，因為無需顯式 close() 調用。

範例：


  fun example() {
      val reader = File("file.txt").bufferedReader()
      try {
          reader.lineSequence().forEach(::print)
      } finally {
          reader.close()
      }
  }

在套用快速修復後：


  fun example() {
      File("file.txt").bufferedReader().use { reader ->
          reader.lineSequence().forEach(::print)
      }
  }
