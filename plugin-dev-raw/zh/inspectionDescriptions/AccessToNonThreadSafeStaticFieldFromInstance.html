報告對非執行緒安全類型的 static 欄位的存取。

從實例方法或非同步塊存取 static 欄位時，多個執行緒都可以存取該欄位。 這可能導致未指明的副作用，例如異常和不正確的結果。

範例：


  class Sample {
    private static final SimpleDateFormat df = new SimpleDateFormat("yyyy-MM-dd");
    String method() {
      return df.format("");
    }
  }

您可以指定哪些類型應被視為非執行緒安全。 僅報告具有這些確切類型或使用這些確切類型進行初始化的欄位，因為可能存在這些類型的執行緒安全子類。
