報告使用類別文字初始化的 Logger 實例，而非使用 Logger 駐留的類別。

當從另一個類別複製貼上時可能會發生這種情況。 這可能導致記錄意外類別下的事件以及錯誤篩選。

使用檢查選項可指定此檢查識別的記錄器工廠類別和方法。

範例：


class AnotherService
class MyService {
    private val logger = LoggerFactory.getLogger(AnotherService::class.java)
}

在套用快速修復後：


class MyService {
    private val logger = LoggerFactory.getLogger(MyService::class.java)
}
