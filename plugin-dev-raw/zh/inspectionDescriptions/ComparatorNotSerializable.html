報告實作 java.lang.Comparator 但不實現 java.io.Serializable 的類。

如果使用不可序列化的比較器來建置 java.util.TreeMap 或 java.util.TreeSet 之類的有序集合，那麼該集合也將不可序列化。 這可能會導致意外且難以診斷的錯誤。

java.lang.Comparator 的子類通常無狀態，因此只需將它們標記為可序列化，即可輕鬆避免此類問題。

範例：


  class Foo implements Comparator { // 警告
      @Override
      public int compare(Object o1, Object o2) {
          /* ... */
      }
  }

在套用快速修復後：


  class Foo implements Comparator, Serializable { // 此處無警告
      @Override
      public int compare(Object o1, Object o2) {
          /* ... */
      }
  }
