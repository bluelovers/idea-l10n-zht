報告錯誤的 Micronaut 資料存儲庫方法參數類型。

範例:


  public class Person {
    private int id;
    private String lastname;
    private Address address;
  }

  public class Address {
    private int zipCode;
  }

  public interface CustomerRepository extends CrudRepository<Person, Integer> {
    List<Person> findByAddress(String lastname);  // 應為 'Address' 類型
    List<Person> findByAddressZipCode(String str);   // 應為 'int' 類型
  }
