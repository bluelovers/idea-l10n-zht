報告無意義的算術表達式。 這樣的表達式包括加零、減零、乘以零或一以及除以一。

這樣的表達式可能是自動重構所致，不太可能出於開發人員的本意。

快速修復簡化了這樣的表達式。

範例: 


  void f(int a) {
    int x = a - a;
    int y = a + 0;
    int res = x / x;
  }

在套用快速修復後：


  void f(int a) {
    int x = 0;
    int y = a;
    int res = 1;
  }

請注意，在極少數情況下，所建議的取代可能不完全等同於所有可能的輸入的原始程式碼。 例如，該檢查建議將 x / x 取代為 1。 但是，如果 x 為零，原始程式碼將拋出 ArithmeticException 或導致 NaN。 另外，如果 x 為 NaN，那麼結果也是 NaN。 這種行為不太可能出於本意。
