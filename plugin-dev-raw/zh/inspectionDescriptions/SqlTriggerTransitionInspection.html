報告觸發器中轉換表變數的不正確用法。

範例 (HSQLDB)：

CREATE TABLE foo(a INT);

CREATE TRIGGER trg
 AFTER DELETE ON foo
BEGIN
   SELECT * FROM NEW;
END;

CREATE TRIGGER trig AFTER INSERT ON foo
   REFERENCING OLD ROW AS newrow
   FOR EACH ROW WHEN (a > 1)
   INSERT INTO foo VALUES (1)

在 HSQLDB 中，DELETE 觸發器只能用於 OLD 狀態，而 INSERT 觸發器只能用於 NEW 狀態。 因此，在前面的範例中，SELECT * FROM NEW; 中的 NEW 和 REFERENCING OLD ROW AS newrow 中的 OLD 將被高亮顯示。
