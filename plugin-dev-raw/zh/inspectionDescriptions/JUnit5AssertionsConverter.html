報告 JUnit 5 測試中對任何junit.framework.Assert、org.junit.Assert 或 org.junit.Assume 類別中的方法的呼叫。

儘管這些測試正常工作，但遷移到 org.junit.jupiter.api.Assertions/org.junit.jupiter.api.Assumptions 有助於避免對舊版 JUnit 的相依。

範例：


  import org.junit.Assert;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assert.assertEquals(4, 2 + 2);
    }
  }

在套用快速修復後：


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assertions.assertEquals(4, 2 + 2);
    }
  }
