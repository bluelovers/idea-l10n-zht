報告 "reactive" 方法中的 subscribe() 調用。

返回 Publisher 類型（包括 Flux 和 Mono）的方法不應直接調用 subscribe() 方法，因為它可能破壞反應式調用鏈。

不要使用 subscribe()，而是考慮使用組合運算符，例如 flatMap()、zip()、then() > 等。

範例：


  Flux<String> stringFlux(){
    Flux<String> flux = Flux.just("abc");
    flux.subscribe(); // <- 在非阻塞上下文中阻塞 'subscribe' 調用
    return flux;
  }
