報告可取代為 ..< 運算符的 until。

每個 until 到 ..< 的取代都不會以任何方式更改語意。

UX 研究表明，與 until 相比，開發者在讀取包含 ..< 的程式碼時的錯誤減少了約 20-30%。

範例：


  fun main(args: Array<String>) {
      for (index in 0 until args.size) {
          println(index)
      }
  }

在套用快速修復後：


  fun main(args: Array<String>) {
      for (index in 0..<args.size) {
          println(index)
      }
  }

此檢查僅在專案或模組的 Kotlin 語言級別為 1.7 或更高時報告，並且提供了 -opt-in=kotlin.ExperimentalStdlibApi 編譯器標誌。
