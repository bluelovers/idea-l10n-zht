報告不宣告 throws CloneNotSupportedException 的 clone() 方法。

如果沒有宣告 throws CloneNotSupportedException，該方法的子類別將不能以標準方式禁止克隆。 此檢查不會報告宣告為 final 的 clone() 方法和 final 類別內的 clone() 方法。

組態檢查：

使用僅對 'protected' 克隆方法發出警告選項可以指示此檢查僅對 protected clone() 方法發出警告。 Effective Java 一書（第二版和第三版）建議在 public 方法上省略 CloneNotSupportedException 宣告，因為不拋出已檢查的異常的方法更容易使用。

範例：


  public class Example implements Cloneable {
    // 方法未宣告 ’throws CloneNotSupportedException‘
    protected Object clone() {
        try {
            return super.clone();
        } catch (CloneNotSupportedException e) {
            return null;
        }
    }
  }
