報告區域變數的顯式給定類型，這些類型顯而易見且冗餘，例如 val f: Foo = Foo()。

範例：


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

在套用快速修復後：


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
