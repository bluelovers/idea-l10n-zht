報告 junit.framework.TestCase 繼承者中的 JUnit 4 @Test 註解方法。 混合 JUnit 的 API 會導致執行測試出現問題。

範例：


  public class MyTest extends TestCase {
      @Test //名稱不是以 "test" 開頭，因此將被忽略
      public void wouldBeIgnored() {}
  
      @Test   //名稱以 "test" 開頭
      @Ignore //因此即便有 @Ignore 註解也會執行
      public void testWouldBeExecuted() {}
  }

提供的修正：

  • 移除 @Ignore 註解並重命名測試方法，使名稱不以“test”開頭。
  • 將 JUnit 3 測試類轉換為 JUnit 4。
