報告可以從分支中提取通用部分的 if 語句。

這些通用部分獨立於條件並使 if 語句更難理解。

範例：


  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

在套用快速修復後：


  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();

這是 2018.1 更新的功能
