報告標準庫中已篩選的集合上的 filter… 調用。

只有在類型不同於結果類型的接收器上調用時，標準庫中 filterNotNull() 或 filterIsInstance 之類的幾個函數才有意義。 在其他情況下，可以忽略此類調用，因為結果將相同。

移除冗餘調用快速修復可用於自動修正程式碼。

範例：


  fun test(list: List<String>) {
      val x = list.filterNotNull() // 快速修復會簡化為 'list'
      val y = list.filterIsInstance<String>() // 快速修復會簡化為 'list'
  }
