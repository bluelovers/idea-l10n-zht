報告對 lambda 參數的賦值，包括複合賦值和遞增或遞減參數。

雖然有時是有意為之，但這種結構可能會引起混淆，通常由拼寫錯誤或使用了錯誤變數引起。

該快速修復會添加新變數的宣告。

範例:


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

在套用快速修復後：


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

使用如果賦值是原始參數的轉換，則忽略選項來忽略根據先前值修改參數值的賦值。
