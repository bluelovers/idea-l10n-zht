報告對 lambda 參數的賦值或修改。 雖然偶爾是有意為之，但此結構可能令人困惑，通常是由拼寫錯誤或使用錯誤變數所致。

該快速修復會新增新變數的宣告。

範例：


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

在套用快速修復後：


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

使用如果賦值是原始參數的轉換，則忽略選項可以忽略根據先前值修改參數值的賦值。
