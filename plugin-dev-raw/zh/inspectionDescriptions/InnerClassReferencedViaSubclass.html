報告對其中的呼叫由宣告類別的子類別而不是宣告類別本身限定的內部類別和嵌套類別的存取。

Java 允許此類別限定，但是此類別存取可能表明繼承和覆寫之間存在難以捉摸的混淆。

範例：


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Sub.Inner s = new Sub.Inner(); // 'Inner' 類別在 'Super' 類別中宣告，但通過 'Sub' 類別參照
    }
  }

在套用快速修復後：


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Super.Inner s = new Super.Inner();
    }
  }
