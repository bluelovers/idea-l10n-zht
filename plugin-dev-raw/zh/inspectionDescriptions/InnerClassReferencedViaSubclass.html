報告對其中的調用由宣告類的子類而不是宣告類本身限定的內部類和嵌套類的存取。

Java 允許此類限定，但是此類存取可能表明繼承和覆寫之間存在難以捉摸的混淆。

範例:


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Sub.Inner s = new Sub.Inner(); // 'Inner' 類在 'Super' 類中宣告，但通過 'Sub' 類引用
    }
  }

在套用快速修復後：


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Super.Inner s = new Super.Inner();
    }
  }
