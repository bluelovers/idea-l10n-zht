報告賦值後未使用的賦值。 如果不使用，最好移除該賦值，以縮短程式碼並避免冗餘分配。

報告的情況如下：

  • 該變數在賦值後從未被讀取。
  • 該變數在被讀取前始終被新值覆寫。
  • 變數初始設定式冗餘（出於以上兩種原因之一）。

設定檢查：

使用報告冗餘初始設定式選項可以報告冗餘的初始設定式： 


  int getI() {
    int i = 0; // 冗餘初始化
    i = 2;
    return i;
  }

使用在可以取代為(i + 1)時報告 ++i 選項，報告 ++i 表達式可以取代為 i + 1 的情況： 


  int preInc(int i) {
    int res = i;
    return ++res;
  }

使用在變更後的值後續未使用時報告 i++ 選項可以報告後續不使用 i++ 表達式的結果的情況： 


  int postInc(int i) {
    int res = i;
    return res++;
  }
