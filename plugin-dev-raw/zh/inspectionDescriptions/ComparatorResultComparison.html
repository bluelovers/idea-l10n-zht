報告將 Comparator.compare 或 Comparable.compareTo 的結果與特定的非零常數進行比較的情況。

根據約定，這些方法可以返回任何正數（不僅僅是 1）或任何負數（不僅僅是 -1），因此與特定數字進行比較是不良做法。 一些廣泛使用的比較方法（例如 String.compareTo）實際上返回 [-1..1] 範圍之外的值，因此這樣的比較可能會導致錯誤的程序行為。

範例:


  void validate(String s1, String s2) {
    // 與 1 比較不正確
    if (s1.compareTo(s2) == 1) {
      throw new IllegalArgumentException("Incorrect order");
    }
  }

在套用快速修復後：


  void validate(String s1, String s2) {
    if (s1.compareTo(s2) > 0) {
      throw new IllegalArgumentException("Incorrect order");
    }
  }
