報告不屬於任何類、方法或欄位的 Javadoc 註釋。

Javadoc 工具忽略懸空的 Javadoc 註釋，並且不會將它們包含在生成的 HTML 文檔中。

範例:


  class A {
    /**
     * Dangling comment
     */
     /**
      * Method javadoc
      */
      public void m(){}
  }

通過快速修復可完全移除這樣的註釋或將它們轉換為塊註釋。 在套用快速修復後：


  class A {
  /*
    Dangling comment
   */
  /**
   * Method javadoc
   */
  public void m(){}
}
