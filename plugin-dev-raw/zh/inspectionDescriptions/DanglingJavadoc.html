報告不屬於任何類別、方法或欄位的 Javadoc 註釋。 Javadoc 工具會忽略懸空的 Javadoc 註釋，並且不會將它們包含在其生成的 HTML 文檔中。

範例：


  class A {
    /**
     * Dangling comment
     */
    /**
     * Method javadoc
     */
     public void m(){}
  }

可以通過快速修復完全刪除此類別註釋或將它們轉換為塊註釋。 在套用快速修復後：


  class A {
    /*
      Dangling comment
    */
    /**
     * Method javadoc
     */
    public void m(){}
  }

使用忽略 JavaDoc 格式的檔案頭註釋選項可以忽略 Java 檔案開頭的註釋。 這些通常是版權訊息。
