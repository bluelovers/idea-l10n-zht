報告不是在相應的 synchronized 語句或 synchronized 方法進行中的 wait()、notify() 和 notifyAll() 調用。

在沒有持有鎖定的對象上調用這些方法會導 IllegalMonitorStateException。 此類結構不一定錯誤，因為在調用該包含方法前可能已獲取必要的鎖定，但即便如此，還是要仔細查看。

示例：


  class Sync {
    private final Object lock = new Object();

    void test() throws InterruptedException {
      synchronized (this) {
        lock.wait(); // 'lock.wait()' 未在 'lock' 上同步
      }
    }
  }
