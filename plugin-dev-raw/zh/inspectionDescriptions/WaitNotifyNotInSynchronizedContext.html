報告不是在相應的 synchronized 語句或 synchronized 方法進行中的 wait()、notify() 和 notifyAll() 呼叫。

在沒有持有鎖定的物件上呼叫這些方法會導 IllegalMonitorStateException。 此類別結構不一定錯誤，因為在呼叫該包含方法前可能已獲取必要的鎖定，但即便如此，還是要仔細檢視。

範例：


  class Sync {
    private final Object lock = new Object();

    void test() throws InterruptedException {
      synchronized (this) {
        lock.wait(); // 'lock.wait()' 未在 'lock' 上同步
      }
    }
  }
