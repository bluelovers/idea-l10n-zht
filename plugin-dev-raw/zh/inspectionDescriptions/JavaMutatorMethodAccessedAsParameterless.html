報告對具有類似轉變器名稱而沒有實參子句的方法的呼叫。

Scala 允許在沒有實參子句的情況下調用無參數 Java 方法。 這主要是為了允許 Scala 程式碼以與 Scala 存取器方法相同的方式使用無參數 Java 存取器方法。 即，不使用任何實參子句並遵守統一存取原則，也就是說客戶端程式碼不應受到將特性實作為欄位或方法的決定的影響。

但是，對於具有副作用的方法，通常使用空實參子句。

該快速修復會新增一個空實參子句。

範例：


  val stringBuilder = new java.util.ArrayList[String]
  stringBuilder.clear

在套用快速修復後：


  val stringBuilder = new java.util.ArrayList[String]
  stringBuilder.clear()
