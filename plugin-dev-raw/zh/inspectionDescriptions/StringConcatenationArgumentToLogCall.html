報告用作 SLF4J 和 Log4j 2 日誌記錄方法實參的非常數字串串聯。 即使未記錄日誌訊息，也會在執行時評估非常數串聯；這會對性能產生負面影響。 建議改用參數化日誌訊息，禁用日誌記錄時不會對其進行評估。

範例：


  public class Vital {
    private static final Logger LOG = LoggerFactory.getLogger(Vital.class);

    public void saveTheWorld(int i, String s, boolean b) {
      LOG.info("saveTheWorld(" + i + ", " + s + ", "  + b + ")");
      // todo
    }
  }

在套用快速修復後：


  public class Vital {
    private static final Logger LOG = LoggerFactory.getLogger(Vital.class);

    public void saveTheWorld(int i, String s, boolean b) {
      LOG.info("saveTheWorld({}, {}, {})", i, s, b);
      // todo
    }
  }

設定檢查：

  • 使用警告物件列表可以忽略某些更高的日誌記錄級別。 即使在生產中也可以啟用更高的日誌記錄級別，並且始終會評估實參。
