在 Kotlin/JS 專案中報告外部接口中不可為 null 的 Boolean 屬性。

外部接口中的 Boolean 屬性應當可為 null，以處理 JavaScript 值可能為 undefined 的情況。

範例：

external interface MyInterface {
    var enabled: Boolean // 警告：應當為 Boolean?
    var visible: Boolean // 警告：應當為 Boolean?
}

在套用快速修復後：

external interface MyInterface {
    var enabled: Boolean?
    var visible: Boolean?
}
