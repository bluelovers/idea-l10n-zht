報告具有以下問題的 Javadoc 註釋和標記：
  • 標記名稱無效
  • 標記描述不完整
  • 標記重複
  • 缺少 Javadoc 描述

範例：


  /**
   * 標記名稱無效
   * @poram param description
   */
  public void sample(int param){
  }

範例：


  /**
   * 指向自身 {@link #sample(int)}
   */
  public void sample(int param){
  }

該快速修復會將未知的 Javadoc 標記添加到使用者定義的附加標記列表中。

使用下面的文本欄位可以定義附加 Javadoc 標記。

使用第一個複選框可以忽略重複的 'throws' 標記。

使用第二個複選框可以忽略描述中第一句話缺失或不完整的問題。

使用第三個複選框可以忽略指向自身的引用。
