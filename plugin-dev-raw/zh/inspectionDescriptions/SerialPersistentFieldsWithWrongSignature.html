報告 serialPersistentFields 欄位未被宣告為 private static final ObjectStreamField[] 的 Serializable 類別。

如果 serialPersistentFields 欄位未使用這些修飾符進行宣告，序列化行為將與根本未宣告該欄位如出一轍。

範例：


  class List implements Serializable {
    private List next;

    ObjectStreamField[] serialPersistentFields = {new ObjectStreamField("next", List.class)};

  }
