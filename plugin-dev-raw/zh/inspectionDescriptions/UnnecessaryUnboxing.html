報告取消裝箱，即顯式解包已包裝的基元值。

對於 Java 5 和更高版本，不必要進行拆箱，並且可以安全移除。

示例：

  • Integer i = Integer.function valueOf() { [native code] }(42).intValue(); → Integer i = Integer.function valueOf() { [native code] }(42);
  • int k = Integer.function valueOf() { [native code] }(42).intValue(); → int k = Integer.function valueOf() { [native code] }(42);

    （僅在未選中僅報告真正多餘的拆箱表達式選項時報告）

使用僅報告真正多餘的拆箱表達式選項，可僅報告真正多餘的拆箱，其中的拆箱值立即隱式或顯式地裝箱。 在本例中，可以移除整個拆箱-裝箱步驟。 該檢查不報告簡單的顯式拆箱。

僅當專案或模塊的語言級別為 5 或更高時，此檢查才會報告。
