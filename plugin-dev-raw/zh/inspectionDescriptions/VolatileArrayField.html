報告被宣告為 volatile 的陣列欄位。 此類別宣告可能令人困惑，因為存取陣列本身遵循 volatile 欄位的規則，但存取陣列的內容時並非如此。

範例：


  class Data {
    private volatile int[] idx = new int[0];
  }

如果需要對陣列內容進行此類別可變存取，不妨改用 java.util.concurrent.atomic 類別： 


  class Data {
    private final AtomicIntegerArray idx = new AtomicIntegerArray(new int[0]);
  }
