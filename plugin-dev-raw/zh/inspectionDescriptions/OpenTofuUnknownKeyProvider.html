檢測 OpenTofu 加密塊組態中不存在的加密密鑰提供程序參照。 例如，對於以下程式碼 

  
    terraform {
      encryption {
        key_provider "pbkdf2" "foo" {
          # Configuration parameters
        }
        method "aes_gcm" "yourname" {
          keys = key_provider.pbkdf2.unknown
        }
        method "aes_gcm" "yourname2" {
          keys = key_provider.pbkdf2.foo
        }
      }
    }
我們應檢查密鑰提供程序參照是否正確。 在這種情況下，應醒目提示 unknown 參照。

有關詳情，請參閱 https://opentofu.org/docs/language/state/encryption/#methods。
