提供 Lombok 註解的一般檢查。

此檢查報告的警告和錯誤旨在與編譯時由 Lombok 註解處理程序報告的警告和錯誤 1:1 符合，例如：

  • 冗餘套用的註解（報告為警告）

    範例：

    
      @ToString
      class Person {
        final String name;
        final int age;

        @Override
        public String toString() {
            return "Person{name='" + name + "', age=" + age + '}';
        }
      }
    快速修復建議移除冗餘註解。
  • 不正確套用的註解（報告為錯誤）

    範例：

    
      @Value
      record Person(String name, int age) {}
    快速修復會建議移除無效註解。
