報告具有 @targetName 註解的定義，這些定義覆寫具有 @targetName 註解的定義，但具有不同的外部名稱。

請參閱 @targetName 註解。

該快速修復會將 @targetName 註解的外部名稱取代為被覆寫定義的外部名稱。

範例：


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("times")
    override def *(i: Int): A = ???

在套用快速修復後：


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???
