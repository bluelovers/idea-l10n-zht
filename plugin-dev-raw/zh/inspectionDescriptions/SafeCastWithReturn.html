報告帶 return 且可以取代為 if 類型檢查的安全轉換。

使用相應函式可以簡化程式碼。

該快速修復會將安全轉換取代為 if 類型檢查。

範例：


  fun test(x: Any) {
      x as? String ?: return
  }

在套用快速修復後：


  fun test(x: Any) {
      if (x !is String) return
  }

同樣，它能檢測到表單中的不安全轉換 


    fun foo(o: Any) {
    o as String ?: return
}

並轉換為類型檢查：


    fun foo(o: Any) {
    if (o !is String) return
}
