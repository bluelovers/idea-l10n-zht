報告帶 return 且可以取代為 if 類型檢查的安全轉換。

該快速修復會將安全轉換取代為 if 類型檢查。

範例：


  fun test(x: Any) {
      x as? String ?: return
  }

在套用快速修復後：


  fun test(x: Any) {
      if (x !is String) return
  }

Analogously, it detects unsafe casts in form 


    fun foo(o: Any) {
    o as String ?: return
}

and transforms into a type check:


    fun foo(o: Any) {
    if (o !is String) return
}
