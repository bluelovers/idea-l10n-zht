報告呼叫的未發生任何可檢測到的狀態更改的 Object.notify() 或 Object.notifyAll()。

Object.notify() 和 Object.notifyAll() 通常用於通知其他執行緒發生了狀態更改。 該狀態更改應發生在包含 Object.notify() 或 Object.notifyAll() 呼叫的同步上下文中，並且發生在呼叫之前。 儘管沒有此類別狀態更改未必就不正確，但肯定值得檢查。

範例：


  synchronized (this) {
    notify();
  }
  // 無狀態更改
  synchronized (this) {
    notify(); // 告知可能冗餘
  }
