報告調用的未發生任何可檢測到的狀態變更的 Object.notify() 或 Object.notifyAll()。

Object.notify() 和 Object.notifyAll() 通常用於通知其他執行緒發生了狀態變更。 該狀態變更應發生在包含 Object.notify() 或 Object.notifyAll() 調用的同步上下文中，並且發生在調用之前。 儘管沒有此類狀態變更未必就不正確，但肯定值得檢查。

範例：


  synchronized (this) {
    notify();
  }
  // 無狀態變更
  synchronized (this) {
    notify(); // 告知可能冗餘
  }
