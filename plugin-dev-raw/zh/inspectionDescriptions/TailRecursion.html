報告尾遞歸，即方法在返回前調用自身作為其最後一項操作。

尾遞歸總是可以取代為迴圈，迴圈的速度快得多。 有些 JVM 執行尾調用優化，而其他一些 JVM 則不執行。 因此，尾遞歸解決方案在不同的虛擬機上可能有截然不同的性能特徵。

範例：


  int factorial(int val, int runningVal) {
    if (val == 1) {
      return runningVal;
    } else {
      return factorial(val - 1, runningVal * val);
    }
  }

在套用快速修復後：


  int factorial(int val, int runningVal) {
    while (true) {
      if (val == 1) {
        return runningVal;
      } else {
        runningVal = runningVal * val;
        val = val - 1;
      }
    }
  }
