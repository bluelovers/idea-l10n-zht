報告尾遞迴，即方法在返回前呼叫自身作為其最後一項動作。

尾遞迴總是可以取代為迴圈，迴圈的速度快得多。 有些 JVM 執行尾呼叫最佳化，而其他一些 JVM 則不執行。 因此，尾遞迴解決方案在不同的虛擬機上可能有截然不同的性能特徵。

範例：


  int factorial(int val, int runningVal) {
    if (val == 1) {
      return runningVal;
    } else {
      return factorial(val - 1, runningVal * val);
    }
  }

在套用快速修復後：


  int factorial(int val, int runningVal) {
    while (true) {
      if (val == 1) {
        return runningVal;
      } else {
        runningVal = runningVal * val;
        val = val - 1;
      }
    }
  }
