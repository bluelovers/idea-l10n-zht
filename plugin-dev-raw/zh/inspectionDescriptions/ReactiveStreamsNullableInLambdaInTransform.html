報告可能在 Reactive Streams 鏈中返回 null 的轉換操作。

Reactive Streams 不支持可 null 值，因為這些值會導致此類程式碼失敗。 該快速修復會建議將 map() 取代為 mapNotNull，從而忽略異常。

範例：

repository.findWithTailableCursorBy()
    .map(e -> (Person)null)
    .doOnNext(System.out::println)

在套用快速修復後：

repository.findWithTailableCursorBy()
    .mapNotNull(e -> (Person)null)
    .doOnNext(System.out::println)
