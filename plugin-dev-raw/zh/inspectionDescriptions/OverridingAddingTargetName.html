報告具有 @targetName 註解的定義，這些定義覆寫沒有 @targetName 註解的定義。

請參閱 @targetName 註解。

該快速修復會移除 @targetName 註解。

範例：


  class A:
    def foo: Int = 1

  class B extends A:
    @targetName("overriddenFoo")
    override def foo: Int = 2

在套用快速修復後：


  class A:
    def foo: Int = 1

  class B extends A:
    override def foo: Int = 2
