報告其中比較的類型為接口，比較的表達式的類別的類型既不實現比較的接口，也不具有實作所比較接口的任何可見子類別的 instanceof 表達式。

儘管這可能是有意為之，但通常此類別結構很可能表明有錯誤，其中得到的 instanceof 表達式的求值結果始終為 false。

範例：


  interface I1 {}

  interface I2 {}

  interface I3 extends I1 {}

  static class Sub1 implements I1 {}

  static class Sub2 extends Sub1 implements I2 {
    void test(Sub1 sub1) {
      if (sub1 instanceof I3) { // 這裡的 'I3' 為不相容的接口
      }
    }
  }
