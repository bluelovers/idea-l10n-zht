報告可以取代為使用 &&、||、==、!= 或 ?: 運算符的條件的 if 語句 。

結果通常更短，但並不總是更清晰，因此不建議在每種情況下都套用此修復程序。

範例：


  if (condition) return true; else return foo;

在套用快速修復後：


  return condition || foo;

組態檢查：

  • 當建議使用 ?: 運算符時，請使用 不建議使用 '?:' 運算符 選項停用警告。 在這種情況下，只會突出顯示 &&、||、== 和 != 建議。 快速修復仍將在編輯器中可用。
  • 使用 忽略鏈式 'if' 語句 選項停用 if-else 鏈的警告 快速修復仍將在編輯器中可用。
