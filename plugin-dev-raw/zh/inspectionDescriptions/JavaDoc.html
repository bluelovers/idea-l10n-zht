報告以下 Javadoc 註釋缺陷：
  • 在需要有 Javadoc 的位置沒有 Javadoc
  • 缺少必要的標記
  • 標記無效或不完整
  • Javadoc 描述缺失或不完整

範例：


  class Main {
  }

在套用快速修復後：


  /**
   *
   */
  class Main {
  }

範例：


  String id(String value) {
    return value;
  }

在套用快速修復後：


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
使用頁籤和相應的檢查選項可定義需要具備 Javadoc 的可見性範圍，以及類別、方法、欄位和內部類別所需的 Javadoc 標記。
