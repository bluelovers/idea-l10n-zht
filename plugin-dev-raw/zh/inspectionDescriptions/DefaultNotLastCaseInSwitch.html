報告其中的 default case 位於另一個 case 前面的 switch 語句或表達式。

此結構造成不必要的困惑。 可以通過快速修復將 default case 移到最後位置。 只有給定分支的結尾有 break/yield 時，才能進行修正。

範例：


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

在套用快速修復後：


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
