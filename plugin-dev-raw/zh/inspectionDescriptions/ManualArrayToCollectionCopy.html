報告使用迴圈將陣列內容複製到集合中的程式碼。

一種更短並且有可能更快（取決於集合實作）的方法是使用 Collection.addAll(Arrays.asList()) 或 Collections.addAll()。

僅報告內部沒有附加語句的迴圈。

範例：


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

在套用快速修復後：


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
