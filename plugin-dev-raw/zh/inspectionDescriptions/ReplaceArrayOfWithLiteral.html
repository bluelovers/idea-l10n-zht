報告可以取代為數組文字 [...] 的 arrayOf 呼叫。

範例：

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // 可取代的 'arrayOf()'
  class MyClass

在套用快速修復後：

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass
