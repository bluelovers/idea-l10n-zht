建議將 for(Entry<?,?> entry : map.entrySet()) {...} 或 map.entrySet().forEach(entry -> ...) 取代為 map.forEach((key, value) -> ...)。

範例


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

在套用快速修復後：


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

啟用 不要報告迴圈選項後，只會報告 entrySet().forEach() 情況。 但是，該快速修復動作也可用於 for 迴圈。

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。
