報告實例特性定義在__init__ 方法外部的問題。

範例：

    class Book:
    def __init__(self):
        self.author = 'Mark Twain'

    def release(self):
        self.year = '1889'

套用快速修復後，程式碼範例變為： 

    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'

    def release(self):
        pass
