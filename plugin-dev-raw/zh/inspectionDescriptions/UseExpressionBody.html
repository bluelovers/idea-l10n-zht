返回可以取代為表達式主體語法的 return 表達式（一行程式碼或 when）。

表達式主體語法建議在樣式指南中給出。

轉換為表達式主體快速修復可用於自動修正程式碼。

範例：


  fun sign(x: Int): Int {
      return when { // <== 可以簡化
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

在套用快速修復後：


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
