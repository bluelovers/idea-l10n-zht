報告格式錯誤的 @Rule 或 @ClassRule 註解用法。 無法執行規則格式錯誤的測試。
  • @Rule 註解成員必須為 public 以及 org.junit.rules.TestRule 或 org.junit.rules.MethodRule 的子類型
  • @ClassRule 註解成員必須為 public 和 static 以及 org.junit.rules.TestRule 的子類型

範例：


  public class MomentousTest {
    @Rule
    private final TemporaryFolder tempFolder =
      new TemporaryFolder();

    // ... 測試轉到此處
  }

通過快速修復來修正修飾符：


  public class MomentousTest {
    @Rule
    public final TemporaryFolder tempFolder =
      new TemporaryFolder();

    // ... 測試轉到此處
  }
