報告未在相應的塊中使用的 catch 參數。 此檢查不會報告任何命名為 "ignore"或 "ignored" 的 catch 參數。

範例：


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ex) {
      println('Catching the exception')
  }

此處的參數 ex 永遠不會在 catch 塊中使用。

在套用快速修復後：


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ignored) {
      println('Catching the exception')
  }
