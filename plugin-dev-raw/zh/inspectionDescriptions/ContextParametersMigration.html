報告應遷移到上下文參數的上下文接收器。

從 Kotlin 2.2.0 開始，上下文接收器已被上下文參數取代。 此檢查有助於將已啟用上下文接收器但尚未遷移到上下文參數語法的專案從上下文接收器遷移到上下文參數。

範例： 


context(MutableList<String>, C)
private fun foo() {
    add("new")
    bar()
    baz()
}

class C {
    fun bar() {}
}

fun C.baz() {}
遷移後： 

context(strings: MutableList<String>, c: C)
private fun foo() {
    strings.add("new")
    c.bar()
    c.baz()
}

class C {
    fun bar() {}
}

fun C.baz() {}
