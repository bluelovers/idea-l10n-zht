報告以異常作為實參的 System.out.println() 調用。

使用 print 語句記錄異常會隱藏堆疊跟踪，這可能會使問題的調查變得複雜。 建議您改用記錄器。

也會報告以異常作為實參的 System.out.print()、System.err.println() 和 System.err.print() 調用。 建議使用記錄器來記錄異常。

例如，不建議使用：


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

用以下程式碼代替：


  try {
      foo();
  } catch (Exception e) {
      logger.warn(e); // 記錄器調用可能有所不同
  }
