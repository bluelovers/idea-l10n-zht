報告沒有參數子句並使用類似轉變器的名稱覆寫 Java 方法的方法。

Scala 允許覆寫沒有參數子句的無形參 Java 方法。 這主要是為了允許 Scala 程式碼以與 Scala 存取器方法相同的方式覆寫 Java 存取器方法。 即，不使用任何參數子句並遵守統一存取原則，也就是說客戶端程式碼不應受到將特性實作為欄位或方法的決定的影響。

但是，對於具有副作用的方法，通常有一個空參數子句。 這將符合里氏取代原則，即當被覆寫的方法有副作用時，覆寫方法也必須被宣告為具有副作用的方法。

該快速修復會新增一個空參數子句。

範例：


  import java.util

  class Test extends util.ArrayList[String] {
    override def clear: Unit = ???
  }

在套用快速修復後：


  import java.util

  class Test extends util.ArrayList[String] {
    override def clear(): Unit = ???
  }
