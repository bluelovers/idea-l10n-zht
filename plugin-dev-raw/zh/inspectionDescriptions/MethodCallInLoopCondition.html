報告迴圈語句條件部分中的方法調用。 在資源高度受限的環境中，此類調用可能會對性能產生不利影響。

不加考慮地套用此檢查的結果可能會對程式碼清晰度和設計產生負面影響。 此檢查適用於 Java ME 和其他資源高度受限的環境。

範例：


  String s = "example";
  for (int i = 0; i < s.length(); i++) {
    System.out.println(s.charAt(i));
  }

在套用快速修復後：


  String s = "example";
  int length = s.length();
  for (int i = 0; i < length; i++) {
    System.out.println(s.charAt(i));
  }

使用該選項可以忽略對常見 Java 迭代方法（例如 Iterator.hasNext()）和具有副作用的已知方法（例如 Atomic*.compareAndSet）的調用。
