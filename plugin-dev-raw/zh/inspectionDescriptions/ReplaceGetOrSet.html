報告對可通過索引運算符 [] 取代的 get 或 set 函數的顯式調用。

Kotlin 允許對類型的預定義運算符集進行自訂實作。 要多載運算符，您可以使用 operator 修飾符標記相應的函數： 


  operator fun get(index: Int) {}
  operator fun set(index: Int, value: Int) {}
以上函數對應於索引運算符。

範例：

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test().get(0) // 可取代的 'get()'
  }

在套用快速修復後：

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test()[0]
  }
